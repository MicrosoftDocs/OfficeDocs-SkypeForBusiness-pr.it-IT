## <a name="installing-the-stress-and-performance-tool"></a>Installazione dello strumento di stress e prestazioni

Eseguire il file di Windows Installer, **CapacityPlanningTool. msi**, in ogni computer client che verrà utilizzato per simulare il traffico degli utenti. Questi computer client non devono eseguire alcun componente di Lync Server.
