# <a name="planning-for-mobile-clients-in-lync-server-2013"></a>Pianificazione per i client mobili in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2013-04-02_

I client per dispositivi mobili Lync 2013 garantiscono una maggiore presenza, messaggistica istantanea, riunioni Lync e chiamate vocali e video tramite Internet o la connessione cellulare. Per una matrice che elenca le caratteristiche e le funzionalità dei client mobili e le confronta con il client desktop, vedere [tabelle di confronto tra client per dispositivi mobili per Lync Server 2013](lync-server-2013-mobile-client-comparison-tables.md).

<div>


> [!NOTE]  
> Lync Server 2013 supporta anche i client mobili di Lync 2010. Per altre informazioni, vedere la pagina relativa alla pianificazione per i client mobili nella raccolta TechNet di Lync <A href="http://go.microsoft.com/fwlink/p/?linkid=235955">http://go.microsoft.com/fwlink/p/?LinkID=235955</A>Server 2010.
