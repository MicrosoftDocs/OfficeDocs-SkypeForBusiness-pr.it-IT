_**Ultimo argomento modificato:** 2013-04-02_

I client per dispositivi mobili Lync 2013 offrono una maggiore presenza, messaggistica istantanea, riunioni di Lync e chiamate vocali e video tramite Internet o la connessione cellulare. Per una matrice che elenca le caratteristiche e le funzionalità dei client mobili e le confronta con il client desktop, vedere [tabelle di confronto client per dispositivi mobili per Lync Server 2013](lync-server-2013-mobile-client-comparison-tables.md).

<div>


> [!NOTE]  
> Lync Server 2013 supporta anche client mobili di Lync 2010. Per ulteriori informazioni, vedere la sezione relativa alla pianificazione per i client mobili nella libreria TechNet di Lync Server 2010 all'indirizzo <A href="https://go.microsoft.com/fwlink/p/?linkid=235955">https://go.microsoft.com/fwlink/p/?LinkID=235955</A> .
