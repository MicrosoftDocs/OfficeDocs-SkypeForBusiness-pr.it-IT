# <a name="web-server-and-services-cmdlets-in-lync-server-2013"></a>Cmdlet per server Web e servizi in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2013-02-18_

Molti componenti di Microsoft Lync Server 2013 sono basati sul Web: questi componenti utilizzano i servizi Web o le pagine webpage per svolgere le proprie attività. I cmdlet per il server Web e i servizi Web consentono di eseguire operazioni quali la configurazione delle impostazioni del server Web e la gestione degli URL semplici. Gli URL semplici agevolano la partecipazione degli utenti a riunioni e conferenze e semplificano l'accesso degli amministratori al pannello di controllo di Lync Server 2013.
