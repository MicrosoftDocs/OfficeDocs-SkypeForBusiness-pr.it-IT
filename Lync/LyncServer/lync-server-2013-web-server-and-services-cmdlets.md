# <a name="web-server-and-services-cmdlets-in-lync-server-2013"></a>Cmdlet per server Web e servizi in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2013-02-18_

Molti componenti di Microsoft Lync Server 2013 sono basati sul Web: questi componenti usano i servizi Web o le webpage per svolgere le proprie attività. I cmdlet Web Server e servizi Web consentono di eseguire operazioni come configurare le impostazioni del server Web e gestire URL semplici. Gli URL semplici consentono agli utenti di partecipare a riunioni e conferenze e semplificano l'accesso al pannello di controllo di Lync Server 2013.
