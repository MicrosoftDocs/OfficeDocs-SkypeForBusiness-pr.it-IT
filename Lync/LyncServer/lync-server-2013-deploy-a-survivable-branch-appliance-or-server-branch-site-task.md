## <a name="to-deploy-the-survivable-branch-appliance"></a>Per distribuire Survivable Branch Appliance

  - La distribuzione di Survivable Branch Appliance è abilitata dal fornitore di Survivable Branch Appliance tramite un'interfaccia utente Web. Per informazioni sulla distribuzione di Survivable Branch Appliance, vedere la documentazione del fornitore di Survivable Branch Appliance.

</div>

<div>

## <a name="to-deploy-the-survivable-branch-server"></a>Per distribuire Survivable Branch Server

  - Installare Lync Server 2013 in un computer su cui è in esecuzione Windows Server 2008 R2, Windows Server 2012 o Windows Server 2012 R2, come installare qualsiasi altro ruolo del server Lync Server 2013.
    
    <div>
    

    > [!NOTE]
    > Per informazioni sull'installazione di Lync Server, vedere <A href="lync-server-2013-deploying-lync-server.md">Deploying Lync server 2013</A> nella documentazione relativa alla distribuzione.

    
    </div>
