# <a name="deploy-a-survivable-branch-appliance-or-server-with-lync-server-2013---branch-site-task"></a>Distribuire un Survivable Branch Appliance o un Survivable Branch Server con Lync Server 2013 - Attività presso il sito di succursale

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2014-10-28_

Eseguire una delle due procedure descritte in questo argomento nel sito della filiale, dopo aver completato le attività in [distribuzione di un Survivable Branch Appliance o un server con Lync server 2013-attività del sito centrale](lync-server-2013-deploying-a-survivable-branch-appliance-or-server-central-site-tasks.md).

<div>


> [!IMPORTANT]
> Per eseguire questa procedura, è necessario essere un membro del gruppo RTCUniversalSBATechnicians.



</div>

<div>

## <a name="to-deploy-the-survivable-branch-appliance"></a>Per distribuire il Survivable Branch Appliance

  - La distribuzione Survivable Branch Appliance è abilitata dal fornitore Survivable Branch Appliance tramite un'interfaccia utente Web. Per informazioni sulla distribuzione di Survivable Branch Appliance, vedere la documentazione del fornitore di Survivable Branch Appliance.

</div>

<div>

## <a name="to-deploy-the-survivable-branch-server"></a>Per distribuire il Survivable Branch Server

  - Installare Lync Server 2013 in un computer che utilizza Windows Server 2008 R2, Windows Server 2012 o Windows Server 2012 R2, così come si installerebbe qualsiasi altro ruolo del server di Lync Server 2013.
    
    <div>
    

    > [!NOTE]
    > Per informazioni sull'installazione di Lync Server, vedere <A href="lync-server-2013-deploying-lync-server.md">distribuzione di Lync server 2013</A> nella documentazione relativa alla distribuzione.

    
    </div>

**Passaggio successivo**: [configurazione degli utenti per la resilienza del sito di succursale in Lync Server 2013](lync-server-2013-configuring-users-for-branch-site-resiliency.md)

</div>
