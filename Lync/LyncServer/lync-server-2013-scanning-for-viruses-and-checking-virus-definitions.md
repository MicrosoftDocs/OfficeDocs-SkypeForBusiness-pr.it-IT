# <a name="scanning-for-viruses-and-checking-virus-definitions-in-lync-server-2013"></a>Ricerca di virus e controllo delle definizioni dei virus in Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2014-05-01_

È consigliabile installare un prodotto antivirus a livello di messaggistica istantanea. La messaggistica istantanea è un'origine ben nota per diffondere rapidamente virus e software dannoso in un'organizzazione. Microsoft Forefront® Security per Lync Server fornisce l'analisi a più motori con virus, software dannoso, protezione dei file e dei filtri delle parole chiave e integrazione senza problemi con Office Communications Server.

Oltre a Forefront Security per Lync Server, è consigliabile installare anche una soluzione antivirus a livello di file per proteggere il file System del server.

È molto importante mantenere aggiornati i motori dello scanner e le definizioni di virus. La configurazione e il monitoraggio dell'integrità degli aggiornamenti assicurano che le informazioni di analisi più aggiornate vengano utilizzate per proteggere sia Office Communications Server che il file System.

<div>


> [!IMPORTANT]  
> Quando si utilizza un software antivirus di terze parti su un server che esegue Lync Server 2013 e Forefront Security per Lync Server, assicurarsi che le cartelle in cui è installato Forefront Security per Lync Server e Lync Server non vengano analizzate per evitare il danneggiamento. Per l'elenco completo delle esclusioni, vedere <A class=uri href="https://support.microsoft.com/kb/943620">https://support.microsoft.com/kb/943620</A> .
