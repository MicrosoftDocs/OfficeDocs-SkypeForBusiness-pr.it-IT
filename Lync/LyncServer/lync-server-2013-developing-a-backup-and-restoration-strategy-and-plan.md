# <a name="developing-a-backup-and-restoration-strategy-and-plan-for-lync-server-2013"></a>Sviluppo di una strategia di backup e ripristino e piano per Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2013-02-17_

L'efficacia delle operazioni di backup e ripristino di Lync Server dipende dalla strategia e dal piano di backup e ripristino. È necessario stabilire una strategia per il backup e il ripristino di Lync Server che si adatta alla strategia complessiva dell'organizzazione e un piano completo e conciso per il backup dei dati e delle impostazioni e, in caso di interruzione, un piano per il ripristino del servizio.

Per il ripristino di emergenza più efficace di un pool Front-End, usare la topologia di ripristino di emergenza del pool di coppie introdotta in Lync Server 2013. Per altre informazioni, vedere [pianificazione per la disponibilità elevata e il ripristino di emergenza in Lync Server 2013](lync-server-2013-planning-for-high-availability-and-disaster-recovery.md).
