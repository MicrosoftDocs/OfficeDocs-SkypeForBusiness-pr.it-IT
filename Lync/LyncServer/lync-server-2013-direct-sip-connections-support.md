# <a name="direct-sip-connections-support-in-lync-server-2013"></a>Supporto delle connessioni SIP dirette in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-06-29_

Lync Server 2013 supporta l'utilizzo di connessioni SIP dirette per connettere Lync Server 2013 a una delle seguenti operazioni:

  - IP-PBX

  - Gateway PSTN

I Mediation Server in un pool di Lync Server 2013 possono controllare più gateway, Session Border Controller (SBCs) forniti dai provider di servizi di telefonia o una loro combinazione. Inoltre, più Mediation Server del pool possono interagire con un singolo gateway.

È possibile utilizzare Microsoft Unified Communications Open Interoperability Program per l'infrastruttura di telefonia aziendale per trovare gateway PSTN, IP-PBX e servizi di trunking SIP qualificati. Per ulteriori informazioni, vedere il sito Web Microsoft Unified Communications Open Interoperability Program all' [http://go.microsoft.com/fwlink/p/?linkId=203309](http://go.microsoft.com/fwlink/p/?linkid=203309)indirizzo.

Per informazioni dettagliate sulle opzioni di distribuzione e topologia per le connessioni SIP dirette, vedere [Direct SIP Connections in Lync Server 2013](lync-server-2013-direct-sip-connections.md) nella documentazione relativa alla pianificazione.
