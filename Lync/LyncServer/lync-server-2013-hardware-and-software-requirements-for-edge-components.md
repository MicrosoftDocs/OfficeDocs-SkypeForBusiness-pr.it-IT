# <a name="hardware-and-software-requirements-for-edge-components-in-lync-server-2013"></a>Requisiti hardware e software per componenti perimetrali in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2013-03-14_

I requisiti hardware e software per i componenti di Edge includono quelli per i componenti software di comunicazione di Lync Server 2013, inclusi Edge Server e gli amministratori facoltativi. Per informazioni dettagliate sui componenti necessari per supportare l'accesso degli utenti esterni e le topologie supportate, vedere [componenti necessari per l'accesso degli utenti esterni in Lync Server 2013](lync-server-2013-components-required-for-external-user-access.md).

<div>

## <a name="hardware-for-edge-servers-and-directors"></a>Hardware per Edge Server e amministratori

Le raccomandazioni e i requisiti hardware sono definiti nella sezione supporto. Per gli Edge Server e per gli amministratori facoltativi, le configurazioni hardware consigliate sono descritte in dettaglio nell'argomento seguente. In pianificazione vedere [piattaforme hardware server per Lync server 2013](lync-server-2013-server-hardware-platforms.md).

</div>

<div>

## <a name="software-requirements-for-edge-servers-and-directors"></a>Requisiti software per Edge Server e amministratori

I requisiti di sistema operativo per Edge Server e per gli amministratori facoltativi sono coerenti con altri ruoli di Lync Server 2013. I sistemi operativi supportati sono dettagliati nel [supporto del sistema operativo server e strumenti in Lync server 2013](lync-server-2013-server-and-tools-operating-system-support.md).

Altri componenti e requisiti software per Edge Server e per gli amministratori facoltativi sono dettagliati in [requisiti software aggiuntivi per Lync Server 2013](lync-server-2013-additional-software-requirements.md).
