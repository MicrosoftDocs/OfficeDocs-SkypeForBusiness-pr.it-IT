# <a name="hardware-and-software-requirements-for-edge-components-in-lync-server-2013"></a>Requisiti hardware e software per i componenti perimetrali in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2013-03-14_

I requisiti hardware e software per i componenti perimetrali includono quelli per i componenti software di comunicazione di Lync Server 2013, inclusi i server perimetrali e i direttori opzionali. Per informazioni dettagliate sui componenti necessari per supportare l'accesso degli utenti esterni e le topologie supportate, vedere [componenti necessari per l'accesso degli utenti esterni in Lync Server 2013](lync-server-2013-components-required-for-external-user-access.md).

<div>

## <a name="hardware-for-edge-servers-and-directors"></a>Hardware per i server perimetrali e i direttori

Gli elementi consigliati e i requisiti hardware sono definiti nella sezione supportabilità. Per i server perimetrali e per i direttori opzionali, le configurazioni hardware consigliate sono descritte nell'argomento seguente. In pianificazione, vedere [server hardware Platforms for Lync server 2013](lync-server-2013-server-hardware-platforms.md).

</div>

<div>

## <a name="software-requirements-for-edge-servers-and-directors"></a>Requisiti software per server perimetrali e direttori

I requisiti del sistema operativo per i server perimetrali e per i direttori opzionali sono coerenti con gli altri ruoli di Lync Server 2013. I sistemi operativi supportati sono descritti in dettaglio nel [supporto del sistema operativo per server e strumenti in Lync server 2013](lync-server-2013-server-and-tools-operating-system-support.md).

I componenti aggiuntivi e i requisiti software per i server perimetrali e per i direttori opzionali sono descritti in [requisiti software aggiuntivi per Lync Server 2013](lync-server-2013-additional-software-requirements.md).

</div>
