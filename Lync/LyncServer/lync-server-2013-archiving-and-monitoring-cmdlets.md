I cmdlet per l'archiviazione e il monitoraggio consentono agli amministratori di gestire l'archiviazione dei messaggi istantanei e delle sessioni di conferenza; per registrare le informazioni dettagliate sulle chiamate; per monitorare Microsoft Lync Server 2013 utilizzando la qualità dell'esperienza (QoE).

> Per ulteriori informazioni sui cmdlet, vedere il &nbsp; Blog di Windows PowerShell di Lync Server all'indirizzo <A href="https://go.microsoft.com/fwlink/p/?linkid=263432">https://go.microsoft.com/fwlink/p/?linkId=263432</A> . Il contenuto di ogni blog e il relativo URL sono soggetti a modifica senza preavviso.
