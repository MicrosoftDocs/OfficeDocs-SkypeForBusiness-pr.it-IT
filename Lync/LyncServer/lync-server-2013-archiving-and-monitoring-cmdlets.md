# <a name="archiving-and-monitoring-cmdlets-in-lync-server-2013"></a>Archiviazione e monitoraggio di cmdlet in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-10-04_

I cmdlet di archiviazione e monitoraggio consentono agli amministratori di gestire l'archiviazione dei messaggi istantanei e delle sessioni di conferenza; per registrare le informazioni dettagliate sulle chiamate; e per monitorare Microsoft Lync Server 2013 usando la qualità dell'esperienza (QoE).


> [!NOTE]
> Per altre informazioni sui cmdlet, vedere il Blog di Lync&nbsp;server in <A href="http://go.microsoft.com/fwlink/p/?linkid=263432">http://go.microsoft.com/fwlink/p/?linkId=263432</A>Windows PowerShell. Il contenuto di ogni blog e il relativo URL sono soggetti a modifiche senza preavviso.
