# <a name="useragent-view-in-lync-server-2013"></a>Visualizzazione UserAgent in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-10-03_
