# <a name="requirements-for-unified-contact-store-in-lync-server-2013"></a>Requisiti per l'archivio contatti unificato in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-10-01_

L'elenco seguente descrive i requisiti per l'implementazione dell'archivio contatti unificato in Lync Server 2013:

  - È necessario eseguire Microsoft Lync Server 2013 ed Exchange 2013.

  - Gli utenti devono usare Lync 2013 per avviare la migrazione dei contatti da Lync Server 2013 a Exchange 2013.

  - È necessario eseguire la migrazione delle cassette postali degli utenti a Exchange 2013.

  - È necessario avere l'autenticazione da server a server configurata tra Lync Server 2013 ed Exchange 2013.
    
    <div>
    

    > [!NOTE]  
    > Per i requisiti dettagliati sulla configurazione dell'autenticazione tra Lync Server 2013 ed Exchange 2013, vedere <A href="lync-server-2013-managing-server-to-server-authentication-oauth-and-partner-applications.md">gestione dell'autenticazione server-Server (OAuth) e delle applicazioni partner in Lync server 2013</A> nella documentazione Operations.
