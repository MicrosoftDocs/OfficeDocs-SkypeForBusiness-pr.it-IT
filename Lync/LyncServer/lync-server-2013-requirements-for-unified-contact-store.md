# <a name="requirements-for-unified-contact-store-in-lync-server-2013"></a>Requisiti per l'archivio contatti unificato in Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-10-01_

Nell'elenco seguente vengono descritti i requisiti per l'implementazione dell'archivio contatti unificato in Lync Server 2013:

  - È necessario eseguire Microsoft Lync Server 2013 ed Exchange 2013.

  - Gli utenti devono utilizzare Lync 2013 per avviare la migrazione dei contatti da Lync Server 2013 a Exchange 2013.

  - Le cassette postali degli utenti devono essere migrate a Exchange 2013.

  - È necessario disporre dell'autenticazione da server a server configurata tra Lync Server 2013 ed Exchange 2013.
    
    <div>
    

    > [!NOTE]  
    > Per informazioni dettagliate sulla configurazione dell'autenticazione tra Lync Server 2013 e Exchange 2013, vedere <A href="lync-server-2013-managing-server-to-server-authentication-oauth-and-partner-applications.md">Managing server-to-Server Authentication (OAuth) and partner Applications in Lync server 2013</A> nella documentazione relativa alle operazioni.
