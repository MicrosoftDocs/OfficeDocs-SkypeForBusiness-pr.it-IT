_**Ultimo argomento modificato:** 2012-06-26_

Le route vocali contengono istruzioni che indicano a Microsoft Lync Server 2013 come instradare le chiamate dagli utenti di VoIP aziendale ai numeri di telefono della rete PSTN (Public Switched Telephone Network) o a un PBX (Private Branch Exchange).
