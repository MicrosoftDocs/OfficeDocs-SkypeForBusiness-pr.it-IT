# <a name="backing-up-lync-server-2013"></a>Backup di Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2013-02-17_

Le procedure descritte in questa sezione illustrano come eseguire il backup di Lync Server in modo da poter recuperare il servizio in caso di interruzione o errore.

È consigliabile sviluppare una strategia di backup e ripristino e pianificare l'organizzazione, come descritto in [sviluppo di una strategia di backup e ripristino e di un piano per Lync Server 2013](lync-server-2013-developing-a-backup-and-restoration-strategy-and-plan.md). Questa strategia e il piano devono includere le procedure specifiche che si prevede di usare. Usare le procedure descritte negli argomenti di questa sezione, insieme ai fogli di lavoro nei fogli di lavoro di [backup e ripristino per Lync Server 2013](lync-server-2013-backup-and-restoration-worksheets.md), per documentare come si prevede di eseguire il backup di una specifica distribuzione di Lync Server.
