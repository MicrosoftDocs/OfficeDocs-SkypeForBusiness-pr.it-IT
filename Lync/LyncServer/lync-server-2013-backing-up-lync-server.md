# <a name="backing-up-lync-server-2013"></a>Backup di Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2013-02-17_

Nelle procedure descritte in questa sezione viene descritto come eseguire il backup di Lync Server, in modo che sia possibile ripristinare il servizio in caso di interruzione o errore.

È consigliabile sviluppare una strategia di backup e ripristino e pianificare la propria organizzazione, come descritto in [sviluppo di una strategia di backup e ripristino e pianificare Lync Server 2013](lync-server-2013-developing-a-backup-and-restoration-strategy-and-plan.md). Questa strategia e il piano devono includere le procedure specifiche che si intende utilizzare. Utilizzare le procedure incluse negli argomenti di questa sezione, insieme ai fogli di lavoro nei fogli di lavoro [per il backup e il ripristino di Lync server 2013](lync-server-2013-backup-and-restoration-worksheets.md), per documentare il modo in cui si prevede di eseguire il backup di una distribuzione di Lync Server specifica.
