# <a name="mobility-support-in-lync-server-2013"></a>Supporto per dispositivi mobili in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-06-29_

Con Lync Server 2013 Mobility, gli utenti possono accedere alla funzionalità Lync 2013 sui propri dispositivi mobili. I dispositivi mobili supportati sono:

  - Dispositivi Apple iOS supportati

  - Windows Phone

  - Android

  - Nokia

Per informazioni dettagliate sulle caratteristiche e i requisiti per dispositivi mobili, vedere [Planning for Mobility in Lync Server 2013](lync-server-2013-planning-for-mobility.md) nella documentazione relativa alla pianificazione.
