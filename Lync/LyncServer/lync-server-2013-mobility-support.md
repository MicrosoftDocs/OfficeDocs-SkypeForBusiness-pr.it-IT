# <a name="mobility-support-in-lync-server-2013"></a>Supporto per i dispositivi mobili in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-06-29_

Con la mobilità di Lync Server 2013, gli utenti hanno accesso alla funzionalità Lync 2013 nei dispositivi mobili. I dispositivi mobili supportati includono i seguenti:

  - Dispositivi iOS Apple supportati

  - Windows Phone

  - Android

  - Nokia

Per informazioni dettagliate sulle caratteristiche e i requisiti per la mobilità, vedere [pianificazione della mobilità in Lync Server 2013](lync-server-2013-planning-for-mobility.md) nella documentazione relativa alla pianificazione.
