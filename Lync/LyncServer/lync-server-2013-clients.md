# <a name="clients-for-lync-server-2013"></a>Client per Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2014-01-23_

Lync Server 2013 supporta diversi tipi di software client che è possibile distribuire agli utenti dell'organizzazione, incluso il software client installato nel computer, i client basati sul Web e i dispositivi mobili. In questo argomento vengono illustrati i client che è possibile usare. Per un confronto dettagliato delle caratteristiche fornite dai client di Lync Server 2013, vedere [tabelle di confronto dei client per Lync server 2013](lync-server-2013-desktop-client-comparison-tables.md).

<div>

## <a name="lync-2013"></a>Lync 2013

Lync 2013 è il client completo per Lync Server. L'interfaccia utente di Lync 2013 è stata completamente riprogettata e include nuove funzionalità integrate, ad esempio la chat persistente (Lync 2010 aveva un client separato per la funzionalità di chat), le conversazioni a schede, l'anteprima video e il video a più parti. Per un riepilogo delle modifiche, vedere [novità per i client in Lync Server 2013](lync-server-2013-what-s-new-for-clients.md).

La configurazione client di Lync 2013 fa parte del programma di installazione di Office nel supporto di installazione.

</div>

<div>

## <a name="online-meeting-add-in-for-lync-2013"></a>Componente aggiuntivo riunione online per Lync 2013

Il componente aggiuntivo riunione online per Lync 2013 supporta la gestione delle riunioni dall'interno del client di messaggistica e collaborazione di Microsoft Outlook. Il componente aggiuntivo riunione online per il software Lync 2013 viene installato automaticamente con Lync 2013.

</div>

<div>

## <a name="lync-web-scheduler"></a>Utilità di pianificazione Web di Lync

Lync Web Scheduler è uno strumento di gestione e pianificazione delle riunioni basato sul Web per gli utenti che non hanno accesso a Microsoft Outlook o che si trovano in un sistema operativo non basato su Windows. Con Lync Web Scheduler gli utenti possono creare nuove riunioni, modificare le riunioni esistenti e inviare inviti tramite il programma di posta elettronica preferito.

</div>

<div>

## <a name="lync-web-app"></a>Lync Web App

Lync Web App è il client di conferenza basato sul Web per le riunioni di Lync Server 2013. In questa versione, l'aggiunta di audio e video per computer a Lync Web App offre un'esperienza di riunione completa per tutti coloro che non hanno un client Lync installato localmente. I partecipanti alla riunione hanno accesso a tutte le caratteristiche di collaborazione e condivisione e ai controlli della riunione relatore.

Se Lync 2013 non è installato nel computer di un utente e l'utente fa clic su un collegamento a una riunione in una convocazione di riunione, viene aperta Lync Web App. È anche possibile configurare la pagina di partecipazione alla riunione per consentire agli utenti di partecipare alle riunioni usando versioni precedenti dei client. vedere [configurazione della pagina di partecipazione alla riunione in Lync Server 2013](lync-server-2013-configuring-the-meeting-join-page.md) nella documentazione relativa alla distribuzione.

A causa dei miglioramenti apportati a Lync Web App, una versione aggiornata di Microsoft Lync 2010 Attendeee non è disponibile per Lync Server 2013. Lync Web App è il client preferito per i partecipanti esterni all'organizzazione. Con Lync Web App non è necessaria un'installazione client locale, anche se le funzionalità audio, video e condivisione richiedono l'installazione di un plug-in durante il primo utilizzo.

</div>

<div>

## <a name="lync-2013-basic"></a>Lync 2013 Basic

Lync 2013 Basic è un client scaricabile per i clienti con una distribuzione locale di Lync Server 2013 con licenza e i clienti che si abbonano a un piano di Microsoft Office 365 che non include il client completo di Lync 2013. Il client Lync Basic include presenza avanzata, contatti, messaggistica istantanea, riunioni Lync e funzionalità vocali di base. Le funzionalità non supportate in Lync Basic includono il video multiparte, l'integrazione di OneNote, il supporto VDI (Virtual Desktop Infrastructure), la ricerca di competenze, la registrazione, le funzionalità vocali aziendali e la gestione delle chiamate avanzate, ad esempio l'inoltro di chiamata e la chiamata del team. Per informazioni dettagliate, vedere [tabelle di confronto dei client per Lync Server 2013](lync-server-2013-desktop-client-comparison-tables.md).

</div>

<div>

## <a name="lync-windows-store-app"></a>App Lync di Windows Store

L'app Lync di Windows Store è un'app Lync ottimizzata per il tocco progettata specificamente per Windows 8,1, Windows 8 e Windows RT. Gli utenti possono scaricare l'app tramite Windows Store cercando "Lync". Per altre informazioni, vedere [tabelle di confronto dei client per Lync server 2013](lync-server-2013-desktop-client-comparison-tables.md), [requisiti delle app Lync di Windows Store per Lync Server 2013](lync-server-2013-lync-windows-store-app-requirements.md)e [distribuzione dell'app Lync di windows Store in Lync Server 2013](lync-server-2013-deploying-lync-windows-store-app.md).

</div>

<div>

## <a name="lync-2013-for-mobile-devices"></a>Lync 2013 per dispositivi mobili

Le app per dispositivi mobili Lync 2013 ora includono funzionalità VoIP (Voice over IP) e video over IP, oltre a contatti, presenza e funzionalità di messaggistica istantanea. Gli utenti di dispositivi mobili possono scegliere di comunicare con altri tramite messaggistica istantanea, chiamate vocali o videochiamate tramite Wi-Fi o la connessione dati del cellulare. Con un solo clic del collegamento alla riunione in un elemento del calendario, gli utenti di dispositivi mobili possono partecipare a riunioni vocali e video. Per altre informazioni sulle app per dispositivi mobili Lync 2013, vedere [pianificazione per i client mobili in Lync Server 2013](lync-server-2013-planning-for-mobile-clients.md).

</div>

<div>

## <a name="supported-clients-from-previous-releases"></a>Client supportati da versioni precedenti

Lync Server 2013 supporta i client seguenti dalle versioni precedenti del server. Quando partecipano alle riunioni, è possibile rendere disponibili agli utenti determinati client precedenti. Per informazioni dettagliate, vedere [configurazione della pagina di partecipazione alla riunione in Lync Server 2013](lync-server-2013-configuring-the-meeting-join-page.md) nella documentazione relativa alla distribuzione.

  - **Lync 2010**   Lync 2010 offre un'esperienza desktop completa, inclusi i messaggi istantanei, la presenza avanzata, la voce, il video, la condivisione e la telefonia. Tuttavia, nessuna delle nuove funzionalità introdotte in Lync Server 2013 sarà disponibile finché il client dell'utente non viene aggiornato a Lync 2013.

  - **Lync 2010 Mobile**   Lync Server 2013 supporta tutte le app di Microsoft Lync 2010 per dispositivi mobili. Microsoft Lync 2010 Mobile offre messaggistica istantanea, presenza avanzata e telefonia per gli utenti dell'organizzazione che si connettono da uno smartphone o da un telefono che esegue una versione professionale di Windows Mobile. È possibile indicare agli utenti di installare Microsoft Lync 2010 Mobile indirizzando il dispositivo al Marketplace dell'app per il proprio cellulare. Per informazioni dettagliate, vedere la pagina relativa alla pianificazione per i client mobili nella documentazione di [http://go.microsoft.com/fwlink/p/?LinkID=235955](http://go.microsoft.com/fwlink/p/?linkid=235955)Lync Server 2010.

  - ****   Il software Lync Phone Edition Lync Phone Edition per i telefoni IP intelligenti, ad esempio i telefoni collegati tramite USB, non è stato aggiornato per Lync Server 2013. Lync Phone Edition continua a essere supportato per l'immissione e la ricezione di chiamate, presenza avanzata e funzionalità audio client per le conferenze.

  - **Assistente di Lync 2010**   il programma di gestione delle chiamate integrato di Microsoft Lync 2010 consente a un receptionist di gestire più conversazioni contemporaneamente tramite la gestione delle chiamate rapide, la messaggistica istantanea e il routing su schermo.

</div>
