# <a name="modify-a-trunk-in-topology-builder-in-lync-server-2013"></a>Modificare un trunk in Generatore di topologia in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-09-21_

Seguire questa procedura per modificare l'indirizzo IP multimediale alternativo e l'identificatore alternativo di bypass di un trunk.

<div>

## <a name="to-modify-the-alternate-media-ip-address-of-a-trunk"></a>Per modificare l'indirizzo IP multimediale alternativo di un trunk

1.  Avviare Lync Server Management Shell: fare clic sul pulsante **Start**, scegliere **tutti i programmi**, **Microsoft Lync Server 2013**e quindi fare clic su **Lync Server Management Shell**.

2.  Eseguire il cmdlet Set-CsPstnGateway e modificare il campo AlternateBypassId in Lync Server Management Shell.
    
        Set-CsPstnGateway -Identity "PstnGateway:<peer FQDN> -RepresentativeMediaIP <IP address>

</div>

<div>

## <a name="to-modify-the-alternate-bypassid-of-a-trunk"></a>Per modificare il BypassID alternativo di un trunk

1.  Avviare Lync Server Management Shell: fare clic sul pulsante **Start**, scegliere **tutti i programmi**, **Microsoft Lync Server 2013**e quindi fare clic su **Lync Server Management Shell**.

2.  Eseguire il cmdlet Set-CsPstnGateway e modificare il campo AlternateBypassId in Lync Server Management Shell.
    
        Set-CsPstnGateway -Identity "PstnGateway:<peer FQDN> -AlternateBypassID <identifier>

</div>
