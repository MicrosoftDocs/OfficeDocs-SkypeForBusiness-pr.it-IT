# <a name="managing-lync-server-2013-archiving"></a>Gestione dell'archiviazione in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-10-10_

Quando si distribuisce l'archiviazione per l'organizzazione, è necessario specificare la configurazione iniziale durante la distribuzione. In alcuni casi, tuttavia, può essere necessario modificare la modalità di implementazione del supporto dell'archiviazione per la gestione giornaliera o per soddisfare i nuovi requisiti dell'organizzazione. Ad esempio, potrebbe essere necessario configurare il supporto dell'archiviazione in modo diverso per un sito, un pool o un utente specifico all'interno dell'organizzazione. Per gli utenti residenti in Lync Server 2013, è possibile creare e personalizzare i criteri di archiviazione e le configurazioni. Se si usa l'integrazione di Microsoft Exchange, è anche necessario configurare le impostazioni di Exchange 2013. Questa sezione fornisce informazioni e procedure per consentire di apportare modifiche alla distribuzione di archiviazione.
