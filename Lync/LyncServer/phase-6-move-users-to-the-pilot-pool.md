# <a name="phase-6-move-users-to-the-pilot-pool"></a>Fase 6: spostare gli utenti nel pool pilota

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-10-16_

È possibile spostare un singolo utente o gruppi di utenti nella nuova distribuzione di Lync Server 2013 utilizzando i due metodi seguenti: Lync Server Control Panel e Lync Server Management Shell. Negli argomenti di questa sezione vengono descritte le attività che è necessario eseguire durante la distribuzione pilota, nonché prima di spostare la distribuzione di Lync Server 2013 da una distribuzione pilota a una distribuzione a livello di produzione.
