# <a name="deploying-mobility-in-lync-server-2013"></a>Distribuzione delle funzionalità per dispositivi mobili in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-09-08_

Quando si distribuisce la funzionalità di mobilità di Lync Server 2013, gli utenti mobili possono usare dispositivi mobili supportati per le funzionalità di Lync, ad esempio messaggistica istantanea, presenza e contatti.

Per informazioni dettagliate sui requisiti per la distribuzione della funzionalità di mobilità, vedere [pianificazione della mobilità in Lync Server 2013](lync-server-2013-planning-for-mobility.md).

Questa sezione illustra i passaggi per la distribuzione e la verifica delle funzionalità per la mobilità e l'individuazione automatica.
