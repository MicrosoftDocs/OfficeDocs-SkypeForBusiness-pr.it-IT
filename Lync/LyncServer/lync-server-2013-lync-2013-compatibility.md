## <a name="public-instant-messaging-clients-and-lync-2013"></a>Client di messaggistica istantanea pubblici e Lync 2013

Se il server è stato configurato per la connettività di messaggistica istantanea pubblica, Lync supporta le funzionalità seguenti con le reti di messaggistica istantanea pubbliche. Lo stato presenza viene filtrato per gli Stati di presenza supportati dal client di messaggistica istantanea pubblica. Per informazioni dettagliate, vedere [pianificazione della connettività di messaggistica istantanea pubblica in Lync server 2013](lync-server-2013-planning-for-public-instant-messaging-connectivity.md) nella documentazione relativa alla pianificazione e [gestione dei criteri di accesso esterno in Lync Server 2013](lync-server-2013-manage-external-access-policy-for-your-organization.md) nella documentazione delle operazioni.

Inoltre, la funzionalità di integrazione XMPP di Lync Server 2013 consente agli utenti di scambiare messaggi istantanei e informazioni sulla presenza con gli utenti di provider di messaggistica istantanea pubblici che usano il protocollo di messaggistica e presenza estensibile, ad esempio Google Talk. Per informazioni dettagliate, vedere [pianificazione della Federazione XMPP (Extensible Messaging and Presence Protocol) in Lync Server 2013](lync-server-2013-planning-for-extensible-messaging-and-presence-protocol-xmpp-federation.md) nella documentazione relativa alla pianificazione.
