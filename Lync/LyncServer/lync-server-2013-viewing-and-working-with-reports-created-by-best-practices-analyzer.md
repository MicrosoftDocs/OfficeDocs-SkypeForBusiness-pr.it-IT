# <a name="viewing-and-working-with-reports-created-by-best-practices-analyzer-in-lync-server-2013"></a>Visualizzazione e utilizzo dei report creati da Best Practices Analyzer in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-06-14_

Quando si usa Best Practices Analyzer per analizzare l'ambiente, Best Practices Analyzer crea report che identificano i problemi e altre informazioni sulla distribuzione. È possibile usare Best Practices Analyzer per visualizzare i report che contengono i risultati della scansione e comprendere i problemi identificati nei report.
