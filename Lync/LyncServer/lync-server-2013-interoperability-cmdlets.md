# <a name="interoperability-cmdlets-in-lync-server-2013"></a>Cmdlet di interoperabilità in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-06-27_

I cmdlet di interoperabilità vengono usati per configurare l'autenticazione da server a server e l'autorizzazione tra Microsoft Lync Server 2013 e altri prodotti server, ad esempio Microsoft Exchange Server 2013. L'autenticazione e l'autorizzazione da server a server consentono a questi server di scambiare e condividere dati senza problemi.

<div>
