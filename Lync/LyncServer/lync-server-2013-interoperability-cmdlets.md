_**Ultimo argomento modificato:** 2012-06-27_

I cmdlet di interoperabilità vengono utilizzati per configurare l'autenticazione da server a server e l'autorizzazione tra Microsoft Lync Server 2013 e altri prodotti server, ad esempio Microsoft Exchange Server 2013. Con l'autenticazione e l'autorizzazione server-server i server riescono a scambiarsi e a condividere i dati facilmente.
