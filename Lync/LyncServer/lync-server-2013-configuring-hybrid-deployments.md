<div id="mainBody">

<div>

## <a name="in-this-section"></a>Argomenti della sezione

  - Se si dispone di una distribuzione Lync locale e si desidera spostare alcuni utenti nel cloud per l'utilizzo di Lync online in Office 365, vedere [Configuring an on-premises Deployment for Hybrid with Lync Online](lync-server-2013-configuring-an-on-premises-deployment-for-hybrid-with-lync-online.md).

  - Se gli utenti sono abilitati per Lync in Lync Online, ma non sono stati abilitati in una distribuzione locale, vedere [Moving users from Lync Online to Lync on-premises in Lync Server 2013](lync-server-2013-moving-users-from-lync-online-to-lync-on-premises.md).
