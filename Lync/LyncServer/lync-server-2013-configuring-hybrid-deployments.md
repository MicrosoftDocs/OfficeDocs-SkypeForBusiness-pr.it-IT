</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2016-05-26_

Questa sezione descrive i passaggi necessari per la configurazione delle distribuzioni ibride di Lync Server 2013.

<div>

## <a name="in-this-section"></a>Contenuto della sezione

  - Se si dispone di una distribuzione Lync locale e si vuole trasferire alcuni utenti nel cloud per usare Lync online in Office 365, vedere [configurazione di una distribuzione locale ibrida con Lync Online](lync-server-2013-configuring-an-on-premises-deployment-for-hybrid-with-lync-online.md).

  - Se gli utenti sono abilitati per Lync in Lync Online, ma che non sono stati abilitati in una distribuzione locale, vedere [spostamento di utenti da Lync Online a Lync locale in Lync Server 2013](lync-server-2013-moving-users-from-lync-online-to-lync-on-premises.md).
