# <a name="topology-cmdlets-jn-lync-server-2013"></a>Cmdlet di topologia JN Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-06-20_

Molti cmdlet di topologia inclusi in Microsoft Lync Server 2013 sono progettati per l'uso con configurazione e generatore di topologie. a causa di questo, esistono diversi cmdlet di topologia che gli amministratori chiameranno raramente direttamente. Ci saranno tuttavia momenti in cui gli amministratori dovranno usare questi cmdlet. Dopo aver creato nuovi account Kerberos, ad esempio, devi eseguire il cmdlet [Enable-CsTopology](https://technet.microsoft.com/en-us/library/Gg398398(v=OCS.15)) per rendere effettive le modifiche. Inoltre, gli amministratori possono eseguire i cmdlet, ad esempio [Test-CsTopology](https://technet.microsoft.com/en-us/library/Gg398127(v=OCS.15)) e [Test-CsComputer](https://technet.microsoft.com/en-us/library/Gg398162(v=OCS.15)) , per garantire che Lync Server 2013 sia stato installato correttamente e funzioni come previsto.
