Prima di eseguire la migrazione a Lync Server 2013, gli aggiornamenti devono essere applicati all'ambiente Office Communications Server 2007 R2. Per informazioni più aggiornate su Office Communications Server 2007 R2, vedere **aggiornamenti per** **Communications Server 2007 R2** at [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802).

Per installare gli aggiornamenti per Office Communications Server 2007 R2, è consigliabile seguire la procedura di **installazione cumulativa del server di aggiornamento del metodo 1** descritta nell'articolo 968802 della Microsoft Knowledge Base "aggiornamenti per Communications Server [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802)2007 R2".
