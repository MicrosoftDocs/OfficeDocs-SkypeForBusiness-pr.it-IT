# <a name="trusted-applications-cmdlets-in-lync-server-2013"></a>Cmdlet per le applicazioni attendibili in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-06-20_

Un'applicazione attendibile è un'applicazione sviluppata da terze parti a cui viene assegnato lo stato attendibile per l'esecuzione come parte di Microsoft Lync Server 2013 ma che non è una parte incorporata del prodotto. Lync Server 2013 fornisce cmdlet che possono essere utilizzati per configurare e gestire le applicazioni attendibili.

<div>
