# <a name="microsoft-lync-server-2013"></a>Microsoft Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2014-03-14_

Il software di comunicazione di Microsoft Lync Server 2013 offre soluzioni di messaggistica istantanea, presenza, conferenza e telefonia in grado di supportare i requisiti di collaborazione a livello aziendale.

Si vuole scaricare una copia della documentazione di Lync Server 2013? È possibile scaricare il file della Guida compilato **LyncServer2013\_ITPro** (con [http://go.microsoft.com/fwlink/p/?LinkID=331103](http://go.microsoft.com/fwlink/p/?linkid=331103)estensione chm) nell'area download Microsoft.

Nelle tabelle seguenti sono elencati gli argomenti della raccolta documenti di Lync Server 2013.
