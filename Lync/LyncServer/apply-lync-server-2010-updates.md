</div>

<div id="mainSection">

<div id="mainBody">

_**Ultimo argomento modificato:** 2012-10-19_

Prima di eseguire la migrazione a Lync Server 2013, è necessario che gli aggiornamenti vengano applicati all'ambiente Lync Server 2010. Per le informazioni più aggiornate su Lync Server 2010, vedere **Updates Resource Center for Lync** at <http://go.microsoft.com/fwlink/p/?linkid=232630>.

Per installare gli aggiornamenti per Lync Server 2010, è consigliabile seguire la procedura del **programma di installazione dell'aggiornamento cumulativo del metodo 1** descritta nell'articolo della Microsoft Knowledge Base "Updates for lync server [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=2493736](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=2493736)2010" all'indirizzo.
