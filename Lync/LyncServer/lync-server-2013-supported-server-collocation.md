# <a name="supported-server-collocation-in-lync-server-2013"></a>Collocazione dei server supportata in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-06-29_

Lync Server 2013 supporta la collocazione di alcuni ruoli e funzionalità del server. I ruoli del server che è possibile collocare dipendono in parte dal fatto che si stia distribuendo un pool Front End o un server Standard Edition.
