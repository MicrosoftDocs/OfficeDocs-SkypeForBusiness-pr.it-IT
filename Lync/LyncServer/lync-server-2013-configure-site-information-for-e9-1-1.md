# <a name="configure-site-information-for-e9-1-1-in-lync-server-2013"></a>Configurare le informazioni sul sito per E9-1-1 in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-10-03_

Per definire un criterio di posizione per le subnet, è necessario eseguire le operazioni seguenti in qualsiasi ordine:

  - Applicare un criterio di posizione al sito di rete.

  - Aggiungere le subnet al sito di rete.

Per informazioni dettagliate sui siti di rete, vedere [siti di Lync Server per Lync server 2013](lync-server-2013-sites.md).

<div>


> [!NOTE]  
> Se si creano siti di rete per un'altra funzionalità vocale e si vuole configurare E9-1-1 usando gli stessi siti, è possibile modificare i siti da usare per E9-1-1.
