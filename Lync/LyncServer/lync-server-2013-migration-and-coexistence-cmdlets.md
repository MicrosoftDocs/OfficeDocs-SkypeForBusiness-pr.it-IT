_**Argomento Ultima modifica:** 2012-06-26_

Il cmdlet [Move-CsLegacyUser](https://technet.microsoft.com/en-us/library/Gg413025(v=OCS.15)) offre la possibilità di trasferire gli account utente da Office Communications Server 2007 o da Microsoft lync Server 2010 a Microsoft lync Server 2013. Se è necessario trasferire un account utente "indietro", ad esempio da Microsoft Lync Server 2013 a Microsoft Lync Server 2010, usare il cmdlet [Move-CsUser](https://technet.microsoft.com/en-us/library/Gg398528(v=OCS.15)) .
