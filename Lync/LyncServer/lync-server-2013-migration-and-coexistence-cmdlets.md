</div>

_**Ultimo argomento modificato:** 2012-06-26_

Il cmdlet [Move-CsLegacyUser](https://technet.microsoft.com/library/Gg413025(v=OCS.15)) consente di spostare gli account utente da Office Communications Server 2007 o Microsoft lync Server 2010 a Microsoft lync Server 2013. Se è necessario spostare un account utente "indietro" (ad esempio, da Microsoft Lync Server 2013 a Microsoft Lync Server 2010), utilizzare il cmdlet [Move-CsUser](https://technet.microsoft.com/library/Gg398528(v=OCS.15)) .
