# <a name="new-unified-contact-store-feature-in-lync-server-2013"></a>Nuova funzionalità archivio contatti unificato in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-09-13_

Lync Server 2013 introduce l'archivio contatti unificato. L'archivio contatti unificato consente agli utenti di conservare tutte le informazioni di contatto in Microsoft Exchange Server 2013. Dopo la migrazione dei contatti Lync degli utenti a Exchange 2013, gli utenti possono accedere ai propri contatti e gestirli da Lync 2013, Outlook o Outlook Web App e i Preferiti rimarranno sincronizzati. Ad esempio, se un utente aggiunge un contatto ai Preferiti in Outlook, il contatto viene visualizzato nel gruppo Preferiti in Lync 2013. Gli utenti non devono essere connessi a Lync per gestire i contatti da Outlook o Outlook Web App.

L'archivio contatti unificato è abilitato per impostazione predefinita. È possibile abilitare o disabilitare gli utenti per l'archivio contatti unificato a livello globale, per sito, per tenant o per singoli o gruppi di persone.
