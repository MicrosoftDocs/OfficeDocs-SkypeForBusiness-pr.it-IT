# <a name="new-unified-contact-store-feature-in-lync-server-2013"></a>Nuova funzionalità archivio contatti unificato in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-09-13_

Lync Server 2013 introduce l'archivio contatti unificato. Archivio contatti unificato consente agli utenti di mantenere tutte le informazioni di contatto in Microsoft Exchange Server 2013. Dopo la migrazione dei contatti Lync degli utenti a Exchange 2013, gli utenti possono accedere e gestire i propri contatti da Lync 2013, Outlook o Outlook Web App e i preferiti rimangono sincronizzati. Ad esempio, se un utente aggiunge un contatto ai Preferiti in Outlook, il contatto viene visualizzato nel gruppo Preferiti in Lync 2013. Non è necessario che gli utenti accedano a Lync per gestire i propri contatti da Outlook o Outlook Web App.

L'archivio contatti unificato è abilitato per impostazione predefinita. È possibile abilitare o disabilitare gli utenti per l'archivio contatti unificato a livello globale, per sito, per tenant o per singoli o gruppi di persone.
