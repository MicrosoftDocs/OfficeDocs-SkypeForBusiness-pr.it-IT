# <a name="install-lync-server-administration-tools"></a>Installare gli strumenti di amministrazione di Lync Server

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-10-02_

In questo argomento viene descritto come installare gli strumenti di amministrazione necessari per la distribuzione e la gestione di Lync Server 2013. È inoltre possibile installarli in altri computer, ad esempio in console amministrative dedicate.

<div>

## <a name="to-install-the-lync-server-2013-administrative-tools"></a>Per installare gli strumenti di amministrazione di Lync Server 2013

1.  Nel server front end di Lync Server 2013 eseguire il programma di installazione di Lync Server 2013.

2.  Nella pagina distribuzione guidata di Lync Server 2013 selezionare **installa strumenti di amministrazione**
    
    ![Distribuzione guidata di Lync Server 2013, pagina di benvenuto](images/JJ205265.5f88ae18-9c3c-42ea-a91a-836ecf5d515f(OCS.15).jpg "Distribuzione guidata di Lync Server 2013, pagina di benvenuto")
