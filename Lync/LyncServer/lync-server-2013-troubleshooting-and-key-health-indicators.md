## <a name="in-this-section"></a>Argomenti della sezione

Per soddisfare gli SLA dell'architettura di riferimento e garantire una transizione agevole ai team di supporto, è necessario definire un approccio di risoluzione dei problemi comune insieme a un insieme obbligatorio di strumenti e approcci per la risoluzione dei problemi, come definito nella [Guida alla rete](http://go.microsoft.com/fwlink/p/?linkid=390677) di Lync Server.

È consigliabile utilizzare System Center Operations Manager per monitorare l'integrità del sistema Lync Server 2013. Inoltre, fare riferimento alla discussione di KHIs nella [Guida alla rete](http://go.microsoft.com/fwlink/p/?linkid=390677) di lync Server 2013 e nel foglio di calcolo di Excel per l'utilizzo con Lync 2013.
