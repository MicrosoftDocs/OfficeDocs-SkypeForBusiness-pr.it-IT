# <a name="monitoring-starting-and-stopping-the-persistent-chat-services-in-lync-server-2013"></a>Monitoraggio, avvio e arresto dei servizi di chat persistente in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-10-06_

I servizi di chat persistente e i servizi di conformità di Persistent Chat fanno parte della topologia di Lync Server 2013 e pertanto possono essere monitorati, interrotti e avviati utilizzando i cmdlet di Windows PowerShell, **Get-CsWindowsService**, **Stop-CsWindowsService**e **Start-CsWindowsService**.
