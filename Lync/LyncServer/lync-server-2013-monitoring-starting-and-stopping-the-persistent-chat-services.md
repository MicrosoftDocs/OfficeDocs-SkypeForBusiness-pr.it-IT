# <a name="monitoring-starting-and-stopping-the-persistent-chat-services-in-lync-server-2013"></a>Monitoraggio, avvio e arresto dei servizi di chat persistente in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-10-06_

I servizi di chat persistenti e i servizi di conformità della chat persistente fanno parte della topologia di Lync Server 2013 e possono quindi essere monitorati, fermati e avviati usando i cmdlet di Windows PowerShell, **Get-CsWindowsService**, **Stop-CsWindowsService**e **Start-CsWindowsService**, rispettivamente.
