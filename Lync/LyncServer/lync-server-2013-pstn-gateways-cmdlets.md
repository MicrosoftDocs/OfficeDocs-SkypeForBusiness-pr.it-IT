# <a name="pstn-gateways-cmdlets-in-lync-server-2013"></a>Cmdlet di gateway PSTN in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-03-21_

I gateway PSTN consentono agli utenti di VoIP aziendale di effettuare telefonate e ricevere telefonate da persone della rete PSTN, ovvero la rete telefonica pubblica commutata. Questi gateway fungono da Bridge tra il Mediation Server e la rete PSTN.

<div>

## <a name="pstn-gateways-cmdlets"></a>Cmdlet per gateway PSTN

I cmdlet [Test-CsPstnOutboundCall](https://technet.microsoft.com/en-us/library/Gg398207(v=OCS.15)) e [Test-CsPstnPeerToPeerCall](https://technet.microsoft.com/en-us/library/Gg398662(v=OCS.15)) consentono di verificare che gli utenti siano in grado di effettuare chiamate tramite la rete PSTN.
