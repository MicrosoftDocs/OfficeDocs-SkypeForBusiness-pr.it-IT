# <a name="monitoring-lync-server-2013-with-system-center-operations-manager"></a>Monitoraggio di Lync Server 2013 con System Center Operations Manager

</div>

<span> </span>

_**Ultimo argomento modificato:** 2015-05-06_

Lync Server Management Pack (MP) è la soluzione di monitoraggio scelta per il monitoraggio di qualsiasi distribuzione di Lync Server.

MP implementa il registro eventi tradizionale e la strumentazione basata su contatori delle prestazioni e consente la strumentazione appena disponibile in Lync Server, ad esempio gli eventi di coppia (esito negativo/esito positivo) per diversi indicatori di integrità chiave e implementa anche completamente le nuove transazioni sintetiche (cmdlet di Windows PowerShell di test-cs \* ).

È possibile trovare il Lync Server 2013 Management Pack e la relativa documentazione relativa all'indirizzo [https://go.microsoft.com/fwlink/p/?LinkId=400468](https://go.microsoft.com/fwlink/p/?linkid=400468) . Questa opzione è consigliata se si esegue System Center Operations Manager 2012.
