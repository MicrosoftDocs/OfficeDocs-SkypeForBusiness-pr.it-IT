# <a name="monitoring-lync-server-2013-with-system-center-operations-manager"></a>Monitoraggio di Lync Server 2013 con System Center Operations Manager

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2015-05-06_

Lync Server Management Pack (MP) è la soluzione di monitoraggio scelta per il monitoraggio di qualsiasi distribuzione di Lync Server.

L'MP implementa il registro eventi tradizionale e la strumentazione basata su contatori delle prestazioni e consente di disporre di una nuova strumentazione disponibile in Lync Server, ad esempio eventi di coppia (errore/successo) per diversi indicatori di integrità chiave e implementa completamente le\* nuove transazioni sintetiche (cmdlet di Windows PowerShell per test-CS).

È possibile trovare il Management Pack di Lync Server 2013 e la relativa documentazione [http://go.microsoft.com/fwlink/p/?LinkId=400468](http://go.microsoft.com/fwlink/p/?linkid=400468)relativa. Questa operazione è consigliata se si esegue System Center Operations Manager 2012.
