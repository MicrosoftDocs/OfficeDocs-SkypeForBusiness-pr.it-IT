# <a name="setting-up-policies-for-archiving-in-lync-server-2013-when-using-exchange-server-integration"></a>Configurazione dei criteri per l'archiviazione in Lync Server 2013 quando si usa l'integrazione di Exchange Server

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-10-09_

Se gli utenti ospitati in Exchange 2013 hanno inserito le cassette postali in blocco sul posto, i criteri di blocco sul posto di Exchange controllano l'archiviazione per tali utenti. Se si usa l'integrazione di Microsoft Exchange per la distribuzione, i criteri di Exchange 2013 eseguono l'override dei criteri di archiviazione di Lync Server per gli utenti residenti in Exchange 2013. Per informazioni sulla configurazione dei criteri di archiviazione di Exchange, vedere la documentazione di Exchange 2013. Per informazioni dettagliate sulla configurazione dei criteri utente per gli utenti ospitati in Lync Server 2013, vedere [configurazione dei criteri utente per l'archiviazione in Lync server 2013](lync-server-2013-setting-up-user-policies-for-archiving-in-lync-server.md) nella documentazione relativa alla distribuzione. Per informazioni dettagliate sul funzionamento dei criteri, vedere funzionamento dell' [archiviazione in Lync Server 2013](lync-server-2013-how-archiving-works.md) nella documentazione per la pianificazione, la documentazione di distribuzione o le operazioni.

<div>


> [!NOTE]
> Se si distribuisce Exchange 2013 e Lync Server 2013 nella stessa foresta, i criteri di blocco sul posto di Exchange 2013 controllano l'archiviazione. Se si distribuiscono Exchange 2013 e Lync Server 2013 in foreste separate, vedere "distribuzione di Lync Server e Microsoft Exchange in foreste diverse" in <A href="lync-server-2013-deployment-checklist-for-archiving.md">elenco di controllo di distribuzione per l'archiviazione in Lync Server 2013</A>.
