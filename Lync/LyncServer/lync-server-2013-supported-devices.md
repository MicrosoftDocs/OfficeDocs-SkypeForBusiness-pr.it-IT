# <a name="supported-devices-in-lync-server-2013"></a>Dispositivi supportati in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-10-08_

Lync Server supporta tutti i telefoni elencati nella tabella seguente. Tieni presente che non tutti i telefoni supportati possono eseguire Lync Phone Edition (alcuni possono eseguire solo versioni precedenti). Usare la tabella per scoprire quali telefoni possono eseguire Lync Phone Edition e dove ottenere informazioni dettagliate su tutti i telefoni supportati.

<div>


> [!NOTE]  
> Sono supportati tre tipi di telefoni: telefoni da tavolo (dispositivi IP o USB portatili progettati per l'uso da parte dei dipendenti presso la propria scrivania), dispositivi di conferenza (telefoni cellulari o IP a portata di mano progettati per l'uso nelle sale riunioni) e telefoni per l'area comune (telefoni IP che sono progettati per l'uso in aree condivise, ad esempio atri, cucine, pavimenti di fabbrica e aree di lavoro condivise, e possono essere personalizzati per creare set di funzionalità di Lync Phone Edition diversi.



</div>

Per informazioni dettagliate sulle tabelle di confronto su tutti i dispositivi ottimizzati per l'utilizzo con Lync, vedere "telefoni e dispositivi qualificati per <http://go.microsoft.com/fwlink/p/?linkid=208938>Microsoft Lync" all'indirizzo. Le tabelle di confronto telefono in questo sito Web includono informazioni sui prezzi e specifiche tecniche.
