# <a name="supported-devices-in-lync-server-2013"></a>Dispositivi supportati in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-10-08_

Lync Server supporta tutti i telefoni elencati nella tabella seguente. Si noti che non tutti i telefoni supportati possono eseguire Lync Phone Edition (alcuni possono eseguire solo versioni precedenti). Utilizzare la tabella per sapere quali telefoni possono eseguire Lync Phone Edition e dove ottenere informazioni dettagliate su tutti i telefoni supportati.

<div>


> [!NOTE]  
> Sono supportati tre tipi di telefoni: telefoni da tavolo (dispositivi IP o USB mobili che sono stati creati per essere utilizzati dai dipendenti nella propria scrivania), dispositivi per conferenze (telefoni IP o USB a mani libere che sono stati creati per essere utilizzati nelle sale riunioni) e telefoni di area comune (telefoni IP che sono progettate per essere utilizzate in aree condivise, ad esempio lobby, cucine, pavimenti di fabbrica e aree di lavoro condivise, e possono essere personalizzate per fornire diversi set di funzionalità di Lync Phone Edition.



</div>

Per informazioni dettagliate sulle tabelle di confronto su tutti i dispositivi ottimizzati per l'utilizzo con Lync, vedere "telefoni e dispositivi qualificati per <https://go.microsoft.com/fwlink/p/?linkid=208938>Microsoft Lync" all'indirizzo. Le tabelle di confronto dei telefoni in questo sito Web includono informazioni sui prezzi e specifiche tecniche.
