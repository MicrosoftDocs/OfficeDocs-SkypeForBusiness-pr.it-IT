# <a name="planning-for-monitoring-in-lync-server-2013"></a>Pianificazione del monitoraggio in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-09-05_

Il servizio di monitoraggio in Microsoft Lync Server 2013 consente agli amministratori di raccogliere i dati di utilizzo, tendenza e qualità dei servizi per le sessioni di comunicazione che si verificano nella propria organizzazione. Il monitoraggio in Lync Server 2013 non richiede più un ruolo del server separato. al contrario, il servizio di monitoraggio è incorporato in ogni Front End Server. Tuttavia, per impostazione predefinita, il monitoraggio non è abilitato in Lync Server 2013. Questo documento consentirà di stabilire se è consigliabile o meno abilitare questo servizio nella propria organizzazione.
