# <a name="planning-for-monitoring-in-lync-server-2013"></a>Pianificazione del monitoraggio in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-09-05_

Il servizio di monitoraggio in Microsoft Lync Server 2013 consente agli amministratori di raccogliere l'uso, la tendenza e la qualità dei dati del servizio per le sessioni di comunicazione che si svolgono nell'organizzazione. Il monitoraggio in Lync Server 2013 non richiede più un ruolo server distinto; il servizio di monitoraggio è invece incorporato in ogni server front-end. Tuttavia, per impostazione predefinita il monitoraggio non è abilitato in Lync Server 2013. Questo documento consente di determinare se il monitoraggio deve essere abilitato nell'organizzazione.
