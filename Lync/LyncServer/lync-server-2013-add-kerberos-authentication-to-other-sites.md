# <a name="in-lync-server-2013-add-kerberos-authentication-to-other-sites"></a>In Lync Server 2013 aggiungere l'autenticazione Kerberos ad altri siti

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2010-11-03_

Se si desidera aggiungere siti a una distribuzione esistente configurata per l'autenticazione Kerberos, è possibile utilizzare un account di autenticazione Kerberos per il nuovo sito o creare un nuovo account. Per informazioni dettagliate sulla creazione di un nuovo account per un sito, vedere [creare un account di autenticazione Kerberos in Lync Server 2013](lync-server-2013-create-a-kerberos-authentication-account.md). Per informazioni dettagliate sull'utilizzo dello stesso account per un nuovo sito mediante l'assegnazione di un account esistente al sito, vedere [assegnare un account di autenticazione Kerberos a un sito in Lync Server 2013](lync-server-2013-assign-a-kerberos-authentication-account-to-a-site.md).
