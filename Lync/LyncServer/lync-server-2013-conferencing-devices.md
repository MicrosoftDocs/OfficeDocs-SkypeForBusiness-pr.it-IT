# <a name="conferencing-devices-in-lync-server-2013"></a>Dispositivi per conferenze in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2014-02-05_

I dispositivi per conferenze portano le funzionalità di Lync Phone Edition nelle sale conferenze, consentendo agli utenti di ascoltare e, a seconda del dispositivo, di visualizzare persone in altre posizioni.

Utilizzare gli argomenti in questa sezione per informazioni su come configurare e gestire i dispositivi per conferenze.

<div>


> [!NOTE]  
> Per ulteriori informazioni sui dispositivi per conferenze supportati per l'utilizzo con Lync Server 2013, vedere la pagina Web <A href="http://technet.microsoft.com/lync/gg278164.aspx">telefoni e dispositivi qualificati per Microsoft Lync</A>.
