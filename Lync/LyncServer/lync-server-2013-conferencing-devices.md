# <a name="conferencing-devices-in-lync-server-2013"></a>Dispositivi di conferenza in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2014-02-05_

I dispositivi di conferenza conferiscono le funzionalità di Lync Phone Edition nelle sale riunioni, consentendo alle persone in sala di ascoltare e, a seconda del dispositivo, vedere persone in altre posizioni.

Usare gli argomenti di questa sezione per informazioni su come configurare e gestire i dispositivi di conferenza.

<div>


> [!NOTE]  
> Per altre informazioni sui dispositivi di conferenza supportati per l'uso con Lync Server 2013, vedere la pagina Web <A href="http://technet.microsoft.com/en-us/lync/gg278164.aspx">telefoni e dispositivi qualificati per Microsoft Lync</A>.
