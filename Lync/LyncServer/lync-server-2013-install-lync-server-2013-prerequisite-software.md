# <a name="install-lync-server-2013-prerequisite-software"></a>Installare il software prerequisito per Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2013-02-21_

Il software prerequisito per il server di chat persistente è lo stesso del software prerequisito per i server front end di Lync Server 2013 e per il server Lync Server 2013 Standard Edition in cui sono installate le funzionalità del server di chat persistente.

Il software prerequisito necessario per il file Store del server di chat persistente è uguale a quello per Lync Server 2013.

Il software prerequisito per i database di SQL Server per il contenuto e la conformità del server della chat persistente è uguale a quello per Lync Server 2013.

Per informazioni dettagliate su tutti i requisiti software per i server di Lync Server 2013, vedere [supporto e requisiti aggiuntivi del server in Lync server 2013](lync-server-2013-additional-server-support-and-requirements.md) nella documentazione relativa alla supportabilità.
