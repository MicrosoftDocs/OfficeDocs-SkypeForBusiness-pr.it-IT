# <a name="checking-for-updates-to-best-practices-analyzer-in-lync-server-2013"></a>Verifica della disponibilità di aggiornamenti per le procedure consigliate Analyzer in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-10-02_

Quando si avvia Best Practices Analyzer, lo strumento offre un'opzione per cercare gli aggiornamenti più recenti allo strumento. Se è disponibile un aggiornamento, è possibile scaricare l'aggiornamento. Se si sceglie di non scaricare gli aggiornamenti o se Best Practices Analyzer non riesce ad accedere a Internet, è possibile continuare a usare la versione già presente nel computer.

<div>


> [!NOTE]  
> Se è necessaria l'autenticazione proxy per accedere a Internet, Best Practices Analyzer non può accedere a nuovi aggiornamenti da scaricare. Tuttavia, è possibile scaricare manualmente l'ultima versione di RtcBPA. msi dall'area download Microsoft <A href="http://go.microsoft.com/fwlink/p/?linkid=266539">http://go.microsoft.com/fwlink/p/?linkId=266539</A>. Dopo aver scaricato il file, è possibile copiarlo nel computer in cui si vuole aggiornare Best Practices Analyzer e usare il file con estensione msi per installare la nuova versione dello strumento nel computer in uso.



</div>

Per aggiornare le regole di Best Practices Analyzer, è necessario eseguire lo strumento come amministratore nel computer locale. Se non è stato effettuato l'accesso con un account che è un membro del gruppo Administrators e vengono rilevati gli aggiornamenti, chiudere Best Practices Analyzer e quindi usare la procedura seguente per avviare il programma.

<div>

## <a name="to-open-best-practices-analyzer-as-administrator-to-check-for-updates"></a>Per aprire Analizzatore procedure consigliate come amministratore per verificare la disponibilità di aggiornamenti

1.  In un computer in cui è installato Best Practices Analyzer, fare clic sul pulsante **Start**, scegliere **Microsoft Lync Server 2013**, fare clic con il pulsante destro del mouse su **Best Practices Analyzer**e quindi scegliere **Esegui come amministratore**.

2.  Specificare le credenziali di un account membro del gruppo Administrators.
