# <a name="start-services-on-the-director-in-lync-server-2013"></a>Avviare i servizi nel server Director in Lync 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-09-08_

Dopo aver installato l'archivio di configurazione locale, aver installato i componenti di Lync Server e aver configurato i certificati su un server Director, è necessario avviare i servizi di Lync Server. È possibile utilizzare la procedura seguente per avviare i servizi in ogni Director della distribuzione.

<div>

## <a name="to-start-services-on-a-director"></a>Per avviare i servizi in un server Director

1.  Nella distribuzione guidata di Lync Server, nella pagina **Lync server 2013** , fare clic sul pulsante **Esegui** accanto a **passaggio 4: Avvia servizi**.

2.  Nella pagina **Avvia servizi** fare clic su **Avanti** per avviare i servizi Lync Server nel server.

3.  Nella pagina **Esecuzione comandi in corso**, dopo l'avvio di tutti i servizi, fare clic su **Fine**.

4.  Al di sotto del **passaggio 4: avviare i servizi**, fare clic su **stato dei servizi (facoltativo)**.

5.  In Microsoft Management Console (MMC) **Servizi** nel server verificare che tutti i servizi di Lync Server 2013 siano in esecuzione.
