# <a name="planning-for-security-in-lync-server-2013"></a>Pianificazione della sicurezza in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2016-06-22_

Usare questa sezione di sicurezza per valutare e gestire i rischi di sicurezza per la distribuzione di Lync Server 2013.

Anche se la distribuzione di Lync Server 2013 è modesta, è probabile che nella rete siano presenti componenti con una propria documentazione di sicurezza. Di conseguenza, è improbabile che questa sezione copra tutti gli aspetti della sicurezza per tutti i componenti e le aree pertinenti alla distribuzione.

Usare questa sezione come punto di partenza per affrontare la sicurezza della distribuzione di Lync Server 2013. Fornisce linee guida e procedure consigliate generali per la valutazione e la gestione dei rischi di sicurezza più comuni. Alla fine di ogni argomento sono elencate altre risorse di prodotto e sicurezza.

<div>


> [!IMPORTANT]  
> La sicurezza è un argomento in continua evoluzione. Man mano che sorgono nuove minacce e soluzioni, i documenti, le soluzioni, i metodi e le procedure obsoleti devono essere sostituiti da materiale aggiornato.
