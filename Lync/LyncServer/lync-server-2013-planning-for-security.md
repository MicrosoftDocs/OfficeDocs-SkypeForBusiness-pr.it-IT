# <a name="planning-for-security-in-lync-server-2013"></a>Pianificazione della sicurezza in Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<span> </span>

_**Ultimo argomento modificato:** 2016-06-22_

Utilizzare questa sezione di sicurezza per valutare e gestire i rischi per la sicurezza per la distribuzione di Lync Server 2013.

Anche se la distribuzione di Lync Server 2013 è modesto, è probabile che nella rete siano presenti componenti che dispongono di una documentazione relativa alla sicurezza. Pertanto, è improbabile che in questa sezione vengano descritti tutti gli aspetti della sicurezza per tutti i componenti e le aree rilevanti per la distribuzione.

Utilizzare questa sezione come punto di partenza per risolvere la sicurezza della distribuzione di Lync Server 2013. Vengono fornite linee guida generali e procedure consigliate per valutare e gestire i rischi di sicurezza più comuni. Alla fine di ogni argomento sono elencate altre risorse di prodotti e di sicurezza.

<div>


> [!IMPORTANT]  
> La sicurezza è un argomento in continua evoluzione. Quando emergono nuove minacce e soluzioni, i documenti obsoleti, le soluzioni, i metodi e le procedure devono essere sostituiti da materiale aggiornato.
