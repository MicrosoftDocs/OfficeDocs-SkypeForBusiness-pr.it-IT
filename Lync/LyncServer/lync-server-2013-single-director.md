# <a name="single-director-in-lync-server-2013"></a>Single Director in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-10-22_

Il Director può essere distribuito in una singola configurazione del Director o come pool di server Director. In questa sezione vengono definite la topologia e la configurazione di un singolo server Director. Se si sta pianificando la distribuzione di un pool di amministratori per la gestione della capacità elevata e per la disponibilità elevata, vedere l'argomento [pool di Director in scala in Lync Server 2013](lync-server-2013-scaled-director-pool.md) per la pianificazione di considerazioni su tale topologia.
