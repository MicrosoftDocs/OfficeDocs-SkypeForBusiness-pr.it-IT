# <a name="port-requirements-for-lync-server-2013"></a>Requisiti della porta per Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2013-03-27_

Lync Server richiede che vengano aperte porte specifiche sul firewall. Inoltre, se l'organizzazione è distribuita in IPsec (Internet Protocol Security), IPsec deve essere disabilitato sull'intervallo di porte usate per il recapito di audio, video e video panoramico.
