# <a name="planning-for-location-based-routing-in-lync-server-2013"></a>Pianificazione del routing Location-Based in Lync Server 2013

_**Ultimo argomento modificato:** 2013-07-31_

Le informazioni contenute in questo argomento riguardano gli aggiornamenti cumulativi per Lync Server 2013: febbraio 2013.

Location-Based routing rende possibile limitare il routing delle chiamate tra endpoint VoIP e endpoint PSTN in base alla posizione delle parti nella chiamata. Location-Based routing è parte dell'infrastruttura di VoIP aziendale di Lync Server 2013. Location-Based routing è una funzionalità di gestione delle chiamate che controlla il modo in cui vengono instradate tramite Lync Server 2013 CU1. Applica le regole di autorizzazione delle chiamate per stabilire se le chiamate possono essere instradate a endpoint PBX o PSTN in base alla posizione geografica del chiamante di Lync.
