# <a name="managing-persistent-chat-user-access-in-lync-server-2013"></a>Gestione dell'accesso degli utenti di chat persistente in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-10-06_

Negli argomenti seguenti viene descritto come utilizzare il pannello di controllo di Lync Server 2013 e la pagina **chat persistente** per gestire l'accesso degli utenti in chat persistente. Per gestire il server Chat persistente, è necessario disporre dei diritti e delle autorizzazioni di amministratore di chat persistente nel sistema. Quando si installa il server Chat persistente, è necessario specificare uno o più utenti che dispongono dei diritti e delle autorizzazioni di amministratore Persistent Chat necessari. Per concedere diritti e autorizzazioni di amministratore ad altri account utente, usare **Gestisci utenti e gruppi di utenti**. Per informazioni dettagliate, vedere [Abilitazione di un utente alla gestione di categorie, chat room e diritti utente e autorizzazioni in Lync Server 2013](lync-server-2013-enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions.md).
