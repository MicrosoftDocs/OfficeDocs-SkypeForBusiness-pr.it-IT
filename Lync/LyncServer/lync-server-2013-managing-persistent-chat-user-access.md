# <a name="managing-persistent-chat-user-access-in-lync-server-2013"></a>Gestione dell'accesso utente a Chat persistente in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-10-06_

Gli argomenti seguenti descrivono come usare il pannello di controllo di Lync Server 2013 e la pagina della **chat persistente** per gestire l'accesso degli utenti in chat persistente. Per gestire il server di chat persistente, è necessario disporre dei diritti e delle autorizzazioni di amministratore della chat persistente nel sistema. Quando si installa il server di chat persistente, è necessario specificare uno o più utenti che dispongono dei diritti e delle autorizzazioni necessarie per l'amministratore della chat persistente. Per concedere i diritti di amministratore e le autorizzazioni per gli account utente aggiuntivi, usare **Gestisci utenti e gruppi**di utenti. Per informazioni dettagliate, vedere [consentire a un utente di gestire categorie, chat room e diritti utente e autorizzazioni in Lync Server 2013](lync-server-2013-enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions.md).
