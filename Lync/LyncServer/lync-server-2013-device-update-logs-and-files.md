# <a name="device-update-logs-and-files-in-lync-server-2013"></a>Registri e file degli aggiornamenti dei dispositivi in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2013-02-20_

I registri degli aggiornamenti dei dispositivi contengono informazioni importanti che è possibile utilizzare per la gestione e la risoluzione dei problemi relativi al servizio Web aggiornamento dispositivi. È possibile modificare le operazioni registrate e rimuovere i registri e gli aggiornamenti dei dispositivi che non si desiderano o che non sono più necessari. In questa sezione viene descritto come utilizzare il pannello di controllo di Lync Server o Lync Server Management Shell per modificare le impostazioni di registrazione, cancellare il log degli aggiornamenti del dispositivo o rimuovere i file di registro dal server.

<div>


> [!NOTE]  
> Per informazioni dettagliate sui file di log degli aggiornamenti dei dispositivi, vedere <A href="http://technet.microsoft.com/library/gg398250(v=ocs.14).aspx">log file types and locations</A> in the Lync Server 2010 TechNet Library. Si noti che il servizio Web aggiornamento dispositivi, come tutti i componenti di Lync Phone Edition, è compatibile con Lync Server 2013 come accade con Lync Server 2010.
