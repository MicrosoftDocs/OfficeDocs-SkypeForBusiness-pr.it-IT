# <a name="device-update-logs-and-files-in-lync-server-2013"></a>Registri e file di aggiornamento dei dispositivi in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2013-02-20_

I registri di aggiornamento dei dispositivi contengono informazioni importanti che è possibile usare per gestire e risolvere i problemi relativi al servizio Web Update del dispositivo. È possibile modificare gli elementi registrati e rimuovere i registri e gli aggiornamenti del dispositivo che non si vuole o non è più necessario. Questa sezione descrive come usare il pannello di controllo di Lync Server o Lync Server Management Shell per modificare le impostazioni di registrazione, cancellare il log degli aggiornamenti del dispositivo o rimuovere i file di log dal server.

<div>


> [!NOTE]  
> Per informazioni dettagliate sui file di log per l'aggiornamento dei dispositivi, vedere <A href="http://technet.microsoft.com/en-us/library/gg398250(v=ocs.14).aspx">registrare i tipi di file e le posizioni</A> nella raccolta TechNet di Lync Server 2010. Si noti che il servizio Web Update per dispositivi, come tutti i componenti di Lync Phone Edition, funziona allo stesso modo con Lync Server 2013 come avviene con Lync Server 2010.
