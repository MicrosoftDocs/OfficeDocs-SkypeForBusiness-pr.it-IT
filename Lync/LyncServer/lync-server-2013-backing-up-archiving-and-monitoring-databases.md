# <a name="backing-up-archiving-and-monitoring-databases-in-lync-server-2013"></a>Eseguire il backup di database di archiviazione e monitoraggio in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2013-02-17_

Se è stata distribuita l'archiviazione o il monitoraggio, è necessario eseguire il backup di questi database in base ai criteri di backup di SQL Server dell'organizzazione.

<div>


> [!NOTE]  
> Le impostazioni per l'archiviazione e il monitoraggio vengono supportate quando si esegue il backup dell'Central Management store. Per informazioni dettagliate, vedere <A href="lync-server-2013-backing-up-core-data-and-settings.md">backup dei dati di base e delle impostazioni in Lync Server 2013</A>.



</div>

Per l'archiviazione e il monitoraggio, è possibile usare uno strumento di SQL Server, ad esempio SQL Server Management Studio, per eseguire un backup manuale oppure usare gli strumenti di gestione di SQL Server per pianificare backup automatici e regolari.
