# <a name="active-directory-domain-services-requirements-support-and-topologies-in-lync-server-2013"></a>Requisiti, supporto e topologie di servizi di dominio Active Directory in Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-10-05_

Prima di Lync Server 2010, Lync Server si basava su servizi di dominio Active Directory per archiviare tutte le impostazioni globali e i gruppi necessari per la distribuzione e la gestione di Lync Server. Ora molte di queste informazioni vengono archiviate nell'archivio di gestione centrale invece che in servizi di dominio Active Directory. Tuttavia, le estensioni dello schema degli oggetti utente, incluse le estensioni dello schema di Lync Server 2013, Lync Server 2010 e Office Communications Server 2007 R2, sono ancora memorizzate in servizi di dominio Active Directory.
