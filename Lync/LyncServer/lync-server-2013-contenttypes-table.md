# <a name="contenttypes-table-in-lync-server-2013"></a>Tabella ContentTypes in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2010-11-07_
