# <a name="call-park-application-cmdlets-in-lync-server-2013"></a>Cmdlet dell'applicazione Call Park in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-03-21_

L'applicazione Call Park consente a un utente di effettuare una chiamata in attesa, quindi recuperare la chiamata da un altro telefono. Usa questi cmdlet per configurare le impostazioni per le orbite di Call Park e per l'applicazione Call Park.

<div>

I cmdlet seguenti possono essere usati per gestire le applicazioni di Call Park.
