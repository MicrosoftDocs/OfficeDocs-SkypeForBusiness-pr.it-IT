# <a name="optional-verify-response-group-deployment-in-lync-server-2013"></a>Opzionale Verificare la distribuzione di Response Group in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-09-11_

Dopo aver configurato Response Group, è necessario verificare la configurazione per assicurarsi che i gruppi di risposte funzionino come previsto. Verificare almeno gli scenari seguenti usando i tipi di utenti seguenti:

**Utenti**

  - Un utente ospitato in Lync Server 2013

  - Un utente esterno che usa la rete PSTN (Public Switched Telephone Network)

  - Un agente ospitato in Lync Server 2013

**Scenari**

  - L'utente di Lync Server 2013 chiama il Response Group.

  - L'utente esterno chiama il Response Group.

  - Un utente chiama il gruppo di risposte mentre l'agente si trova in un'altra chiamata e passa alla coda.
