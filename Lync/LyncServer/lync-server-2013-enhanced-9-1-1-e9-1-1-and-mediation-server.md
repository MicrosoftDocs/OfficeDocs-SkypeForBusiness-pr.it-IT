# <a name="enhanced-9-1-1-e9-1-1-and-mediation-server-in-lync-server-2013"></a>Servizi di emergenza avanzati e Mediation Server in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-09-29_

Il Mediation Server offre funzionalità estese in modo da poter interagire correttamente con i provider di servizi avanzati di 9-1-1 (E9-1-1). Non è necessaria alcuna configurazione speciale nel server Mediation; le estensioni SIP necessarie per l'interazione E9-1-1 sono, per impostazione predefinita, incluse nel protocollo SIP del server di mediazione per le interazioni con un peer gateway (gateway PSTN, IP-PBX o SBC di un provider di servizi di telefonia Internet, incluso il servizio E9-1-1) Provider

Se il trunk SIP a un provider di servizi E9-1-1 può essere terminato in un pool di Mediation Server esistente o richiederà che i server di mediazione autonoma dipendano dal fatto che il SBC-1-1 può interagire con un pool di server di mediazione. Per informazioni dettagliate, vedere [Trunk M:N in Lync Server 2013](lync-server-2013-m-n-trunk.md).
