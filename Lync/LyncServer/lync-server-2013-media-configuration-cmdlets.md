# <a name="media-configuration-cmdlets-in-lync-server-2013"></a>Cmdlet di configurazione multimediale in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-03-21_

I cmdlet relativi alla configurazione multimediale consentono di specificare elementi come il livello di crittografia supportato e la risoluzione video massima consentita.

<div>

## <a name="media-configuration-cmdlets"></a>Cmdlet di configurazione multimediale

Usare i cmdlet seguenti per la configurazione multimediale.

**Configurazione multimediale**

  - <span></span>  
    [Get-CsMediaConfiguration](https://technet.microsoft.com/en-us/library/Gg398128(v=OCS.15))

  - <span></span>  
    [New-CsMediaConfiguration](https://technet.microsoft.com/en-us/library/Gg425881(v=OCS.15))

  - <span></span>  
    [Remove-CsMediaConfiguration](https://technet.microsoft.com/en-us/library/Gg398705(v=OCS.15))

  - <span></span>  
    [Set-CsMediaConfiguration](https://technet.microsoft.com/en-us/library/Gg398580(v=OCS.15))
