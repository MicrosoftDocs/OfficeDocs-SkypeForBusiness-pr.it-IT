# <a name="phase-3-deploy-lync-server-2013-pilot-pool"></a>Fase 3: distribuire il pool pilota di Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-10-19_

In questa sezione vengono illustrati i passaggi necessari per distribuire un pool pilota di Lync Server 2013. La distribuzione di Lync Server 2013 richiede l'utilizzo di generatore di topologie per definire la topologia e i componenti che si desidera distribuire, la preparazione dell'ambiente per la distribuzione dei componenti di Lync Server 2013, la pubblicazione della progettazione della topologia al primo front-end Server e quindi l'installazione e la configurazione del software Lync Server 2013 per i componenti per la distribuzione. Al termine, la distribuzione del pool pilota di Lync Server 2013 coesisterà con un pool Lync Server 2010 esistente.
