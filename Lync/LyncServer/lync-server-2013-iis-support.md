# <a name="iis-support-in-lync-server-2013"></a>Supporto di IIS in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-06-29_

Diversi componenti di Lync Server 2013 richiedono Internet Information Services (IIS). Le versioni supportate di IIS includono:

  - IIS 8.0

  - IIS 7.5

Per informazioni dettagliate sui requisiti di IIS per i componenti di Lync Server 2013, vedere i [requisiti di Internet Information Services (IIS) in Lync server 2013](lync-server-2013-internet-information-services-iis-requirements.md) nella documentazione relativa alla pianificazione.
