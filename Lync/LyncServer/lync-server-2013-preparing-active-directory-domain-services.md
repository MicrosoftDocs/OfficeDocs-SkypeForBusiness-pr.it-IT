# <a name="preparing-active-directory-domain-services-for-lync-server-2013"></a>Preparazione di Servizi di dominio Active Directory per Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2013-02-21_

Prima di distribuire e utilizzare Lync Server 2013, è necessario preparare i servizi di dominio Active Directory estendendo lo schema e quindi creando e configurando gli oggetti. Le estensioni dello schema aggiungono le classi e gli attributi di Active Directory richiesti da Lync Server.

Negli argomenti di questa sezione viene descritto come preparare Servizi di dominio Active Directory per la distribuzione di Lync Server e come assegnare le autorizzazioni di installazione e unità organizzativa (OU). Per informazioni dettagliate sulle modifiche dello schema necessarie per Lync Server, vedere [estensioni dello schema di Active Directory, classi e attributi utilizzati da Lync server 2013](lync-server-2013-active-directory-schema-extensions-classes-and-attributes-used-by-lync-server.md).
