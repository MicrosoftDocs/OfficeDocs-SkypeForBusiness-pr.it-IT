# <a name="managing-call-admission-control-in-lync-server-2013"></a>Gestione del controllo di ammissione di chiamata in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-11-01_
