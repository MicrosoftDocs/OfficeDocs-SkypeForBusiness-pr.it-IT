# <a name="remove-a-legacy-authorized-host-in-lync-server-2013-optional"></a>Rimuovere un host autorizzato legacy in Lync Server 2013 (facoltativo)

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-09-05_

Quando si esegue la migrazione del controllo delle chiamate remote a una distribuzione di Lync Server, è necessario rimuovere le voci host autorizzate legacy (note come *voci di applicazioni attendibili* in Lync Server) per qualsiasi gateway SIP/CSTA nella distribuzione legacy. È necessario servirsi degli strumenti di amministrazione per rimuovere le voci host autorizzato rispettivamente dalle distribuzioni dell'uno o dell'altro tipo.
