# <a name="remove-a-legacy-authorized-host-in-lync-server-2013-optional"></a>Rimuovere un host autorizzato legacy in Lync Server 2013 (facoltativo)

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-09-05_

Quando si esegue la migrazione del controllo delle chiamate remote a una distribuzione di Lync Server, è necessario rimuovere le voci di host autorizzati legacy (note come *voci di applicazione attendibili* in Lync Server) per qualsiasi gateway SIP/CSTA nella distribuzione legacy. È necessario usare gli strumenti di amministrazione per rimuovere rispettivamente le voci di host autorizzati da tali distribuzioni.
