# <a name="backing-up-file-stores-in-lync-server-2013"></a>Backup di archivi di file in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2013-02-17_

Il backup degli archivi di file di Lync Server include tutti i file e le cartelle usati dai componenti di Lync Server.

<div>

## <a name="to-back-up-file-stores"></a>Per eseguire il backup di archivi di file

1.  Per trovare le posizioni specifiche degli archivi di file di Lync Server, aprire Generatore di topologie e cercare nel nodo **archivi file** .

2.  USA Robocopy o un altro strumento di gestione del file System per copiare ogni archivio\\di file in $backup filestore.
