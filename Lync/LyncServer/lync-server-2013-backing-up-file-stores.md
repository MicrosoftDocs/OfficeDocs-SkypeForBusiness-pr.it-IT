# <a name="backing-up-file-stores-in-lync-server-2013"></a>Backup degli archivi di file in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2013-02-17_

Il backup degli archivi file di Lync Server include tutti i file e le cartelle utilizzati dai componenti di Lync Server.

<div>

## <a name="to-back-up-file-stores"></a>Per eseguire il backup degli archivi file

1.  Per trovare le posizioni specifiche degli archivi file di Lync Server, aprire Generatore di topologie e cercare nel nodo **archivi file** .

2.  Utilizzare Robocopy o un altro strumento di gestione del file System per copiare ogni archivio\\file in $backup filestore.
