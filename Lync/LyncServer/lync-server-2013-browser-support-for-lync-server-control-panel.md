</div>

<div id="mainSection">

<div id="mainBody">

_**Argomento Ultima modifica:** 2012-09-28_

Lync Server 2013 supporta l'uso di generatore di topologia e il pannello di controllo di Lync Server 2013 è supportato quando si usano i browser seguenti:

  - Internet Explorer 10, 32 bit

  - Internet Explorer 9, 32 bit

  - Internet Explorer 8, 32 bit
