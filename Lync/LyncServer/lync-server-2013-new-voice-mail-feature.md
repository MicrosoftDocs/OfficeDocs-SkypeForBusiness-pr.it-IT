# <a name="new-voice-mail-feature-in-lync-server-2013"></a>Nuova funzionalità di segreteria telefonica in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-10-05_

Lync Server 2013 introduce l'escape per la segreteria telefonica, un miglioramento per la gestione della segreteria telefonica. Questa nuova funzionalità può rilevare quando una chiamata è stata instradata alla segreteria telefonica e impedire che la chiamata venga immediatamente instradata alla segreteria telefonica dell'utente senza dare all'utente la possibilità di rispondere alla chiamata. Questo scenario si verifica quando l'utente abilita la chiamata simultanea al cellulare e il telefono cellulare è disattivato, non è più a portata di batteria o fuori campo. La funzionalità di escape della segreteria telefonica rileva che la chiamata è stata immediatamente risolta dalla segreteria telefonica dell'utente e disconnette la chiamata alla segreteria telefonica per dispositivi mobili. La chiamata continua a squillare sugli altri endpoint dell'utente dando all'utente la possibilità di rispondere alla chiamata. Se l'utente non risponde alla chiamata, la chiamata viene instradata alla segreteria telefonica aziendale.
