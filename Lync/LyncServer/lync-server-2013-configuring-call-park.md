# <a name="configuring-call-park-in-lync-server-2013"></a>Configurazione del parcheggio di chiamata in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-10-30_

Parcheggio di chiamata consente a un utente di VoIP aziendale di inserire una chiamata in attesa da un telefono e quindi di recuperare la chiamata in un secondo momento componendo un numero interno (noto come *orbita*del parcheggio di chiamata) da qualsiasi telefono.

I componenti utilizzati dal parcheggio di chiamata vengono installati e abilitati automaticamente nel front end server o nel server Standard Edition quando si distribuisce VoIP aziendale. Tuttavia, è necessario configurare il parcheggio di chiamata prima che sia disponibile per gli utenti.

In questa sezione viene illustrata la configurazione del parcheggio di chiamata.
