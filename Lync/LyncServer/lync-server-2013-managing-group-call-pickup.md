# <a name="managing-group-call-pickup-in-lync-server-2013"></a>Gestione del prelievo delle chiamate di gruppo in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2013-02-22_

Aggiornamento cumulativo per Lync Server 2013: febbraio 2013 introduce il prelievo delle chiamate di gruppo come nuova funzionalità VoIP aziendale. Il prelievo delle chiamate di gruppo consente agli utenti di VoIP aziendale di raccogliere le chiamate che squillano per un altro utente componendo un numero di gruppo di prelievo di chiamata.

Negli argomenti di questa sezione vengono fornite procedure dettagliate per le attività da eseguire per configurare il prelievo delle chiamate di gruppo nella distribuzione.
