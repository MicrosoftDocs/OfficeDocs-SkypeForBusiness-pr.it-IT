# <a name="lync-web-app-supported-platforms-for-lync-server-2013"></a>Piattaforme supportate di Lync Web App per Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2017-02-07_

Per usare Lync Web App, è necessario disporre di una delle seguenti combinazioni di sistema operativo e browser supportate.

<div>


> [!NOTE]  
> È necessario un plug-in per alcune funzionalità di Lync Web App, tra cui la voce, il video, la condivisione e la visualizzazione della condivisione dello schermo in corso basato su computer. È possibile installare il plug-in di condivisione quando si partecipa alla riunione o quando si avvia una di queste funzionalità. 1<BR>

## <a name="hardware-requirements"></a>Requisiti hardware

I requisiti hardware per i computer sono determinati dal sistema operativo e dal browser. Le funzionalità vocali e telefoniche richiedono un microfono e altoparlanti, auricolare con microfono o un dispositivo equivalente compatibile con il computer. Le caratteristiche video richiedono un dispositivo video compatibile con il computer. Per informazioni dettagliate sul supporto hardware video e la qualità video prevista, vedere [requisiti per i video client di Lync per Lync Server 2013](lync-server-2013-lync-client-video-requirements.md).
