# <a name="server-software-and-infrastructure-support-in-lync-server-2013"></a>Supporto dell'infrastruttura e del software server in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-06-29_

Il supporto software per i componenti server, inclusi tutti i ruoli e lo spazio di archiviazione del server Lync Server 2013, include sistemi operativi supportati, software per database, software per l'infrastruttura e altro software necessario per supportare funzionalità specifiche. Include anche la virtualizzazione dei componenti del server.
