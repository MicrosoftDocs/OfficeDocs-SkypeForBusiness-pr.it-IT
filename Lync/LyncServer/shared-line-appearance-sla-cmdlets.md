# <a name="shared-line-appearance-sla-cmdlets"></a>Cmdlet per l'aspetto delle linee condivise (SLA)

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2016-04-12_

I cmdlet relativi alla Single Line Appearance (SLA) consentono di configurare un numero di telefono condiviso con più delegati che possono raccogliere quella linea.

<div>

## <a name="sla-cmdlets"></a>Cmdlet di SLA

Utilizzare i cmdlet seguenti per SLA.

**Configurazione del contratto di servizio**

  - <span></span>  
    [Get-CsSlaConfiguration](https://technet.microsoft.com/library/Mt703200(v=OCS.15))

  - <span></span>  
    [Set-CsSlaConfiguration](https://technet.microsoft.com/library/Mt703202(v=OCS.15))

  - <span></span>  
    [Remove-CsSlaConfiguration](https://technet.microsoft.com/library/Mt703201(v=OCS.15))

**Configurazione di delegati SLA**

  - <span></span>  
    [Add-CsSlaDelegates](https://technet.microsoft.com/library/Mt703199(v=OCS.15))

  - <span></span>  
    [Remove-CsSlaDelegates](https://technet.microsoft.com/library/Mt703203(v=OCS.15))
