# <a name="shared-line-appearance-sla-cmdlets"></a>Cmdlet per l'aspetto delle linee condivise (SLA)

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2016-04-12_

I cmdlet relativi all'aspetto a singola riga (SLA) consentono di configurare un numero di telefono condiviso con più delegati che possono prelevare tale linea.

<div>

## <a name="sla-cmdlets"></a>Cmdlet SLA

Usare i cmdlet seguenti per SLA.

**Configurazione di SLA**

  - <span></span>  
    [Get-CsSlaConfiguration](https://technet.microsoft.com/en-us/library/Mt703200(v=OCS.15))

  - <span></span>  
    [Set-CsSlaConfiguration](https://technet.microsoft.com/en-us/library/Mt703202(v=OCS.15))

  - <span></span>  
    [Remove-CsSlaConfiguration](https://technet.microsoft.com/en-us/library/Mt703201(v=OCS.15))

**Configurazione di delegati SLA**

  - <span></span>  
    [Add-CsSlaDelegates](https://technet.microsoft.com/en-us/library/Mt703199(v=OCS.15))

  - <span></span>  
    [Remove-CsSlaDelegates](https://technet.microsoft.com/en-us/library/Mt703203(v=OCS.15))

</div>
