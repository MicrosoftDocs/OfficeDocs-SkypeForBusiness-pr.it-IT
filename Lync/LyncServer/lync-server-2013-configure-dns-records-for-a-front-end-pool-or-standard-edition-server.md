# <a name="configure-dns-records-in-lync-server-2013-for-a-front-end-pool-or-standard-edition-server"></a>Configurare record DNS in Lync Server 2013 per un pool Front End o un server Standard Edition

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Argomento Ultima modifica:** 2012-10-01_

Lync Server 2013 usa il DNS (Domain Name System) per registrare e gestire i record per il nome di dominio appropriato alla risoluzione dell'indirizzo IP. È necessario configurare i record DNS necessari per la distribuzione prima di utilizzare il server Standard Edition o il pool Front-end. I collegamenti seguenti forniranno indicazioni sui record che devono essere creati per consentire il corretto funzionamento di Lync Server 2013.
