_**Ultimo argomento modificato:** 2012-10-01_

Lync Server 2013 utilizza il DNS (Domain Name System) per registrare e gestire i record per il nome di dominio appropriato alla risoluzione dell'indirizzo IP. Prima di utilizzare il server Standard Edition o il pool Front End, è necessario configurare i record DNS necessari per la distribuzione.
