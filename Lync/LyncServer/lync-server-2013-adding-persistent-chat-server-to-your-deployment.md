# <a name="adding-persistent-chat-server-to-your-deployment-in-lync-server-2013"></a>Aggiunta del server Chat persistente alla distribuzione in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-09-12_

Dopo aver installato il software prerequisito su ogni server in cui si prevede di distribuire Lync Server 2013, il server Chat persistente, è necessario utilizzare Generatore di topologie per aggiungere il supporto del server Chat persistente alla topologia e quindi pubblicare la topologia.
