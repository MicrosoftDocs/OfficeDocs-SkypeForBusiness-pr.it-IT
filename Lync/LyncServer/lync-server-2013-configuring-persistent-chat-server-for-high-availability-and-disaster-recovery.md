_**Ultimo argomento modificato:** 2012-10-01_

Lync Server 2013, servizi del server Chat persistente utilizza una configurazione del *pool esteso* per il ripristino di emergenza. Un pool esteso è un pool con computer distribuiti tra due data center fisici, ma che si trovano all'interno di un singolo sito logico di Lync Server.
