# <a name="back-up-systems-and-data"></a>Eseguire il backup di sistemi e dati

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Ultimo argomento modificato:** 2012-09-18_

Prima di iniziare la migrazione a Lync Server 2013, è consigliabile eseguire un backup completo del sistema e documentare il sistema esistente, incluso un inventario degli account utente ospitati in ogni pool, in modo da poter ripristinare Lync Server 2010, se necessario. Sono disponibili diversi strumenti e programmi per eseguire il backup e il ripristino di dati, impostazioni e sistemi.

Per ulteriori informazioni e procedure, vedere [backup e ripristino di Lync Server 2010](https://go.microsoft.com/fwlink/p/?linkid=265417) .
