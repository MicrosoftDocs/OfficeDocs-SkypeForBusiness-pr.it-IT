# <a name="direct-routing-media-path-country-codes"></a><span data-ttu-id="de274-103">Direct Routing media path country codes</span><span class="sxs-lookup"><span data-stu-id="de274-103">Direct Routing media path country codes</span></span>

<span data-ttu-id="de274-104">Quando si sceglie un percorso di routing per gli elementi multimediali, per impostazione predefinita il routing diretto assegna sempre un data center in base all'indirizzo IP pubblico del controller dei confini della sessione (SBC) e seleziona sempre il percorso più vicino al data center SBC.</span><span class="sxs-lookup"><span data-stu-id="de274-104">When choosing a routing path for media, Direct Routing, by default, always assigns a datacenter based on the public IP address of the Session Border Controller (SBC), and always selects the path closest to the SBC datacenter.</span></span>

<span data-ttu-id="de274-105">Tuttavia, in alcuni casi il percorso del supporto predefinito potrebbe non essere il percorso ottimale del supporto; Ad esempio, un indirizzo IP pubblico di un intervallo negli Stati Uniti potrebbe essere assegnato a un indirizzo SBC situato in Europa.</span><span class="sxs-lookup"><span data-stu-id="de274-105">However, in some cases the default media path might not be the optimal media path; for example, a public IP from a United States range might be assigned to an SBC located in Europe.</span></span> 

<span data-ttu-id="de274-106">Usando il parametro -MediaRelayRoutingLocationOverride con i cmdlet New-CsOnlinePSTNGateway e Set-CsOnlinePSTNGateway, è possibile specificare l'area geografica preferita per il traffico multimediale.</span><span class="sxs-lookup"><span data-stu-id="de274-106">By using the -MediaRelayRoutingLocationOverride parameter with the New-CsOnlinePSTNGateway and Set-CsOnlinePSTNGateway cmdlets, you can specify the preferred region for media traffic.</span></span> <span data-ttu-id="de274-107">Ad esempio, il comando seguente specifica che l'area geografica preferita è la Germania:</span><span class="sxs-lookup"><span data-stu-id="de274-107">For example, the following command specifies that the preferred region is Germany:</span></span>

<span data-ttu-id="de274-108">Set-CSOnlinePSTNGateway -Identity sbc1.contoso.com –MediaRelayRoutingLocationOverride DE</span><span class="sxs-lookup"><span data-stu-id="de274-108">Set-CSOnlinePSTNGateway -Identity sbc1.contoso.com –MediaRelayRoutingLocationOverride DE</span></span> 

<span data-ttu-id="de274-109">Si noti che Microsoft consiglia di impostare questo parametro solo se i registri delle chiamate indicano chiaramente che l'assegnazione predefinita del data center per il percorso del supporto non usa il percorso più vicino al data center SBC.</span><span class="sxs-lookup"><span data-stu-id="de274-109">Note that Microsoft only recommends setting this parameter if the call logs clearly indicate that the default assignment of the datacenter for the media path does not use the path closest to the SBC datacenter.</span></span>
