# <a name="whats-the-status-of-your-port-orders"></a>Qual è lo stato degli ordini di portabilità?

Per visualizzare lo stato dell'ordine di trasferimento, nel riquadro di spostamento sinistro dell'interfaccia di amministrazione di Microsoft Teams passare a >**ordini di trasferimento** **vocale** >  e quindi fare clic su **Cronologia ordini**. Ogni stato dell'ordine di portabilità è elencato nella colonna **Stato** . Scopri [quanto tempo è necessario per trasferire i numeri](../phone-number-calling-plans/port-order-overview.md#how-long-does-it-take-to-port-numbers) per ottenere informazioni sul processo di ordine. 

La tabella seguente elenca gli stati dell'ordine di trasferimento, nonché le azioni che puoi eseguire se necessario.

|**Stato**|**Puoi visualizzare l'ordine?**|**Puoi modificare l'ordine?**|**Puoi annullare l'ordine?**|**Puoi eliminare l'ordine?**|**Descrizione**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Elaborazione** <br/> |Sì  <br/> |No  <br/> |Sì  <br/> |No  <br/> |L'amministratore ha creato l'ordine ed è stato ricevuto da Microsoft.  <br/> |
|**Vettore di contatto** <br/> |Sì  <br/> |No  <br/> |Sì  <br/> |No  <br/> |L'ordine è stato ricevuto e approvato da Microsoft e stiamo collaborando con il gestore precedente per ottenere l'approvazione.  <br/> |
|**Trasferimento approvato** <br/> |Sì  <br/> |No  <br/> |Sì  <br/> |No  <br/> |L'ordine è stato accettato dal gestore precedente ed è stata impostata la data FOC (Firm Order Commitment).  <br/> |
|**Trasferimento in sospeso** <br/> |Sì  <br/> |No  <br/> |No  <br/> |No  <br/> |Il trasferimento è a meno di 24 ore di distanza, quindi l'ordine non può più essere modificato o annullato.  <br/> |
|**Errore** <br/> |No  <br/> |Sì  <br/> |Sì  <br/> |Sì (in questo momento, non puoi eliminare l'ordine di trasferimento in caso di errore. L'ordine di trasferimento deve essere ricreato oppure è necessario contattare il [Service Desk di TNS](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md).  <br/> |Il gestore precedente ha rifiutato l'ordine.  <br/> |
|**Completato** <br/> |Sì  <br/> |No  <br/> |No  <br/> |No  <br/> |I numeri sono stati trasferiti correttamente.  <br/> |
|**Annullato** <br/> |No  <br/> |Sì  <br/> |No  <br/> |No  <br/> |L'amministratore ha annullato l'ordine.  <br/> |

Per istruzioni dettagliate complete, vedere [Trasferire i numeri di telefono in Teams](transfer-phone-numbers-to-teams.md).

Se hai bisogno di aiuto o se hai bisogno di ottenere più numeri di telefono, contatta il [Service Desk di TNS](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md).
