<a name="presenter-producer-and-attendee-capabilities-in-a-teams-live-event"></a>Funzionalità di relatore, produttore e partecipante in un evento live di Teams
======================================================

Microsoft Teams supporta gli eventi live trasmessi a un ampio pubblico online. Mentre il ruolo di partecipante è passivo, il relatore e il produttore dell'evento hanno accesso a numerose funzionalità di riunione, tra cui:  

- Chat (include foto e adesivi)
- Registrazione
- File
- DOMANDE E RISPOSTE

Questo articolo descrive le funzionalità disponibili in un evento live di Teams.

## <a name="presenter-and-producer"></a>Relatore e produttore

I ruoli di relatore e produttore includono:

- Relatori dell'organizzazione
- Relatori di altre organizzazioni. I relatori sono designati dal produttore e hanno bisogno di un invito personale dal produttore.

Relatori e produttori hanno accesso a tutte le funzionalità dell'evento live.

| |  | |||
|---------|----------------|----------------|---------------------|------------|--------------|
|  **Funzionalità**       | Prima dell'evento | Durante l'evento | Dopo l'evento |
| Chat | Sì | Sì| Sì |
| Registrazione | N/D |N/D | Sì |
| File | Sì |Sì | Sì |
| Domande/Risposte, se abilitato dal produttore| Sì | Sì | Sì |
|||||||

## <a name="live-event-attendee"></a>Partecipante all'evento live

Un partecipante all'evento live è una persona che ha il collegamento all'evento live o alla registrazione dell'evento. Non sono necessarie autorizzazioni speciali per partecipare all'evento o per visualizzarlo. Un partecipante che visualizza l'evento live in tempo reale può partecipare alle domande e risposte, se autorizzato dal relatore e/o dal produttore. 

| |  | |||
|---------|----------------|----------------|---------------------|------------|--------------|
|  **Funzionalità**       | Prima dell'evento | Durante l'evento | Dopo l'evento |
| Domande/Risposte, se abilitato dal produttore | Sì | Sì | Sì |
| Registrazione | N/D | N/D| Sì |
| Controlli per la riproduzione video, se abilitati da produttore | Sì | Sì | Sì |
|||||||
