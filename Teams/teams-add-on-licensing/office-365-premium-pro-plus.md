# <a name="microsoft-365-apps-for-enterprise"></a>App Microsoft 365 per le aziende

Se è già stato acquistato il piano [Microsoft 365 Apps for Enterprise](https://products.office.com/en/business/office-365-proplus-business-software) , ecco le licenze per i componenti aggiuntivi che è necessario acquistare per ottenere altre funzionalità per i team.

> [!IMPORTANT]
> Anche se è possibile acquistare i piani di audioconferenza, sistema telefonico e chiamate per abilitare le funzionalità di chiamata vocale, è consigliabile valutare un piano Enterprise E5 per verificare se è più conveniente.


|Per aggiungere questa caratteristica |Ecco cosa devi fare |
|:---------------------------|:--------------------------------|
|Audioconferenza <br/> |Acquistare il componente aggiuntivo audioconferenza. <br/>Acquistare 1 licenza di audioconferenza per ogni persona che sta per pianificare o ospitare una riunione di accesso esterno. I partecipanti non hanno bisogno di una licenza. <br/>Una volta acquistate le licenze, Microsoft sarà il provider di servizi di audioconferenza. |
|Numeri verdi per l'accesso esterno alle conferenze e la possibilità di effettuare chiamate in uscita da una conferenza per aggiungere qualcuno chiamando qualsiasi numero di telefono nel mondo <br/> |Acquistare [crediti per comunicazioni](../set-up-communications-credits-for-your-organization.md).|
|Sistema telefonico <br/> |<ol><li>[Passare a un piano di Office 365 per le aziende diverso](https://support.office.com/article/73318661-8f33-478b-bcc7-fb8d69dbb22a). Se si hanno app Microsoft 365 per le aziende, è consigliabile aggiornare il piano a un piano Enterprise E5 per ottenere le caratteristiche di cloud Voice. Se esegui l'aggiornamento a un piano Enterprise E1 o E3, apparirà l'opzione per acquistare il componente aggiuntivo Sistema telefonico. <br/></li><li>Acquisto di un componente aggiuntivo al Sistema telefonico. <br/></li><li>Acquistare un [piano](../calling-plans-for-office-365.md) per le chiamate per Office 365. <br/>In alternativa, utilizza il [servizio telefonico del tuo provider esistente](microsoft-teams-add-on-licensing.md#bkmk_existing).</li></ol>  <br/> |
|Acquistare un piano per le chiamate <br/> |<ol><li>[Passare a un piano di Office 365 per le aziende diverso](https://support.office.com/article/73318661-8f33-478b-bcc7-fb8d69dbb22a). <br/></li><li>Acquisto di un componente aggiuntivo al Sistema telefonico.</li><li>Acquistare un [piano](../calling-plans-for-office-365.md)per le chiamate.<br/> È possibile acquistarlo solo dopo aver acquistato il componente aggiuntivo per il sistema telefonico. <br/> |
