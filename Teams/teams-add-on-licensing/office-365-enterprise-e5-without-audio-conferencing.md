# <a name="office-365-enterprise-e5-without-audio-conferencing"></a>Office 365 Enterprise E5 senza Audioconferenza

Se è già stato acquistato il piano Office 365 Enterprise E5 (senza servizi di audioconferenza), ecco le licenze per i componenti aggiuntivi Microsoft teams che è necessario acquistare per ottenere altre funzionalità.

|Per aggiungere questa caratteristica|Ecco cosa devi fare |
|:------------------|:--------------------------|
|Audioconferenza <br/> | Per ottenere questo componente aggiuntivo, usare il [pulsante Cambia piano](https://support.office.com/article/73318661-8f33-478b-bcc7-fb8d69dbb22a), se disponibile, per passare al piano Office 365 Enterprise E5 che include i servizi di audioconferenza. In questo modo, saranno disponibili anche altre funzionalità. </li></ol> |
|Il Sistema telefonico è incluso! <br/> | Acquistare un [piano](../calling-plans-for-office-365.md) per le chiamate per Office 365. <br/>  In alternativa, utilizza il [servizio telefonico del tuo provider esistente](microsoft-teams-add-on-licensing.md#bkmk_existing).  <br/> |
|La segreteria telefonica cloud è inclusa. <br/> |Non è necessario eseguire altre operazioni.  <br/> |
