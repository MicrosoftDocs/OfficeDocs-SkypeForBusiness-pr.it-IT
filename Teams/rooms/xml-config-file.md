# <a name="manage-a-microsoft-teams-rooms-console-settings-remotely-with-an-xml-configuration-file"></a>Gestire le impostazioni della console di Microsoft Teams Rooms in remoto con un file di configurazione XML

Questo articolo illustra la gestione remota delle impostazioni predefinite usate da un dispositivo Microsoft Teams Room, inclusa l'applicazione di un tema personalizzato. Illustra come creare un file di impostazioni master e crea collegamenti alle discussioni su come posizionarle in base alle esigenze nei dispositivi gestiti in remoto.
  
È possibile modificare le impostazioni predefinite dei dispositivi gestiti in remoto aggiornando un file XML master e inviando copie alle console gestite. È anche possibile implementare temi personalizzati sulle console di Microsoft Teams Room con file di configurazione XML.
  
## <a name="create-an-xml-configuration-file"></a>Creare un file di configurazione XML

Per creare un file di impostazioni, è possibile usare qualsiasi editor di testo. La **tabella Elementi XML** illustra gli elementi mostrati in questo file di SkypeSettings.xml di configurazione (nome file obbligatorio).
  
```XML
<SkypeSettings>
    <AutoScreenShare>true</AutoScreenShare>
    <HideMeetingName>true</HideMeetingName>
    <UserAccount>
        <SkypeSignInAddress>RanierConf@contoso.com</SkypeSignInAddress>
        <ExchangeAddress>RanierConf@contoso.com</ExchangeAddress>
        <ModernAuthEnabled>false</ModernAuthEnabled>
        <DomainUsername>Seattle\RanierConf</DomainUsername>
        <Password>password</Password>
        <ConfigureDomain>domain1, domain2</ConfigureDomain>
    </UserAccount>
    <IsTeamsDefaultClient>false</IsTeamsDefaultClient>
    <BluetoothAdvertisementEnabled>true</BluetoothAdvertisementEnabled>
    <AutoAcceptProximateMeetingInvitations>false</AutoAcceptProximateMeetingInvitations>
    <SkypeMeetingsEnabled>false</SkypeMeetingsEnabled>
    <TeamsMeetingsEnabled>true</TeamsMeetingsEnabled>
    <WebExMeetingsEnabled>true</WebExMeetingsEnabled>
    <UseCustomInfoForThirdPartyMeetings>true</UseCustomInfoForThirdPartyMeetings>
    <CustomDisplayNameForThirdPartyMeetings>Rainier Conference Room</CustomDisplayNameForThirdPartyMeetings>
    <CustomDisplayEmailForThirdPartyMeetings>RanierConf@contoso.com</CustomDisplayEmailForThirdPartyMeetings> 
    <DualScreenMode>true</DualScreenMode>
    <DuplicateIngestDefault>false</DuplicateIngestDefault>
    <DisableTeamsAudioSharing>false</DisableTeamsAudioSharing>
    <CortanaEnabled>false</CortanaEnabled>
    <CortanaWakewordEnabled>false</CortanaWakewordEnabled>
    <SendLogs>
        <EmailAddressForLogsAndFeedback>RanierConf@contoso.com</EmailAddressForLogsAndFeedback>
        <SendLogsAndFeedback>true</SendLogsAndFeedback>
    </SendLogs>
    <Devices>
        <MicrophoneForCommunication>Microsoft LifeChat LX-6000</MicrophoneForCommunication>
        <SpeakerForCommunication>Realtek High Definition Audio</SpeakerForCommunication>
        <DefaultSpeaker>Polycom CX5100</DefaultSpeaker>
        <ContentCameraId>USB\VID_046D&amp;PID_0843&amp;MI_00\7&amp;17446CF2&amp;0&amp;0000</ContentCameraId>
        <ContentCameraInverted>false</ContentCameraInverted>
        <ContentCameraEnhancement>true</ContentCameraEnhancement>
    </Devices>
    <Theming>
        <ThemeName>Custom</ThemeName>
        <CustomThemeImageUrl>file name</CustomThemeImageUrl>
        <CustomThemeColor>
            <RedComponent>100</RedComponent>
            <GreenComponent>100</GreenComponent>
            <BlueComponent>100</BlueComponent>
        </CustomThemeColor>
    </Theming>
    <CoordinatedMeetings enabled="true"> 
        <TrustedAccounts>room@contoso.com</TrustedAccounts>
        <Settings> 
            <Audio default="false" enabled="false" />
            <Video default="false" enabled="true" /> 
            <Whiteboard default="false" enabled="false"/>
        </Settings> 
    </CoordinatedMeetings>
</SkypeSettings>
```

Se un valore variabile è del tipo errato, gli elementi non sono in ordine, gli elementi non vengono chiusi o viene rilevato un altro errore, il formato del file XML non è *corretto.* Durante l'elaborazione di un file XML in formato non corretto, le impostazioni trovate fino al momento in cui si verifica l'errore vengono applicate e il resto del file viene ignorato. Gli elementi sconosciuti nel file XML vengono ignorati. Se un parametro viene omesso, rimane invariato nel dispositivo. Se il valore di un parametro non è valido, il valore precedente rimane invariato.

## <a name="manage-console-settings-with-an-xml-configuration-file"></a>Gestire le impostazioni della console con un file di configurazione XML

All'avvio, se una console di Microsoft Teams Rooms trova un file XML denominato SkypeSettings.xml situato in , applica le impostazioni di configurazione indicate dal file XML e quindi elimina il `C:\Users\Skype\AppData\Local\Packages\Microsoft.SkypeRoomSystem_8wekyb3d8bbwe\LocalState` file XML.
  
A seconda del numero di dispositivi per sale di Microsoft Teams dell'organizzazione e di come si sceglie di gestirli, è possibile inserire il file di configurazione XML in diversi modi. Dopo aver inserito il file nella console, riavviarlo per elaborare le modifiche alla configurazione. Il file di configurazione XML viene eliminato al termine dell'elaborazione. I metodi di gestione suggeriti per i dispositivi delle sale di Microsoft Teams vengono discussi in:
  
- [Configurazione di Criteri di gruppo per sale di Microsoft Teams](rooms-operations.md#GroupPolicy)
- [Gestione remota con PowerShell](rooms-operations.md#RemotePS) e [configurazione di un elemento file](https://technet.microsoft.com/library/cc772536%28v=ws.11%29.aspx)

Puoi utilizzare qualsiasi metodo che desideri, purché tu possa usarlo per trasferire i file e avviare un riavvio sul dispositivo console. Il file deve essere leggibile, scrivibile ed eliminabile dall'account utente locale del dispositivo. Preferibilmente è di proprietà di un utente con privilegi completi. Se le autorizzazioni per i file non sono impostate correttamente, il software potrebbe non riuscire ad applicare le impostazioni, potrebbe non riuscire a eliminare il file al completamento dell'elaborazione e potrebbe persino verificarsi un arresto anomalo.
  
## <a name="custom-theme-images"></a>Immagini tema personalizzato

<a name="Themes"> </a>

Il file di immagine del tema personalizzato deve essere inserito nella `C:\Users\Skype\AppData\Local\Packages\Microsoft.SkypeRoomSystem_8wekyb3d8bbwe\LocalState` cartella. Immettere il nome e l'estensione del file nella \<CustomThemeImageUrl\> variabile.
  
Il file di immagine deve avere dimensioni esattamente di 3840X1080 pixel e deve essere uno dei formati di file seguenti: jpg, jpeg, png e bmp. Se l'organizzazione vuole un'immagine personalizzata, un progettista di grafici può usare [il modello di photoshop tema personalizzato.](../downloads/ThemingTemplateMicrosoftTeamsRooms_v2.1.psd) Contiene ulteriori dettagli sulla posizione dei vari elementi dell'interfaccia utente rispetto al resto dell'immagine di un tema e sulle aree che vengono visualizzate sulle console e sugli schermi.
  
Il file di configurazione XML deve essere aggiornato all'avvio del dispositivo per riconoscere l'immagine del tema. Dopo l'elaborazione e l'eliminazione del nuovo file XML, il file grafico del tema viene eliminato dalla directory.
  
## <a name="locate-the-content-camera-usb-instance-path"></a>Individuare il percorso dell'istanza USB di Fotocamera del contenuto

Per individuare il percorso dell'istanza:

1. Accedere alle impostazioni di Windows nella console Di Microsoft Teams Room.
2. Immettere la password dell'amministratore.
3. Da un prompt dei comandi digitare `devmgmt.msc` per visualizzare Gestione dispositivi.
4. In **Gestione dispositivi,** esaminare il nodo **Dispositivi di** acquisizione immagini e individuare la fotocamera del contenuto.
5. Fare clic con il pulsante destro del mouse sulla fotocamera e aprire **Proprietà.**
6. Selezionare la **scheda Dettagli** e individuare la proprietà Percorso **istanza dispositivo** nell'elenco a discesa.
7. Il valore visualizzato è il percorso dell'istanza del dispositivo da impostare nel file di configurazione XML. Quando si specifica il percorso in XML, sostituire la e commerciale (&) con `&amp;` .
