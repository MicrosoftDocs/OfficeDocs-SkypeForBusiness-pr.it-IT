# <a name="microsoft-teams-meeting-attendance-report"></a>Report sulla partecipazione alle riunioni di Microsoft Teams

Gli organizzatori della riunione possono visualizzare e scaricare un report sulla partecipazione alla riunione. Trovare questo report nel **riquadro** Partecipanti della riunione facendo clic sulla freccia di download come illustrato di seguito. È possibile scaricare il report come file . File CSV (formato testo).

:::image type="content" source="../media/meetings-attendance-download.JPG" alt-text="Controllo per il download dei report sulle presenze delle riunioni in Microsoft Teams":::

Per i tenant dell'istruzione, questo report è utile per tenere traccia della partecipazione degli studenti alle classi online. Ad esempio, l'insegnante può scaricare il report di partecipazione all'inizio della classe come un modo semplice per effettuare una "chiamata a rullino". Per altre informazioni, vedere [Scaricare i report sulle presenze in Teams.](https://support.office.com/article/download-attendance-reports-in-teams-ae7cf170-530c-47d3-84c1-3aedac74d310)

Gli amministratori possono decidere se gli organizzatori della riunione possono scaricare i report sulla partecipazione alle riunioni impostando un criterio per le riunioni di Teams. Per impostazione predefinita, la possibilità di scaricare il report è disattivata. Per la procedura per attivare questa funzionalità, vedere Impostazioni dei criteri [riunione - Report partecipazione alla riunione.](../meeting-policies-in-teams-general.md#meeting-attendance-report)

Se gli organizzatori della riunione devono accedere a più dati sulla partecipazione alla  riunione rispetto al report disponibile all'interno della riunione, è possibile assegnare il ruolo di lettore report in modo che possano accedere ai report di amministrazione di Teams. Per altre informazioni, vedere Chi [può accedere ai report attività di Teams.](../teams-activity-reports.md#who-can-access-the-teams-activity-reports)
