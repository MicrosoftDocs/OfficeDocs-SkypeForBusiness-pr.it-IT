# <a name="toll-free-dialing-restrictions-in-panama"></a>Restrizioni per la composizione dei numeri verde in Panama

[Trova un altro paese o area](../toll-free-dialing-limitations-and-restrictions.md)geografica o scopri di più sulle limitazioni e restrizioni di composizione dei numeri verde.


|**Dettagli**|**Descrizione**|
|:-----|:-----|
|Sono disponibili le chiamate a numero verde?  <br/> |Sì  <br/> |
|Formati di composizione  <br/> | 00-800-XXX-XXXX <br/> |
|Codice ISO del paese/area geografica  <br/> |PA  <br/> |
|Isola/territorio coperto  <br/> |Non applicabile  <br/> |
|Isola/territorio non coperto  <br/> |Non applicabile  <br/> |
|Disponibilità di linee cablate/fisse  <br/> |Solo da C&amp;W.  <br/> |
|Disponibilità rete mobile  <br/> |No  <br/> |
|Commenti, restrizioni e problemi di accesso  <br/> |Il cliente deve comporre il numero 011 (il codice internazionale panamá) prima del numero verde.  <br/> |
