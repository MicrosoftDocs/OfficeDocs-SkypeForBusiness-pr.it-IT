# <a name="customize-teams-phone-system-with-calling-plan"></a>Personalizzare il sistema telefonico di Teams con il piano per chiamate

La [guida alla configurazione del sistema telefonico di Teams con piano](set-up-overview.md) per chiamate consente di avviare il bundle del sistema telefonico con piano per chiamate con le opzioni di base in modo da poterlo usare immediatamente. Queste opzioni potrebbero essere sufficienti per le aziende di piccole dimensioni che sono agli inizi o se si sta eseguendo un progetto pilota in un'organizzazione più grande. Si ottengono nuovi numeri di telefono e un menu di chiamata.

Tuttavia, probabilmente vorrai personalizzare il sistema telefonico con il piano per le chiamate per soddisfare al meglio le esigenze della tua azienda. Ad esempio, probabilmente vorrai spostare i numeri di telefono esistenti (denominati portabilità del numero) nel sistema telefonico con piano per chiamate.

| Attività        | Descrizione       |
|-------------|-------------------|
| [Spostare i numeri di telefono nel sistema telefonico con il piano per chiamate](port-phone-numbers.md) | Trasferisci i tuoi numeri di telefono esistenti nel sistema telefonico con il piano per chiamate. |
| [Configurare i criteri di chiamata](set-up-policies.md) | Controllare se gli utenti possono effettuare chiamate private, usare l'inoltro di chiamata, delegare le chiamate e così via. |
| [Creare utenti aggiuntivi](create-users.md) | Aggiungere altri utenti e assegnare loro numeri di telefono nuovi o esistenti. |
| [Configurare le code delle chiamate](./create-a-phone-system-call-queue-smb.md) | Creare una coda in cui i chiamanti possono attendere il prossimo agente disponibile. |
