# <a name="customize-teams-phone-system-with-calling-plan"></a>Personalizzare Teams Sistema telefonico con piano chiamate

La [guida Teams Sistema telefonico configurazione](set-up-overview.md) del piano per le chiamate consente di Sistema telefonico il pacchetto piano per chiamate con le opzioni di base, in modo da poterlo usare immediatamente. Queste opzioni potrebbero essere sufficienti per le aziende di piccole dimensioni che sono agli inizi o se si sta eseguendo un progetto pilota in un'organizzazione più grande. Si ottengono nuovi numeri di telefono e un menu di chiamata.

Ma è probabile che tu voglia personalizzare i Sistema telefonico piano per chiamate per soddisfare meglio le tue esigenze aziendali. Ad esempio, è probabile che tu voglia spostare i numeri di telefono esistenti (chiamati portabilità dei numeri) in Sistema telefonico piano chiamate.

| Attività        | Descrizione       |
|-------------|-------------------|
| [Spostare i numeri di telefono in Sistema telefonico piano chiamate](port-phone-numbers.md) | Porta i tuoi numeri di telefono esistenti Sistema telefonico piano chiamate. |
| [Configurare i criteri di chiamata](set-up-policies.md) | Controllare se gli utenti possono effettuare chiamate private, usare l'inoltro di chiamata, delegare le chiamate e così via. |
| [Creare utenti aggiuntivi](create-users.md) | Aggiungere altri utenti e assegnare loro numeri di telefono nuovi o esistenti. |
| [Configurare le code delle chiamate](./create-a-phone-system-call-queue-smb.md) | Creare una coda in cui i chiamanti possono attendere il prossimo agente disponibile. |
