# <a name="user-migration"></a>Migrazione degli utenti

Una procedura consigliata generalmente accettata per le migrazioni consiste nel creare diversi utenti di test e usarli per eseguire test dei sistemi. Dopo aver spostato e testato questi account, è consigliabile identificare un gruppo di utenti della produzione pilota e spostare gli account e eseguire i test di convalida. Quando si ottengono risultati soddisfacenti, è possibile trasferire il resto degli utenti alla nuova distribuzione.
  
Per altre informazioni sull'abilitazione degli utenti per Skype for Business Server 2019, vedere la documentazione relativa alla distribuzione di Skype for Business Server 2019.
