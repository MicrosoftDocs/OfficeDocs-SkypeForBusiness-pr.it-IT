# <a name="administering-servers-after-migration"></a>Amministrazione dei server dopo la migrazione

In linea di massima, è necessario utilizzare lo strumento di amministrazione corrispondente alla versione del server che si desidera gestire. Non è possibile installare strumenti di amministrazione da versioni diverse nello stesso computer.
  
> [!IMPORTANT]
> Dopo aver distribuito Skype for Business Server pool pilota, non è possibile utilizzare Generatore di topologie o Il Pannello di controllo per gestire le risorse 2019. È necessario utilizzare gli strumenti 2019 per gestire tutte le risorse della versione corrente e precedente.
