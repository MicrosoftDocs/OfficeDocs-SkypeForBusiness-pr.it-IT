# <a name="phase-4-move-test-users-to-the-pilot-pool"></a>Fase 4: trasferire gli utenti di test nel pool pilota

È possibile trasferire un singolo utente o gruppi di utenti alla nuova distribuzione di Microsoft Skype for Business Server 2019 usando i due metodi seguenti: Pannello di controllo di Skype for Business Server e Skype for Business Server Management Shell. Gli argomenti di questa sezione descrivono le attività che è necessario completare durante la distribuzione pilota, nonché prima di spostare la distribuzione di Skype for Business Server 2019 da una distribuzione pilota a una distribuzione a livello di produzione.
