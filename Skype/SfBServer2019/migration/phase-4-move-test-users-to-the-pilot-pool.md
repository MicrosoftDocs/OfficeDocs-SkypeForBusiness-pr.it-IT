# <a name="phase-4-move-test-users-to-the-pilot-pool"></a>Fase 4: spostare gli utenti di test nel pool pilota

È possibile spostare un singolo utente o gruppo di utenti nella nuova distribuzione di Microsoft Skype for Business Server 2019 utilizzando i due metodi seguenti: Skype for Business Server Pannello di controllo e Skype for Business Server Management Shell. Negli argomenti di questa sezione vengono descritte le attività che è necessario completare durante la distribuzione pilota e prima di spostare la distribuzione di Skype for Business Server 2019 da una distribuzione pilota a una distribuzione a livello di produzione.
