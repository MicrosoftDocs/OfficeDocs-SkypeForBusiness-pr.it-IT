# <a name="view-current-users-in-legacy-pool"></a>Visualizzare gli utenti correnti nel pool legacy

Prima di imparare i diversi modi in cui è possibile spostare gli utenti tra i pool, è innanzitutto necessario determinare quali utenti sono presenti nel pool legacy. La colonna **pool di registrazione** identifica gli utenti configurati per il pool legacy. Questi sono gli utenti di test che verranno spostati nel pool di Skype for Business Server 2019.
  
### <a name="to-see-the-list-of-users-in-the-legacy-pool"></a>Per visualizzare l'elenco degli utenti nel pool legacy

1. Accedere al front end server legacy con un account membro del gruppo RTCUniversalServerAdmins o membro del ruolo amministrativo CsAdministrator o CsUserAdministrator.
    
2. Aprire il **Pannello di controllo di Skype for Business Server**.
    
3. Fare clic su **utenti**, fare clic su **Cerca**e quindi su **trova**.
