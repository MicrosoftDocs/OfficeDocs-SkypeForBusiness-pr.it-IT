# <a name="user-statistics-and-organization-information"></a>Statistiche sugli utenti e informazioni sull'organizzazione

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

Il dashboard di Skype for Business dà una rapida visione d'insieme di come è configurato il servizio per la tua azienda. Se non hai ancora configurato Skype for Business, consulta [Configurare Skype for Business online](set-up-skype-for-business-online.md) per avere istruzioni dettagliate.
  
## <a name="how-to-change-skype-for-business-online-user-statistics"></a>Modificare le statistiche degli utenti di Skype for Business online



|**Proprietà** <br/> |**Come modificare** <br/> |
|:-----|:-----|
|Utenti di Skype for Business Online  <br/> |Per aggiungere o impedire agli utenti di usare l'app Skype for Business, vedere Assegnare o rimuovere licenze per [Microsoft 365 per](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) le aziende o Rimuovere licenze dagli utenti di Microsoft 365 per [le aziende](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1).  <br/> Per un elenco completo delle attività da eseguire quando un utente lascia l'azienda, vedere Rimuovere un ex [dipendente da Microsoft 365](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1).  <br/> |
|Utenti abilitati per le audioconferenze  <br/> |Per abilitare le persone per le audioconferenze, vedere Provare o acquistare [audioconferenza in](../audio-conferencing-in-office-365/try-or-purchase-audio-conferencing-in-office-365.md)Microsoft 365 o Office 365 .  <br/> |
|Utenti nel cloud  <br/> |Questo è il numero di Skype for Business online che hanno Microsoft 365 o Office 365 account.  <br/> Per aggiungere persone all'abbonamento Microsoft 365 o Office 365, ovvero creare account Microsoft 365 o Office 365, vedere Aggiungere utenti singolarmente o in blocco a [Office 365 -](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec)Guida per gli amministratori.  <br/> |
|Utenti sincronizzati e ospitati online  <br/> |Numero di utenti ibridi creati in una distribuzione locale Skype for Business Server ma sincronizzati con Microsoft 365 o Office 365.  <br/> Per impostare la connettività ibrida in Skype for Business, consulta [Pianificare connettività ibrida tra Skype for Business Server e Skype for Business Online](../../SfbHybrid/hybrid/configure-hybrid-connectivity.md?bc=%2fSkypeForBusiness%2fbreadcrumb%2ftoc.json&toc=%2fSkypeForBusiness%2ftoc.json)  <br/> |
   
> [!NOTE]
> La voce Non disponibile indica che non sono presenti informazioni o che si sono verificati errori. 
  
## <a name="how-to-change-information-about-your-organization"></a>Come modificare le informazioni sulla propria organizzazione



|**Proprietà** <br/> |**Come modificare** <br/> |
|:-----|:-----|
|Nome organizzazione  <br/> |Questo è il nome immesso al momento dell'Microsoft 365 o Office 365. Per cambiarlo, vedi [Modificare indirizzo, contatto tecnico e altro della propria organizzazione](https://support.office.com/article/a36e5a52-4df2-479e-bb97-9e67b8483e10).  <br/> |
|ID organizzazione  <br/> |Non è possibile modificare l'ID organizzazione.  <br/> |
|Data creazione organizzazione  <br/> |Non è possibile modificare questo valore.  <br/> |
|Sincronizzazione di Active Directory attiva  <br/> |Indica se è in corso la sincronizzazione dell'organizzazione con un'altra organizzazione Skype for Business online locale.  <br/> Per configurare Active Directory locale, vedere Configurare la sincronizzazione della directory per Microsoft 365 [o Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
|Domini organizzazione  <br/> |Quando si configurano inizialmente Microsoft 365 o Office 365, questi sono i domini registrati.  <br/> Per aggiungere un dominio, vedere [Aggiungere più domini a](https://support.office.com/article/2d2fa996-b760-411d-a5cc-190d63f13207)Microsoft 365 o Office 365 .  <br/> Per rimuovere un dominio, vedere [Rimuovere un dominio da](https://support.office.com/article/f09696b2-8c29-4588-a08b-b333da19810c)Microsoft 365 o Office 365 .  <br/> |
