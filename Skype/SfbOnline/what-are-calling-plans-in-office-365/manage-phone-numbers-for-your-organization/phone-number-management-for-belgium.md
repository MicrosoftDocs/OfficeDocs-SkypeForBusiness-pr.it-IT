# <a name="phone-number-management-for-belgium"></a>Gestione dei numeri di telefono per il Belgio

Usa questa tabella per trovare informazioni su come ottenere e gestire i numeri di telefono in Belgio per Skype for Business online e Microsoft Teams. 
  
Per ulteriori informazioni, consulta [Gestire i numeri di telefono per la propria organizzazione](manage-phone-numbers-for-your-organization.md).
  
|**Cosa vuoi fare?**|**Scarica i moduli corretti e inviaceli**|
|:-----|:-----|
|Ottenere nuovo utente numeri di telefono che non sono incluse le Skype per interfaccia di amministrazione di Business.   <br/> |In Belgio non è possibile ottenere numeri di telefono per la propria organizzazione utilizzando l'interfaccia di amministrazione Skype for Business, devi inviare una richiesta di nuovo numero di telefono. <br/><br/>Scarica uno di questi moduli: <br/> [Richiesta di nuovo numero di telefono per il Belgio (numeri geografici) (v.3.0)(de-be)](../../downloads/new-number-request-forms/new-phone-number-request-for-belgium-(geographic-numbers)-(v.3.0)-(de-be).pdf) <br/> [Richiesta di nuovo numero di telefono per il Belgio (numeri geografici) (v.3.0)(fr-be)](../../downloads/new-number-request-forms/new-phone-number-request-for-belgium-(geographic-numbers)-(v.3.0)-(fr-be).pdf) <br/> [Richiesta di nuovo numero di telefono per il Belgio (numeri geografici) (v.3.0)(nl-be)](../../downloads/new-number-request-forms/new-phone-number-request-for-belgium-(geographic-numbers)-(v.3.0)-(nl-be).pdf) <br/>  <br/>Compilalo e [inviacelo](mailto:ptneu@microsoft.com).  <br/> 
|Ottenere nuovi numeri di telefono del servizio che non sono incluse le Skype per interfaccia di amministrazione di Business. <br/> |<br/>Scarica uno di questi moduli:<br/>[Nuova richiesta numerico di telefono per il Belgio (numeri assistenza) (v. 1.2) (de-essere)](../../downloads/new-number-request-forms/New-phone-number-request-for-belgium-(service-numbers)-(v.1.2)-(de-be).pdf)<br/>[Nuova richiesta numerico di telefono per il Belgio (numeri assistenza) (v. 1.2) (nl-essere)](../../downloads/new-number-request-forms/New-phone-number-request-for-belgium-(service-numbers)-(v.1.2)-(nl-be).pdf) <br/> [Nuova richiesta numerico di telefono per il Belgio (numeri assistenza) (v. 1.2) (fr-essere)](../../downloads/new-number-request-forms/New-phone-number-request-for-belgium-(service-numbers)-(v.1.2)-(fr-be).pdf)<br/><br/>Compilalo e [inviacelo](mailto:ptneu@microsoft.com).
|Devi far trasferire a noi numeri di telefono.  <br/> | Scaricare uno di questi moduli. <br/> <br/> I numeri di telefono utente: <br/>[Lettera di autorizzazione (LOA) per il Belgio (numeri geografici) (v.1.0) (de-essere)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(geographic-numbers)-(v.1.0)-(de-be).pdf) <br/> [Lettera di autorizzazione (LOA) per il Belgio (numeri geografici) (v.1.0) (fr-essere)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(geographic-numbers)-(v.1.0)-(fr-be).pdf) <br/> [Lettera di autorizzazione (LOA) per il Belgio (numeri geografici) (v.1.0) (nl-essere)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(geographic-numbers)-(v.1.0)-(nl-be).pdf)  <br/>  <br/><br/> Per i numeri verdi: <br/>[Lettera di autorizzazione (LOA) per il Belgio (numeri verdi) (v.1.0) (de-essere)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(toll-free-numbers)-(v.1.0)-(de-be).pdf) <br/> [Lettera di autorizzazione (LOA) per il Belgio (numeri verdi) (v.1.0) (fr-essere)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(toll-free-numbers)-(v.1.0)-(fr-be).pdf) <br/> [Lettera di autorizzazione (LOA) per il Belgio (numeri verdi) (v.1.0) (nl-essere)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(toll-free-numbers)-(v.1.0)-(nl-be).pdf) <br/> <br/>   Compilalo e [inviacelo](mailto:ptneu@microsoft.com). <br/> |

|Devi cambiare il numero di un utente in un numero di servizio.  <br/> |[Inviaci](mailto:ptneu@microsoft.com) l'elenco dei numeri di telefono che vuoi convertire in numeri di servizio. <br/> |
