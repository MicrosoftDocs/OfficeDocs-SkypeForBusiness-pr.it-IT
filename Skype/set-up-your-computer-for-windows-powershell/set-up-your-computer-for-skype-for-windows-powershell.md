# Configurare il computer per Skype per Business Online gestione usando Windows PowerShell

> [!IMPORTANT]
> Il presente articolo è stato tradotto automaticamente, vedere la dichiarazione di non responsabilità.  
  
Gestione remota di Skype for Business online utilizzando Microsoft PowerShell è supportata solo nei computer che esegue una delle seguenti sistemi operativi a 64 bit:
  
- Windows 10
    
- Windows 8.1
    
- Windows 8
    
- Windows Server 2012 R2
    
- Windows Server 2012
    
- Windows Server 2008
    
- Windows 7
    
Oltre il sistema operativo supportato, il computer deve anche eseguire le operazioni seguenti:
  
- PowerShell
    
- Modulo del connettore di Skype for Business Online
    
È possibile trovare informazioni sul download e installazione di questi elementi negli argomenti seguenti:
  
- [Download e installazione del modulo Skype for Business Online Connector](download-and-install-the-skype-for-business-online-connector-module.md)
    
- [Download e installazione di Windows PowerShell 3.0](download-and-install-windows-powershell-3-0.md)
