# <a name="toll-free-dialing-restrictions-in-indonesia"></a>Restrizioni per comporre il numero verde Indonesia

[Trovare un altro paese](../what-are-calling-plans-in-office-365/toll-free-dialing-limitations-and-restrictions.md), o ulteriori informazioni sulle limitazioni di comporre il numero verde e restrizioni.


|**Ulteriori informazioni**|**Descrizione**|
|:-----|:-----|
|È il numero verde chiamata disponibili?  <br/> |Sì  <br/> |
|Formati di composizione  <br/> | 001 803 CCC XXXX <br/>  007 803 CCC XXXX <br/> |
|Codice ISO paese/area geografica  <br/> |ID  <br/> |
|Isola/territorio coperto  <br/> |Non applicabile  <br/> |
|Non vengono illustrato isola/territorio  <br/> |Non applicabile  <br/> |
|Disponibilità di riga cablata fisso  <br/> |Sì  <br/> |
|Disponibilità della rete per dispositivi mobili  <br/> |Parziale. Vedere i commenti.  <br/> |
|Commenti, limitazioni e problemi di accesso  <br/> |Per maggiore copertura sono disponibili due formati dei numeri. Il formato XXXX CCC 007 803 formato numero è possibile accedere dalla rete Telkom PT, tra cui Telkomsel e XL per dispositivi mobili. Il formato numero XXXX CCC 001 803 è possibile accedere dalla rete Indosat. Per dispositivi mobili: matrice cellulare (prefisso 0855, 0858, 0815 e 0816) solo.  <br/> |
