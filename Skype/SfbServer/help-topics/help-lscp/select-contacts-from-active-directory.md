# <a name="select-contacts-from-active-directory"></a>Selezionare contatti da Active Directory
 
Usare la finestra di dialogo **Seleziona da Active Directory** per trovare i contatti che si desidera abilitare per Skype for business. Nella casella di ricerca digitare l'indirizzo SIP o il nome visualizzato del contatto che si desidera trovare. Per recuperare tutti i contatti disponibili, lasciare vuota la casella di ricerca.
