# <a name="uncommitted-voice-configuration-settings-review"></a>Impostazioni di configurazione vocale non salvate: Revisione
 
Usare la finestra di dialogo **impostazioni di configurazione vocale** non impegnata per esaminare tutte le modifiche apportate. Scorrere la finestra di dialogo per esaminare tutte le modifiche di cui non è stato eseguito il commit oppure fare clic sulle frecce per espandere o comprimere le diverse sezioni. Al termine, fare clic su **Chiudi**. Dopo aver chiuso questa finestra di dialogo, è possibile modificare le modifiche o eseguirne il commit.
  
Per informazioni dettagliate sulle diverse procedure che è possibile eseguire tramite il pannello di controllo di Skype for Business Server, vedere [gestire Skype for Business server 2015](../../manage/manage.md).
