# <a name="select-routes"></a>Selezionare itinerari
 
Usare la finestra di dialogo **Seleziona Route** per trovare le route vocali esistenti che è possibile usare.
  
> [!NOTE]
> Per recuperare tutte le route vocali disponibili, non digitare nulla nella casella di ricerca e quindi fare clic sull'icona di ricerca. 
  
Per informazioni dettagliate sulle diverse procedure che è possibile eseguire tramite il pannello di controllo di Skype for Business Server, vedere [gestire Skype for Business server 2015](../../manage/manage.md).
