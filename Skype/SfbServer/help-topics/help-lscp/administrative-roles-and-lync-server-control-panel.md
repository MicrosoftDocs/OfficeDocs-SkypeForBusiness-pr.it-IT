Con RBAC, i privilegi amministrativi vengono concessi assegnando gli utenti a ruoli amministrativi predefiniti. Skype for Business Server include un ricco set di ruoli amministrativi predefiniti e consente inoltre di creare nuovi ruoli e specificare un elenco personalizzato di cmdlet per ogni nuovo ruolo. È inoltre possibile aggiungere script di cmdlet alle attività consentite sia dei ruoli predefiniti che di quelli RBAC. Per informazioni dettagliate, inclusi i ruoli amministrativi predefiniti disponibili in Skype for Business Server, vedere controllo di accesso basato sui ruoli.
