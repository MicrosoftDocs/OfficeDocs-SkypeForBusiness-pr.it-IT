Skype for Business Server include un set completo di ruoli amministrativi incorporati e consente inoltre di creare nuovi ruoli e specificare un elenco personalizzato di cmdlet per ogni nuovo ruolo. Rende inoltre possibile l'aggiunta di script di cmdlet alle attività consentite sia dei ruoli RBAC predefiniti che di quelli personalizzati. Per informazioni dettagliate, inclusi i ruoli amministrativi predefiniti disponibili in Skype for Business Server, vedere Role-Based Access Control.
