# <a name="select-exchange-um-auto-attendant"></a>Selezionare un operatore automatico di messaggistica unificata di Exchange
 
Usare la finestra di dialogo **Seleziona Operatore automatico messaggistica unificata di Exchange** per selezionare un operatore automatico di messaggistica unificata di Exchange che gestirà le chiamate in arrivo a un intervallo di numeri non assegnati.
  
Nella casella di ricerca digitare tutto il nome dell'operatore automatico di messaggistica unificata di Exchange che si vuole usare o parte di esso. Per visualizzare tutti gli operatori automatici di messaggistica unificata di Exchange disponibili, lasciare vuota la casella di ricerca e fare clic sull'icona di ricerca.
  
Nell'elenco di operatori automatici di messaggistica unificata di Exchange restituito dalla ricerca selezionare quello che si vuole usare per gestire le chiamate in arrivo all'intervallo di numeri non assegnati.
