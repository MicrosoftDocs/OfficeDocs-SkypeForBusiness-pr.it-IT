# <a name="select-exchange-um-auto-attendant"></a>Selezionare operatore automatico di messaggistica unificata di Exchange
 
Usare la finestra di dialogo **Seleziona operatore automatico di messaggistica unificata di Exchange** per selezionare un operatore automatico di messaggistica unificata di Exchange che gestirà le chiamate in arrivo in un intervallo di numeri non assegnati.
  
Nella casella di ricerca digitare tutto o parte del nome dell'operatore automatico di messaggistica unificata di Exchange che si vuole usare. Per restituire tutti gli operatori automatici di messaggistica unificata di Exchange disponibili, abbandonare la casella di ricerca e fare clic sull'icona di ricerca.
  
Nell'elenco degli operatori automatici di messaggistica unificata di Exchange restituiti dalla ricerca selezionare l'operatore automatico che si vuole usare per gestire le chiamate in arrivo all'intervallo di numeri non assegnati.
