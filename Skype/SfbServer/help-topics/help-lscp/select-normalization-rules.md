# <a name="select-normalization-rules"></a>Selezionare regole di normalizzazione
 
Le regole di normalizzazione utilizzano le espressioni regolari di .NET Framework per tradurre i numeri di telefono composti in formato E. 164. A ogni dial plan deve essere assegnata una o più regole di normalizzazione.
  
Per utilizzare una regola di normalizzazione esistente, selezionarla nell'elenco e quindi fare clic su **OK**.
  
Per informazioni dettagliate sulle diverse procedure che è possibile eseguire utilizzando il pannello di controllo di Skype for Business Server, vedere [Manage Skype for Business server 2015](../../manage/manage.md).
