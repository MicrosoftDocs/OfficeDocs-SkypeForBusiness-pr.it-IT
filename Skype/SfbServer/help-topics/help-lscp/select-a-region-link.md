# <a name="select-a-region-link"></a>Selezionare un collegamento area
 
Utilizzare la finestra di dialogo **Collegamento area** per trovare i collegamenti dell'area disponibili nell'ambiente. Per utilizzare un collegamento dell'area esistente, fare clic su un collegamento dell'area nell'elenco e quindi fare clic su **OK**.
  
> [!NOTE]
> Per recuperare tutti i collegamenti area disponibili, lasciare vuota la casella di ricerca e quindi fare clic sull'icona di ricerca.
