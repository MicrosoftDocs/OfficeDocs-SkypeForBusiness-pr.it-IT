Per ulteriori informazioni sull'aggiunta di un pool di server Director, vedere [Scenarios for the Director](/previous-versions/office/lync-server-2013/lync-server-2013-scenarios-for-the-director) e [Setting Up the Director](/previous-versions/office/lync-server-2013/lync-server-2013-setting-up-the-director).
