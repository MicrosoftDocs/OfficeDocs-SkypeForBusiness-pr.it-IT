# <a name="add-mediation-server-nexthop"></a>Aggiungere l'hop successivo di Mediation Server
 
Per definire l'hop successivo per un Mediation Server che si sta definendo, selezionare dall'elenco a discesa del **pool hop successivo** il server Enterprise Edition o il server Standard Edition che fungerà da hop successivo per il Mediation Server. Fare clic su **Avanti** per continuare.
