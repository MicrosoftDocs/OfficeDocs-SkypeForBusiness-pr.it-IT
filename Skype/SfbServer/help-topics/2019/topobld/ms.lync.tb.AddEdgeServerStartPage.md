# <a name="add-edge-server-support"></a>Aggiungere supporto per il server perimetrale

È possibile aggiungere il supporto per l'accesso degli utenti esterni quando si distribuisce la topologia iniziale o in un secondo momento. Per informazioni dettagliate sull'aggiunta di server perimetrali a una topologia esistente, vedere [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) nella documentazione relativa alla distribuzione di server perimetrali.
