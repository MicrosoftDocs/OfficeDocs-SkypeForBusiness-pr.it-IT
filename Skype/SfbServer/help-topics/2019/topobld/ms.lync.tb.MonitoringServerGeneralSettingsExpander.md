# <a name="monitoring-server-general-settings-expander"></a>Espansione delle impostazioni generali di Monitoring Server
 
In Generatore di topologie è possibile modificare le proprietà di un singolo Monitoring Server facendo clic con il pulsante destro del mouse su Monitoring Server nell'albero della console e poi su **Azione** sulla barra degli strumenti oppure facendo clic su un'attività nel riquadro Azioni e su **Modifica proprietà** e poi modificando una di queste opzioni:
  
- **FQDN**, per modificare il nome di dominio completo del server che si vuole distribuire come Monitoring Server.
    
- **Archivio SQL**, per cambiare l'istanza di SQL Server da usare per il monitoraggio del database di SQL Server. Se si cambia il database di SQL Server di un Monitoring Server, è necessario riavviare il Monitoring Server perché la modifica abbia effetto.
