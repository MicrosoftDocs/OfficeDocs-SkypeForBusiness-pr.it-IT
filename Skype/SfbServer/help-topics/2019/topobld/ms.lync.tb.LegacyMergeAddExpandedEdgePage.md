# <a name="legacy-merge-add-expanded-edge"></a>Unione legacy - Aggiungere Expanded Edge
 
Skype for Business Server non supporta l'Unione di una configurazione del server perimetrale di Office Communications Server 2007 R2 legacy. Il server perimetrale espanso deve essere convertito in una configurazione di server perimetrale consolidato prima dell'unione delle informazioni sul server perimetrale.
