# <a name="external-application-next-hop-settings-expander"></a>Espansione delle impostazioni dell'hop successivo dell'applicazione esterna
 
È possibile specificare il server dell'hop successivo del pool di server applicazioni attendibili selezionando il pool Enterprise Edition front end o il server Standard Edition front end dall'elenco a discesa. Un Director o un pool di Director non è un'opzione valida per l'hop successivo del server applicazioni attendibili e non verrà visualizzato nell'elenco. Fare clic su **OK** per accettare e salvare le modifiche. Fare clic su **Annulla** per rimuovere le modifiche e uscire dalla pagina delle proprietà.
