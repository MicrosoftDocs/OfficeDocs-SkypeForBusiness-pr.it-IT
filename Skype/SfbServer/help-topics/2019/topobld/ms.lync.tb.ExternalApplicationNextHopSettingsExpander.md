# <a name="external-application-next-hop-settings-expander"></a>Espansione delle impostazioni dell'hop successivo dell'applicazione esterna
 
È possibile specificare il server hop successivo del pool di server applicazioni attendibili selezionando nell'elenco a discesa il pool edizione Enterprise Front End o edizione Standard Front End Server. Un Director o un pool di Director non è un'opzione valida per l'hop successivo del server applicazioni attendibili e non verrà visualizzato nell'elenco. Fare **clic su OK** per accettare e salvare le modifiche. Fare clic su **Annulla** per rimuovere le modifiche e uscire dalla pagina delle proprietà.
