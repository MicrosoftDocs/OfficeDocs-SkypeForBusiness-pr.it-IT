# <a name="add-mediation-server"></a>Aggiungere Mediation Server
 
Per definire il Mediation Server, specificare un nome di dominio completo (FQDN) per il pool di Mediation Server. Se si sta creando un pool di server di mediazione, verrà specificato il nome del pool per i server di mediazione che si sta distribuendo. In un altro passaggio sarà possibile definire gli FQDN dei singoli server. Se si distribuisce un singolo Mediation Server, il nome di dominio completo specificato sarà il nome di dominio completo del server di mediazione singolo. Per informazioni dettagliate, vedere [distribuire un Mediation Server in Generatore di topologia in Skype for Business Server](../../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
