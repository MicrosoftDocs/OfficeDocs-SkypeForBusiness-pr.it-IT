# <a name="add-mediation-server"></a>Aggiungere Mediation Server
 
Per definire il Mediation Server, specificare un nome di dominio completo (FQDN) per il relativo pool. Se si sta creando un pool di Mediation Server, sarà necessario specificare il nome del pool per i Mediation Server da distribuire. In un altro passaggio sarà possibile definire gli FQDN dei singoli server. Se si sta distribuendo un solo Mediation Server, l'FQDN da specificare sarà l'FQDN del Mediation Server in questione. Per informazioni dettagliate, [vedere Deploy a Mediation Server in Topology Builder in Skype for Business Server](../../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
