# <a name="archiving-server-scope-node"></a>Nodo dell'ambito del server di archiviazione
 
In Generatore di topologie è possibile eseguire attività correlate al server che esegue l'archiviazione facendo clic con il pulsante destro del mouse sul pool nell'albero della console e scegliendo **modifica proprietà**. Nella pagina **Modifica proprietà** è possibile cambiare il nome di dominio completo (FQDN) del server che esegue l'archiviazione, il database di archiviazione di SQL Server e l'archivio file.
