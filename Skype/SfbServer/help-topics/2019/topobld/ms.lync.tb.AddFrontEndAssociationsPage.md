# <a name="add-front-end-associations"></a>Aggiungere associazioni di Front End Server

È ora possibile abilitare il supporto di funzionalità specifiche che richiedono la distribuzione di altri server associando ruoli del server al pool Front End. L'associazione può anche essere eseguita in un secondo momento. I ruoli del server che possono essere associati a un pool Front End includono:

- A/V Edge Server. Per informazioni dettagliate sull'implementazione di un A/V Edge Server, vedere [Planning for Conferencing](https://technet.microsoft.com/library/983a272a-e1b3-4d70-8f84-836b092fe526.aspx) nella documentazione relativa alla pianificazione.

> [!IMPORTANT]
> Se si abilita ora il supporto di una qualsiasi di queste funzionalità, la struttura della topologia da pubblicare includerà i componenti server necessari per implementare le singole funzionalità selezionate. Per pubblicare una topologia senza che vengano generati errori, è necessario che i computer fisici siano aggiunti al dominio. Se ad esempio si abilita ora il supporto dell'archiviazione, sarà necessario distribuire un server di archiviazione e configurare le opzioni di archiviazione appropriate prima di avviare l'archiviazione delle comunicazioni per l'organizzazione.
