# <a name="file-store-settings-expander"></a>Espansione delle impostazioni dell'archivio file
 
Per modificare le proprietà degli archivi file definiti:
  
Per quanto riguarda il nome della condivisione file, è possibile definire il nome di una nuova condivisione. È quindi necessario creare la condivisione nel server specificato nella proprietà FQDN file server.
  
> [!NOTE]
> Non è possibile modificare l'FQDN del server specificato nella proprietà FQDN file server. Se la condivisione file dell'archivio file deve essere posizionata in un altro server, sarà necessario creare una nuova voce per l'archivio file.
