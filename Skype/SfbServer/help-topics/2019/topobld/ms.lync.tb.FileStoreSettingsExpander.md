# <a name="file-store-settings-expander"></a>Espansione delle impostazioni dell'archivio file
 
Per modificare le proprietà degli archivi di file definiti:
  
Per il nome della condivisione file, è possibile definire il nome di una nuova condivisione. Devi quindi creare la condivisione nel server denominata nella proprietà nome di dominio completo (FQDN) del file server.
  
> [!NOTE]
> Non è possibile modificare il nome di dominio completo del server specificato nella proprietà FQDN file server. Se è necessario inserire la condivisione file in un altro server, è possibile creare una nuova voce di archivio file.
