# <a name="skype-for-business-server-create-new-expander"></a>Creazione di un nuovo espansione in Skype for Business Server
 
È possibile creare nuovi ruoli del server in Generatore di topologie facendo clic con il pulsante destro del mouse su **Skype for Business Server,** scegliendo Nuovo e selezionando il ruolo del server da creare.
