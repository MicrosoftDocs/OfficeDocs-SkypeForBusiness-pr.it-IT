# <a name="skype-for-business-server-create-new-expander"></a>Skype for Business Server crea nuovo Expander
 
È possibile creare nuovi ruoli del server nel generatore di topologia facendo clic con il pulsante destro del mouse su **Skype for Business Server**, selezionando **nuovo**e selezionando il ruolo del server da creare.
