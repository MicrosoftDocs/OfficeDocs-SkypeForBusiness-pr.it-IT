# <a name="skype-for-business-server-create-new-expander"></a>Skype for Business Server Crea nuovo espansione
 
È possibile creare nuovi ruoli del server nel Generatore di topologie facendo clic con il pulsante destro del mouse su **Skype for Business Server**, scegliendo Nuovo e selezionando il ruolo del server da creare.
