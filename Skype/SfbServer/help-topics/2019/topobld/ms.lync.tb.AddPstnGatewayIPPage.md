# <a name="define-the-ip-address-of-the-ip-or-pstn-gateway"></a>Definire l'indirizzo IP del gateway PSTN o IP

Si definiscono gli indirizzi IPv4 o IPv6 per il gateway PSTN (IP o Phone Switched Telephone Network) selezionando **Abilita IPv4** o **Enable IPv6**. Puoi scegliere di **usare tutti gli indirizzi IP configurati** o **limitare l'utilizzo del servizio agli indirizzi IP selezionati**.

Se si seleziona **Usa tutti gli indirizzi IP configurati**, verranno usati tutti gli indirizzi configurati nel gateway. Facoltativamente, selezionare **limita l'utilizzo del servizio agli indirizzi IP selezionati**, che consente la voce dell' **indirizzo IP PSTN** . Immettere l'indirizzo IPv4 o IPv6 assegnato per il gateway.
