# <a name="add-director-machine"></a>Aggiungere il computer Director
 
Specificare il nome di dominio completo (FQDN) di ogni computer che si vuole aggiungere come Director in questo pool. Dopo aver aggiunto un computer all'elenco, è possibile aggiornare l'FQDN del computer o rimuoverlo dal pool in qualsiasi momento prima della pubblicazione della topologia. Dopo aver pubblicato la topologia, la modifica dell'FQDN richiede l'eliminazione del server in Generatore di topologia e l'aggiunta di un nuovo server al pool con il nuovo nome di dominio completo.
