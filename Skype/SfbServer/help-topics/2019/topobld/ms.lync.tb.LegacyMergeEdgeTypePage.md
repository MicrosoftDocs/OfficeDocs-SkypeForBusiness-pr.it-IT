# <a name="legacy-merge-edge-type"></a>Unione legacy - Tipo di server perimetrale
 
Per un **Server perimetrale singolo** o un **Server perimetrale consolidato con carico bilanciato**, fare clic su **Avanti** per specificare le informazioni di configurazione. Per un **Edge espanso**, è necessario modificare la configurazione di ogni server in una configurazione perimetrale consolidata prima che possa essere unita.
