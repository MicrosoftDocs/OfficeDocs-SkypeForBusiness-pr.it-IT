Dopo aver aggiunto un computer all'elenco, è possibile aggiornare il nome di dominio completo del computer o rimuoverlo dal pool, in qualsiasi momento prima della pubblicazione della topologia. Dopo aver pubblicato la topologia, la modifica dell'FQDN richiede l'eliminazione del server in Generatore di topologia e l'aggiunta di un nuovo server al pool con il nuovo nome di dominio completo. Per informazioni dettagliate sulla progettazione della topologia di un pool di Edge, vedere definire la topologia di Edge nella documentazione relativa alla distribuzione di Edge.
