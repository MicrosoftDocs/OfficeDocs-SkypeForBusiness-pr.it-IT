# <a name="legacy-merge-add-machine"></a>Unione legacy - Aggiungere un computer
 
Per definire i computer che costituiscono il pool di server perimetrali, immettere il nome di dominio completo (FQDN) di almeno un computer che fa parte del pool perimetrale consolidato con bilanciamento del carico legacy.
