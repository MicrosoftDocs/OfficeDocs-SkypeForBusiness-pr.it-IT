<span data-ttu-id="eff0d-105">Modificare le proprietà del server Office Web Apps configurato.</span><span class="sxs-lookup"><span data-stu-id="eff0d-105">You edit the properties of the configured Office Web Apps Server.</span></span> <span data-ttu-id="eff0d-106">È possibile modificare le proprietà seguenti:</span><span class="sxs-lookup"><span data-stu-id="eff0d-106">The following properties are available to edit:</span></span>
