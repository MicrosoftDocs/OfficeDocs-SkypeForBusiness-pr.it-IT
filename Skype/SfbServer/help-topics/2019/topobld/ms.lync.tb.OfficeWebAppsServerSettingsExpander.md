# <a name="edit-office-web-apps-server-settings"></a>Modificare le impostazioni di Office Online Server

Si modificano le proprietà del server Office Web Apps configurato. Le proprietà seguenti sono disponibili per la modifica:

 **FQDN del server Office Web Apps**: questa proprietà definisce il nome di dominio completo del server Office Web Apps e deve corrispondere a un record DNS (Domain Name System) host a o AAAA (se viene usato IPv6).

 **URL di individuazione del server di Office Web Apps**: URL (Uniform Resource Locator) per l'accesso client al server Office Web Apps, potrebbe essere necessario modificare questo indirizzo da quello predefinito se il server è posizionato in un'altra area di rete diversa dalla rete interna per la distribuzione.

Selezionare la casella di controllo il **Server Office Web Apps viene distribuito in una rete esterna** se questo server è distribuito nella rete perimetrale o in un'altra area di rete esterna al firewall interno che separa la rete perimetrale, le reti meno attendibili e Internet dalla distribuzione interna.
