# <a name="edge-next-hop-settings-expander"></a>Espansione delle impostazioni dell'hop successivo del server perimetrale

Per impostare o modificare il **Pool hop successivo** che verrà usato dai server perimetrali per comunicare con l'infrastruttura interna, selezionare nell'elenco un server Director, un pool di server Director, un Front End Server o un pool Front End Server. Nell'elenco vengono visualizzati solo i direttori o i server front-end configurati in Generatore di topologia.

Per informazioni dettagliate sulla definizione e sulla configurazione delle impostazioni per i servizi Edge, vedere [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx).
