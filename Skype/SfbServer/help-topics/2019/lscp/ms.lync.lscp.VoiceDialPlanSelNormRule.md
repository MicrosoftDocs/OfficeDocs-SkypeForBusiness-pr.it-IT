# <a name="select-normalization-rules"></a>Selezionare regole di normalizzazione
 
Le regole di normalizzazione utilizzano espressioni regolari di .NET Framework per convertire i numeri di telefono composto nel formato E.164. A ogni dial plan deve essere assegnata una o più regole di normalizzazione.
  
Per utilizzare una regola di normalizzazione esistente, selezionarla nell'elenco e quindi fare clic su **OK**.
