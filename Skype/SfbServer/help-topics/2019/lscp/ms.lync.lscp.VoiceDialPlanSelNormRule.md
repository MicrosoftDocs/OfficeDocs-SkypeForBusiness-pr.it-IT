# <a name="select-normalization-rules"></a>Selezionare regole di normalizzazione
 
Nelle regole di normalizzazione vengono utilizzate le espressioni regolari di .NET Framework per convertire nel formato E.164 i numeri di telefono che sono stati composti. A ogni dial plan devono essere assegnate una o più regole di normalizzazione.
  
Per utilizzare una regola di normalizzazione esistente, selezionarla nell'elenco e quindi fare clic su **OK**.
