# <a name="users-view-mobility-policy"></a>Utenti: visualizzare i criteri di mobilità

La finestra di dialogo **Visualizza-criteri di mobilità** consente di visualizzare le proprietà dei criteri di mobilità selezionati.

Per informazioni dettagliate sull'assegnazione di criteri di mobilità, vedere [assegnare un criterio di mobilità per utente](https://technet.microsoft.com/library/d8bf997f-4bc7-48d3-973b-323505f55e9d.aspx) e [configurare i criteri di mobilità](https://technet.microsoft.com/library/595536e0-9bb3-49a3-8d13-1a77351ebc62.aspx).
