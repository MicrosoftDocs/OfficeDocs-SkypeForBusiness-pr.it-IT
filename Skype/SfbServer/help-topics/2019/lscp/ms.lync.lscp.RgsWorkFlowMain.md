# <a name="response-groups-workflow"></a>Flusso di lavoro Response Group

I Response Group sono costituiti da gruppi di agenti, code e flussi di lavoro. I flussi di lavoro di Response Group definiscono le operazioni eseguite quando l'applicazione Response Group riceve una chiamata telefonica.

Nella   -  pagina **flusso di lavoro** Response Groups viene visualizzato un elenco di tutti i flussi di lavoro di Response Group definiti per l'organizzazione.

## <a name="tasks-you-can-perform"></a>Attività che è possibile eseguire

Dalla pagina dei flussi di lavoro di **Response Group** è possibile eseguire le attività seguenti  -   :

- Creare o modificare un flusso di lavoro di un gruppo di risposta

- Creare o modificare un flusso di lavoro interattivo

## <a name="ui-reference"></a>Riferimenti UI

Nell'elenco seguente sono descritti i comandi della pagina.

- **Creare o modificare un flusso di lavoro** Apre lo strumento di configurazione di Response Group per la creazione o la modifica di un flusso di lavoro.

- **Aggiorna** Consente di aggiornare l'elenco dei flussi di lavoro.

L'elenco seguente descrive i campi presenti nella pagina.

- **Nome** Nome univoco assegnato al flusso di lavoro.

- **Servizio** Servizio **ApplicationServer** che ospita il flusso di lavoro.

- **Indirizzo SIP** L'indirizzo SIP del gruppo che risponderà alle chiamate al flusso di lavoro.

- **Telefono cellulare** Il numero di telefono che viene chiamato per raggiungere questo Response Group.

- **Lingua** La lingua utilizzata per il riconoscimento vocale e la sintesi vocale.

- **IVR** Indica se il flusso di lavoro è un gruppo di risposta o un flusso di lavoro interattivo.

- **Attivata** Indica se il flusso di lavoro è attivato per la ricezione di chiamate.

Per informazioni dettagliate sulle caratteristiche e le funzionalità di Response Group, vedere [Plan for the Response Group Application in Skype for Business Server](../../../plan-your-deployment/enterprise-voice-solution/response-group.md) nella documentazione relativa alla pianificazione. Per informazioni dettagliate sull'utilizzo dei flussi di lavoro di Response Group, vedere [Managing Response Group workflows](https://technet.microsoft.com/library/42cfccdd-2844-4875-b4e3-813e1df15f08.aspx) nella documentazione relativa alle operazioni.
