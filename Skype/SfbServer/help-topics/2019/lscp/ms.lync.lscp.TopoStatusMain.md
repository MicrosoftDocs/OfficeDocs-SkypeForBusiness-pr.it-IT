# <a name="topology-status"></a>Topologia: stato

La pagina **topologia**: **stato** consente di visualizzare lo stato dei server nella topologia di Skype for business.

## <a name="tasks-you-can-perform"></a>Attività che è possibile eseguire

È possibile eseguire le attività seguenti nella **topologia**: pagina **stato** :

- [Visualizzare un elenco di computer che eseguono Skype for Business Server o Lync Server](https://technet.microsoft.com/library/44eeec27-8b99-44f0-b0bd-622c12393d34.aspx)

- [View Details About a Service](https://technet.microsoft.com/library/bc8e8202-cd68-47e4-95b2-bb36e51cc124.aspx)

- [Avviare o arrestare i servizi](https://technet.microsoft.com/library/1c70b4ec-9de5-4f7a-a3c9-c0eb76710505.aspx)

- [Impedire nuove connessioni ai servizi](https://technet.microsoft.com/library/977dcc5c-2aac-48ef-86a1-a8d47b4d9e74.aspx)
