# <a name="pin-set-successfully"></a>Impostazione del PIN completata
 
Per usare la posta elettronica per inviare il PIN all'utente, fare clic su **Apri l'applicazione di posta elettronica per inviare il nuovo PIN all'utente**. Per copiare il PIN nel messaggio di posta elettronica, selezionare **Mostra PIN**.
