# <a name="assign-properties"></a>Assegna proprietà
 
Usare la finestra di dialogo **Assegna proprietà** per specificare una descrizione e un ID sito di rete su più subnet contemporaneamente. In **Descrizione**Digitare la descrizione che si vuole usare per le subnet selezionate. In **ID sito di rete**selezionare l'ID sito di rete che si desidera assegnare alle subnet selezionate. Per assegnare queste proprietà alle subnet selezionate, fare clic su **OK**.
