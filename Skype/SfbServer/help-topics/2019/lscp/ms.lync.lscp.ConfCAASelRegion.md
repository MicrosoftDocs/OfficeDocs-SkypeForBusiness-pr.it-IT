# <a name="select-regions"></a>Selezionare aree geografiche
 
Utilizzare la **finestra di dialogo** Seleziona aree per trovare le aree disponibili nell'ambiente. Per utilizzare un'area esistente, selezionarla nell'elenco e quindi fare clic su **OK**.
  
> [!NOTE]
> Per recuperare tutte le aree disponibili, lasciare vuota la casella di ricerca e quindi fare clic sull'icona di ricerca.
