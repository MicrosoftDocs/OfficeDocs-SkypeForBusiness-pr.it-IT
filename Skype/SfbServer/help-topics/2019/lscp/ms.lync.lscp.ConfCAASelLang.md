# <a name="select-languages"></a>Selezionare lingue

Selezionare le lingue che si desidera utilizzare per il numero di accesso per le conferenze telefoniche con accesso esterno. Per informazioni dettagliate, vedere [Create or Modify a Dial-in Conferencing Access Number](/previous-versions/office/lync-server-2013/lync-server-2013-create-or-modify-a-dial-in-conferencing-access-number).
