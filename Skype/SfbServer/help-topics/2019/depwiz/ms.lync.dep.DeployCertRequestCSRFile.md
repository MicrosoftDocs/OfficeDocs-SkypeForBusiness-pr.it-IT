# <a name="certificate-request-certificate-signing-request"></a>Richiesta di certificato (richiesta di firma del certificato)
 
Per creare una richiesta offline, è necessario prima di tutto creare una richiesta di firma del certificato (CSR). Digitare il percorso e il nome del file per la RSI nella casella di testo **nome file** . È possibile fare clic su **Sfoglia** per scegliere un percorso per creare la RSI. Fare clic su **Avanti** per continuare.
  
Fare clic su **indietro** per tornare alla schermata precedente. Fare clic su **Annulla** per uscire senza apportare modifiche al sistema.
  
> [!IMPORTANT]
> Il file creato in questo passaggio viene inviato all'autorità di certificazione (CA) pubblica, alla CA offline interna o inviato all'amministratore della CA per l'elaborazione. Nota il percorso e il nome del file in modo da poterlo recuperare e inviare la RSI alla CA. Il percorso predefinito è la directory documenti della persona che ha eseguito la creazione guidata certificato.
