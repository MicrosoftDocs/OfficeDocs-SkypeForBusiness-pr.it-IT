# <a name="import-certificate-intro"></a>Importa certificato (Intro)
 
Per importare un certificato, è necessario specificare un percorso per il file di certificato. Nella casella di testo **selezionare il file di certificato** è possibile digitare il percorso completo e il nome del file oppure fare clic sul pulsante **Sfoglia** e passare al percorso e al nome del file (in genere, un file con estensione p7b, pfx o CER).
  
Se il certificato contiene una chiave privata, selezionare il **file di certificato della casella di controllo contiene la chiave privata del certificato**. Quando questa casella di controllo è selezionata, l'input di testo della **password** è abilitato. Se si dispone di un certificato con una chiave privata associata, una password viene in genere inserita nella chiave privata quando il certificato viene creato. Si inserisce la password per la chiave privata per consentire l'importazione del certificato e della chiave privata nell'archivio certificati. Dopo avere fornito le informazioni per il percorso del file del certificato e, se necessario, la password della chiave privata, fare clic su **Avanti**.
  
> [!IMPORTANT]
> Se non si conosce la password della chiave privata, l'importazione non riuscirà.
