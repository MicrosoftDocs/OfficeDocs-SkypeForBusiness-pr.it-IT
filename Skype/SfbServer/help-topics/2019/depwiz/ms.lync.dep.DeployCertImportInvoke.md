# <a name="import-certificate-invoke"></a>Importare un certificato (Invoke)
 
Nella pagina **Esecuzione comandi in corso** esaminare le informazioni di riepilogo relative al certificato importato. Nell'elenco a discesa selezionare il file di registro Importazione certificato. Verificare che tale file sia visibile nell'elenco a discesa. Fare clic su **Visualizza registro** per visualizzare i dettagli dell'importazione del certificato. Dopo avere finito di leggere il riepilogo e il file di registro, fare clic su **Fine**.
