# <a name="import-certificate-invoke"></a>Importa certificato (Invoke)
 
Nella pagina **esecuzione dei comandi** esaminare le informazioni di riepilogo per il certificato importato. Nell'elenco a discesa selezionare il file di registro del certificato di importazione. Verificare che il file di log del certificato di importazione sia visibile nell'elenco a discesa. Fare clic su **Visualizza log** per esaminare i dettagli dell'importazione di certificati. Dopo aver completato la revisione del riepilogo e del file di log, fare clic su **fine**.
