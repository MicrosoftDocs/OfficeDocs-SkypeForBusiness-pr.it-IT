# <a name="certificate-assignment-invoke"></a>Assegnazione di un certificato (Invoke)
 
Nella pagina **Esecuzione comandi in corso** vengono visualizzate le azioni eseguite dalle attività per assegnare il certificato al server. Al completamento delle attività, è possibile visualizzare il registro con i dettagli sul processo di assegnazione. Per visualizzare il file di registro, fare clic su **Visualizza registro**.
