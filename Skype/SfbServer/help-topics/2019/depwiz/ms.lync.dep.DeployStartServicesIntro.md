# <a name="startservices-intro"></a>Avviare servizi (Intro)
 
Per avviare i servizi nel server di ruoli di Skype for Business Server che si sta distribuendo, esaminare la schermata introduttiva. Si noti che i comandi emessi per avviare il server Skype for business non attendono l'avvio del servizio. Questo è il massimo sforzo per l'avvio dei servizi, di cui è consigliabile monitorare lo stato effettivo mediante Microsoft Management Console (MMC) dei servizi Windows. Fare clic su **Avanti** per avviare l'attività **Avvia servizi**.
