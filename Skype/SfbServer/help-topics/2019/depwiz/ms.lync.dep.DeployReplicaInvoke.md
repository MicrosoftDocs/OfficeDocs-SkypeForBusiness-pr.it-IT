<span data-ttu-id="106c4-p102">Nel riquadro **Comandi di esecuzione** viene visualizzato lo stato delle attività per l'installazione della replica di sola lettura locale dell'archivio di gestione centrale. While the tasks are running, the task pane is updated as the commands are run. You should review the task status and note any errors or warnings that are displayed. When the tasks are complete, the message **Task status: Completed** is displayed, and the drop-down list for the log files displays the names of the log files. To view the log files that were created, choose a log file to view and click **View Log**. Review all log files for success, errors, and warning messages.</span></span>
