# <a name="install-or-update-skype-for-business-server-system"></a>Installare o aggiornare il sistema Skype for Business Server

Per avviare l'installazione o aggiornare una distribuzione esistente di Skype for Business Server, fare clic su **Installa o Aggiorna Skype for Business Server**. La procedura descritta in questa sezione consente di installare i componenti specifici di un determinato ruolo del server, in base al documento del generatore di topologia. Prima di iniziare questa serie di passaggi, è necessario definire la distribuzione in Generatore di topologia. È necessario essere connessi al computer da distribuire come amministratori locali ed essere membri del gruppo Domain Admins nel dominio in cui si trova il server. Il computer deve essere aggiunto al dominio prima di poter eseguire questo passaggio.

Se si esegue l'aggiornamento di un server attualmente distribuito, è necessario apportare le modifiche desiderate in Generatore di topologie.
