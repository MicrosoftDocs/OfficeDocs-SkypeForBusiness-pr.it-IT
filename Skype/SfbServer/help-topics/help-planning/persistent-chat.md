# <a name="persistent-chat-planning-tool"></a>Chat persistente (strumento di pianificazione)
 
Il server di chat persistente consente agli utenti di partecipare a conversazioni multiparte, basate su argomenti che persistono nel tempo.
  
Per altre informazioni sulla preparazione dell'ambiente per il server della chat persistente, vedere [pianificare il server di chat persistente in Skype for Business server 2015](../../plan-your-deployment/persistent-chat-server/persistent-chat-server.md).
