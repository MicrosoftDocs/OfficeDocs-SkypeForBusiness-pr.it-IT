# <a name="dial-in-conferencing-planning-tool"></a>Servizi di conferenza telefonica con accesso esterno (strumento di pianificazione)
 
Le conferenze telefoniche con accesso esterno di Skype for Business Server 2015 consentono agli utenti di partecipare alla parte audio di una conferenza di Skype for Business Server 2015 usando un telefono PSTN (Public Switched Telephone Network) senza richiedere un provider di servizi di audioconferenza di terze parti.
  
Per informazioni sulla pianificazione dell'abilitazione dei servizi di conferenza telefonica con accesso esterno per l'organizzazione, vedere l'argomento seguente e gli argomenti a cui si collega: 
  
- [Pianificare i servizi di conferenza in Skype for Business Server 2015](../../plan-your-deployment/conferencing/conferencing.md)
