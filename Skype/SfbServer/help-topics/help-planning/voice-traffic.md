# <a name="skype-for-business-server-voice-traffic-planning-tool"></a>Skype for Business Server Strumento di pianificazione del traffico vocale
 
Lo strumento di pianificazione utilizza informazioni sul traffico degli utenti per determinare il numero di porte da utilizzare nella distribuzione. Questo numero determina a sua volta il numero di Mediation Server e gateway che saranno necessari. Per informazioni dettagliate, vedere [Estimating voice usage and traffic for Skype for Business Server 2015](../../plan-your-deployment/capacity/estimating-voice-traffic.md).
