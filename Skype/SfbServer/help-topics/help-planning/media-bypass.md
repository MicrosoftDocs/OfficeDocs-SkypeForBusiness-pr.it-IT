# <a name="media-bypass-planning-tool"></a>Bypass multimediale (strumento di pianificazione)
 
Il bypass multimediale si riferisce alla rimozione del Mediation Server dal percorso multimediale ogni volta che è possibile per le chiamate il cui segnale attraversa il Mediation Server. Per altre informazioni, Vedi [pianificare il bypass multimediale in Skype for Business 2015](../../plan-your-deployment/enterprise-voice-solution/media-bypass.md).
