# <a name="prepare-current-forest-universal-group-location"></a>Preparare la foresta corrente (percorso gruppo universale)
 
Devi indicare il punto in cui devi creare i gruppi universali nella foresta. La pagina **posizione del gruppo universale** usa il dominio in cui il computer a cui è stato effettuato l'accesso è un membro dell'impostazione predefinita. Se si preferisce creare gruppi universali in **un dominio diverso da quello predefinito,** selezionare FQDN del **dominio** e digitare il nome di dominio completo (FQDN) nella casella di testo. È utile fornire un dominio alternativo quando si esegue l'attività di preparazione della foresta da un dominio diverso da quello in cui si vogliono creare i gruppi universali. Fare clic su **Avanti** per continuare.
