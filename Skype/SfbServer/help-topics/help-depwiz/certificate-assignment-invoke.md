# <a name="certificate-assignment-invoke"></a>Assegnazione certificato (Invoke)
 
Nella pagina **comandi in esecuzione** vengono visualizzate le azioni intraprese dalle attività per assegnare il certificato al server. Una volta completate le attività, è possibile visualizzare il log che descrive in dettaglio il processo di assegnazione del certificato. Per visualizzare il file di registro, fare clic su **Visualizza registro**.
