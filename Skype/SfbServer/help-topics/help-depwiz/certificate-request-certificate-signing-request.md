# <a name="certificate-request-certificate-signing-request"></a>Richiesta di certificato (richiesta di firma del certificato)
 
Per creare una richiesta offline, è necessario innanzitutto creare una richiesta di firma del certificato. Digitare il percorso e il nome del file per la richiesta di firma del certificato nella casella di testo **Nome file**. È possibile fare clic su **Sfoglia** per selezionare un percorso in cui creare la richiesta di firma del certificato. Fare clic su **Avanti** per continuare.
  
Fare clic su **Indietro** per tornare alla schermata precedente. Fare clic su **Annulla** per uscire senza apportare modifiche al sistema.
  
> [!IMPORTANT]
> Il file creato in questo passaggio viene inviato all'autorità di certificazione (CA) pubblica o alla CA offline interna oppure viene inoltrato all'amministratore CA per l'elaborazione. Prendere nota del percorso e del nome del file, in modo da poterlo recuperare e inoltrare la richiesta di firma del certificato alla CA. Il percorso predefinito corrisponde alla directory Documenti della persona che esegue la Configurazione guidata certificati.
