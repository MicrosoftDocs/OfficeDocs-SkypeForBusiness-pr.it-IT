# <a name="prepare-current-forrest-intro"></a>Preparare la foresta corrente (Intro)
 
Per avviare l'attività **Prepara foresta**, fare clic su **Avanti**. L'attività **prepara Forest** consente di creare impostazioni globali e di creare gruppi universali per l'uso da Skype for Business Server 2015.
