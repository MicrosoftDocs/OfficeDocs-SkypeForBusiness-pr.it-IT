# <a name="prepare-current-forrest-intro"></a>Preparare la foresta corrente (Intro)
 
Per avviare l'attività **Prepara foresta**, fare clic su **Avanti**. **L'esecuzione dell'attività** Prepara foresta consente di creare impostazioni globali e gruppi universali da utilizzare Skype for Business Server 2015.
