# <a name="certificate-request-online-offline"></a>Richiesta di certificato (Online Offline)
 
Per creare una richiesta ritardata o offline, selezionare **Prepara la richiesta per l'invio posticipato (richiesta di certificato non in linea)**. Per creare una richiesta immediata a un'autorità di certificazione online (in genere un sistema interno dell'infrastruttura a chiave pubblica dell'organizzazione), selezionare **Invia immediatamente la richiesta a un'autorità di certificazione online**.
  
Dopo aver effettuato una selezione, fare clic su **Avanti** per continuare. Se si desidera annullare senza apportare modifiche, fare clic su **Annulla**. Per tornare al passaggio precedente, fare clic su **Indietro**.
