Per definire **l'FQDN di Survivable Branch Appliance**, digitare il nome di dominio completo (FQDN) (come definito in Domain Name System (DNS)) del Survivable Branch Appliance o del Survivable Branch Server nella casella di testo **FQDN** . Fare clic su **Avanti** per continuare.
