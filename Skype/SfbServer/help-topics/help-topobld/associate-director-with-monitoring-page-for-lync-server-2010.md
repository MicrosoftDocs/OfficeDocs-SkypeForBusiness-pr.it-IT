# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Amministratore associato con la pagina di monitoraggio per Lync Server 2010
 
Per configurare un server di monitoraggio e associarlo al pool di Director o Director:
  
In **Monitoring Server** selezionare dall'elenco un server di monitoraggio.
  
Se non sono presenti server di monitoraggio e si vuole creare ora un server di monitoraggio, fare clic su **nuovo**.
  
 **Indietro**: Torna alla finestra di dialogo precedente
  
 **Fine**: consente di eseguire il commit delle modifiche e completare la procedura guidata
  
 **Annulla**: Elimina le modifiche e Annulla la configurazione
  
 **Guida**: Visualizza le informazioni della guida, ad esempio questo argomento
