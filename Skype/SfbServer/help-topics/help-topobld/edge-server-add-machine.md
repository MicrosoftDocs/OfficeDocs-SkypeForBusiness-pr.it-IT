# <a name="edge-server-add-machine"></a>Server perimetrale - Aggiungere un computer

Specificare il nome di dominio completo (FQDN) di ogni computer che si vuole aggiungere come server perimetrale nel pool. Dopo aver aggiunto un computer all'elenco, è possibile aggiornare l'FQDN del computer o rimuoverlo dal pool in qualsiasi momento prima di pubblicare la topologia. Dopo aver pubblicato la topologia, per poter modificare l'FQDN è necessario eliminare il server in Generatore di topologie e poi aggiungere al pool un nuovo server con il nuovo FQDN. Per informazioni dettagliate sulla progettazione della topologia per un pool di server perimetrali, vedere [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) nella documentazione relativa alla distribuzione di server perimetrali.
