# <a name="edge-server-add-machine"></a>Aggiungi computer Edge Server

Specificare il nome di dominio completo (FQDN) di ogni computer che si vuole aggiungere come server perimetrale in questo pool. Dopo aver aggiunto un computer all'elenco, è possibile aggiornare il nome di dominio completo del computer o rimuoverlo dal pool, in qualsiasi momento prima della pubblicazione della topologia. Dopo aver pubblicato la topologia, la modifica dell'FQDN richiede l'eliminazione del server in Generatore di topologia e l'aggiunta di un nuovo server al pool con il nuovo nome di dominio completo. Per informazioni dettagliate sulla progettazione della topologia di un pool di Edge, vedere definire la topologia di [Edge](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) nella documentazione relativa alla distribuzione di Edge.
