# <a name="persistent-chat-next-hop-settings-expander"></a>Espansione delle impostazioni generali dell'hop successivo di Persistent Chat
 
È possibile configurare il **pool dell'hop successivo** di chat persistente selezionando un front end server o un pool Front end dall'elenco.
  
 **OK** Accetta le modifiche apportate nella finestra di dialogo e ne esegue il commit.
  
 **Annulla** Rimuove le modifiche e chiude la finestra di dialogo.
  
 **?** Visualizza questa schermata della Guida.
