# <a name="persistent-chat-next-hop-settings-expander"></a>Espansione delle impostazioni dell'hop successivo di Chat persistente
 
Si configura il **pool hop successivo** della chat persistente selezionando un server front-end o un pool Front-end dall'elenco.
  
 **OK** Consente di accettare e salvare le modifiche nella finestra di dialogo.
  
 **Annulla** Consente di eliminare le modifiche e di chiudere la finestra di dialogo.
  
 **Guida** Consente di visualizzare questa schermata della Guida.
