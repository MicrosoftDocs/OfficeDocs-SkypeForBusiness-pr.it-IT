# <a name="legacy-merge-summary-page"></a>Pagina Riepilogo Unione legacy

La migrazione da Microsoft Office Communications Server 2007 R2 a Lync Server 2013 usa le funzionalità di generatore di topologie per unire le topologie esistenti. La schermata di riepilogo specifica tutte le azioni e le selezioni effettuate durante la configurazione della fusione di Office Communications Server 2007 R2 nella configurazione di generatore di topologia di Lync Server 2013.

Esaminare il riepilogo e determinare se le selezioni sono state definite correttamente. Procedere con la fusione se si è soddisfatti delle selezioni riflesse nel riepilogo.
