# <a name="add-front-end-associations-2010"></a>Aggiungere associazioni front-end 2010

È possibile abilitare il supporto per funzionalità specifiche che richiedono la distribuzione di altri server associando i ruoli del server al pool Front-end in questo momento. L'associazione può anche essere eseguita in un secondo momento. I ruoli del server che possono essere associati a un pool Front End includono:

1. Server di archiviazione. Per informazioni dettagliate sulla pianificazione e l'implementazione del supporto per l'archiviazione, vedere [definizione dei requisiti dell'organizzazione per l'archiviazione](https://technet.microsoft.com/library/ce0fc0f6-7704-4b80-bf19-a1fa9818fc7a.aspx) nella documentazione di pianificazione e [definizione e configurazione di un pool Front-end o di un server Standard Edition](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) e [distribuzione Archiviazione](https://technet.microsoft.com/library/a89edd16-12d5-4602-ad2f-194b47d1188e.aspx) nella documentazione relativa alla distribuzione.

2. Server di monitoraggio. Per informazioni dettagliate sull'implementazione del supporto per il monitoraggio, vedere [pianificazione del monitoraggio](https://technet.microsoft.com/library/26cead5a-183c-42f1-a4b0-0e8d61c6159d.aspx) nella documentazione relativa alla pianificazione e [definire e configurare un pool Front end o un server Standard Edition](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) e [distribuire il monitoraggio](https://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx) nella distribuzione documentazione.

3. A/V Edge service. Per informazioni dettagliate sull'implementazione di un servizio A/V Edge, vedere [Panoramica dell'accesso degli utenti esterni](https://technet.microsoft.com/library/97aded6c-5fa3-4225-95a6-9ad094d61654.aspx).
