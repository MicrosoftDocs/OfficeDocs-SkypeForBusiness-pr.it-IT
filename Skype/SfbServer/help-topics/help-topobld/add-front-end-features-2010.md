# <a name="add-front-end-features-2010"></a>Aggiungere funzionalità di Front End Server 2010

Selezionare ogni caratteristica che si vuole abilitare per il pool Front-end. È necessario distribuire almeno la messaggistica istantanea, che è sempre abilitata e non può essere deselezionata. È inoltre possibile distribuire altre funzionalità, ad esempio servizi di conferenza, VoIP aziendale e controllo di ammissione alle chiamate (CAC). Alcune caratteristiche dipendono dalla topologia.

Per informazioni dettagliate su come specificare le caratteristiche per un server standard o un pool di Enterprise Edition, vedere [definire e configurare un pool Front end o un server Standard Edition](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) nella documentazione relativa alla distribuzione.
