# <a name="associate-front-end-with-av-mcu"></a>Associare Front End con AV MCU
 
A/V Conferencing consente le comunicazioni audio e video in tempo reale tra gli utenti, ovvero se hanno dispositivi client appropriati, come auricolari per conferenze audio e webcam per videoconferenze. Se la distribuzione supporta i servizi di conferenza e si abilitano sia i servizi di conferenza Web che i servizi di conferenza A/V, è possibile collocare l'A/V Conferencing Server con il front end Server) oppure distribuire uno o più server di conferenza A/v autonomo (pool A/V Conferencing). Se si sceglie l'opzione per la distribuzione di un server di conferenza A/V autonomo, è necessario definirla in Generatore di topologie.
  
Tutti i pool di un sito e i pool di più siti centrali possono usare lo stesso A/V Conferencing Server, se l'utilizzo non supera la capacità di un/V Conferencing Server.
