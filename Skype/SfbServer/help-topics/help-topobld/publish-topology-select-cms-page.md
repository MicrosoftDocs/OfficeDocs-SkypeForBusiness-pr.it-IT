# <a name="publish-topology-select-cms-page"></a>Pubblica topologia, pagina Seleziona l'archivio di gestione centrale

È possibile pubblicare la topologia configurata utilizzando Generatore di topologie. Viene chiesto di selezionare da un elenco il front end server o il pool Front End assumerà il ruolo di Holding The Central Management store. Solo un front end server o un pool Front end può contenere questo ruolo in un determinato momento.

Per ulteriori informazioni su ciò che è l'archivio di gestione centrale, vedere [Move the Lync server 2010 Central Management Server to Lync server 2013](https://technet.microsoft.com/library/30cc98f2-1916-4dbe-99d0-8df5368ed3ec.aspx).
