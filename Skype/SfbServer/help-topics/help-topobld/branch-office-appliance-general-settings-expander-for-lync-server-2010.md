# <a name="branch-office-appliance-general-settings-expander-for-lync-server-2010"></a>Espansione delle impostazioni generali di Branch Office Appliance per Lync Server 2010
 
Per modificare le proprietà di Survivable Branch Appliance o Survivable Branch Server, in **Generale** è necessario configurare:
  
- **FQDN**: digitare il nome di dominio completo del Survivable Branch Appliance o del Survivable Branch Server.
    
    **Usa tutti gli indirizzi IP configurati** utilizza gli indirizzi IP configurati nel Survivable Branch Appliance o nel Survivable Branch Server per tutti gli scopi.
    
    **Limita utilizzo servizio a indirizzi IP selezionati**: configurare l'indirizzo distinto che definisce il server e l'indirizzo IP da usare per PSTN.
    
    **Indirizzo IP primario**: l'indirizzo IP viene definito e configurato per tutti gli scopi, ad eccezione delle funzioni associate a PSTN .
    
    **Indirizzo IP PSTN**: indirizzo IP associato alle funzioni PSTN (Public Switched Telephone Network).
    
- Le associazioni **vengono** configurate per garantire che altri ruoli del server siano configurati e associati al Survivable Branch Appliance o al Survivable Branch Server.
    
    **Associare il server di archiviazione** Selezionare nell'elenco il server di archiviazione che si desidera associare al Survivable Branch Appliance o al Survivable Branch Server
    
    Fare **clic su** Nuovo se non è stato creato il server di archiviazione che si desidera associare a questo Survivable Branch Appliance o Survivable Branch Server
    
    **Associare Monitoring Server** Selezionare nell'elenco il Monitoring Server che si desidera associare al Survivable Branch Appliance o al Survivable Branch Server
    
    Fare **clic su** Nuovo se non è stato creato il Monitoring Server che si desidera associare a questo Survivable Branch Appliance o Survivable Branch Server
    
    **Associa pool di server perimetrali (per componenti multimediali)** Selezionare nell'elenco il server perimetrale o il pool di server perimetrali che si desidera associare al Survivable Branch Appliance o al Survivable Branch Server
    
  **OK** Accetta le modifiche apportate nella finestra di dialogo e ne esegue il commit.
  
  **Annulla** Rimuove le modifiche e chiude la finestra di dialogo.
  
  **?** Visualizza questa schermata della Guida.
