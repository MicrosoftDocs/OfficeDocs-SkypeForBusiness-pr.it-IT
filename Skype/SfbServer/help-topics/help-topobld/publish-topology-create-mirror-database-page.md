# <a name="publish-topology-create-mirror-database-page"></a>Pagina Pubblicare una topologia e creare database mirror
 
Questa è la documentazione preliminare ed è soggetta a modifiche. Gli argomenti vuoti sono inclusi come segnaposto.
  
Generatore di topologie analizza tutte le impostazioni di configurazione nel documento della topologia definita. Se sono stati configurati database di SQL Server con mirroring, viene visualizzata una schermata che Visualizza l'archivio mirror, l'archivio principale, il percorso di condivisione file e i percorsi di database. Si seleziona una casella di controllo associata a un archivio mirror del database per creare il database con mirroring e il percorso di condivisione file. Deselezionare una casella di controllo per un mirror del database se non si vuole creare e distribuire il database nel server SQL definito.
  
 Se si seleziona per scegliere di non installare uno o tutti questi database in questo momento, è possibile cancellare i database che non verranno installati. In un secondo momento, scegli di usare il **database di installazione** facendo clic con il pulsante destro del mouse sulla maggior parte degli elementi del nodo in Generatore di topologia
