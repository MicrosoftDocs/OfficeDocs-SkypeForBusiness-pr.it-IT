# <a name="edge-next-hop-settings-expander"></a>Espansione delle impostazioni dell'hop successivo del server perimetrale

Per impostare o modificare il **pool di hop successivo** che i server perimetrali utilizzeranno per comunicare con l'infrastruttura interna, selezionare un server Director, un pool di Director, un front-end o un pool di server front end dall'elenco. Nell'elenco vengono visualizzati solo i Director o i front end server configurati in Generatore di topologie.

Per informazioni dettagliate sulla definizione e sulla configurazione delle impostazioni per i servizi Edge, vedere [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx).
