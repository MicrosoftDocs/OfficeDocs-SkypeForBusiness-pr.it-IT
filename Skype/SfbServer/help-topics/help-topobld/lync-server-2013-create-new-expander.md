# <a name="lync-server-2013-create-new-expander"></a>Espansione delle impostazioni di creazione di nuovi elementi di Lync Server 2013
 
È possibile creare nuovi ruoli del server nel nodo Lync Server 2013 di Generatore di topologie facendo clic con il pulsante destro del mouse su **Lync Server 2013,** scegliere **Nuovo,** quindi selezionare il ruolo del server da creare.
