# <a name="lync-server-2013-create-new-expander"></a>Creare nuova espansione in Lync Server 2013
 
È possibile creare nuovi ruoli del server nel nodo Lync Server 2013 di generatore di topologia facendo clic con il pulsante destro del mouse su **Lync server 2013**, selezionare **nuovo**e quindi selezionare il ruolo del server da creare.
