<span data-ttu-id="e2f45-105">Si modificano le proprietà del server Office Web Apps configurato.</span><span class="sxs-lookup"><span data-stu-id="e2f45-105">You edit the properties of the configured Office Web Apps Server.</span></span> <span data-ttu-id="e2f45-106">Le proprietà seguenti sono disponibili per la modifica:</span><span class="sxs-lookup"><span data-stu-id="e2f45-106">The following properties are available to edit:</span></span>

 <span data-ttu-id="e2f45-107">**FQDN del server Office Web Apps**: questa proprietà definisce il nome di dominio completo del server Office Web Apps e deve corrispondere a un record DNS (Domain Name System) host a o AAAA (se viene usato IPv6).</span><span class="sxs-lookup"><span data-stu-id="e2f45-107">**Office Web Apps Server FQDN**: This property defines the fully qualified domain name of the Office Web Apps Server and should match a domain name system (DNS) host A or AAAA (if IPv6 is being used) record.</span></span>

 <span data-ttu-id="e2f45-108">**URL di individuazione del server di Office Web Apps**: URL (Uniform Resource Locator) per l'accesso client al server Office Web Apps, potrebbe essere necessario modificare questo indirizzo da quello predefinito se il server è posizionato in un'altra area di rete diversa dalla rete interna per la distribuzione.</span><span class="sxs-lookup"><span data-stu-id="e2f45-108">**Office Web Apps Server discovery URL**: The uniform resource locator (URL) for client access to the Office Web Apps Server, you may need to edit this address from its default if the server is placed in another network zone other than the internal network for your deployment.</span></span>
