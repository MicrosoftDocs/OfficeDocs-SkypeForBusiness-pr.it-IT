<span data-ttu-id="e5dc1-105">Si modificano le proprietà del server Office Web Apps configurato.</span><span class="sxs-lookup"><span data-stu-id="e5dc1-105">You edit the properties of the configured Office Web Apps Server.</span></span> <span data-ttu-id="e5dc1-106">Le proprietà seguenti sono disponibili per la modifica:</span><span class="sxs-lookup"><span data-stu-id="e5dc1-106">The following properties are available to edit:</span></span>

 <span data-ttu-id="e5dc1-107">**FQDN del server Office Web Apps**: questa proprietà definisce il nome di dominio completo del server Office Web Apps e deve corrispondere a un record DNS (Domain Name System) host a o AAAA (se viene usato IPv6).</span><span class="sxs-lookup"><span data-stu-id="e5dc1-107">**Office Web Apps Server FQDN**: This property defines the fully qualified domain name of the Office Web Apps Server and should match a domain name system (DNS) host A or AAAA (if IPv6 is being used) record.</span></span>

 <span data-ttu-id="e5dc1-108">**URL di individuazione del server di Office Web Apps**: URL (Uniform Resource Locator) per l'accesso client al server Office Web Apps, potrebbe essere necessario modificare questo indirizzo da quello predefinito se il server è posizionato in un'altra area di rete diversa dalla rete interna per il distribuzione.</span><span class="sxs-lookup"><span data-stu-id="e5dc1-108">**Office Web Apps Server discovery URL**: The uniform resource locator (URL) for client access to the Office Web Apps Server, you may need to edit this address from its default if the server is placed in another network zone other than the internal network for your deployment.</span></span>

<span data-ttu-id="e5dc1-109">Selezionare la casella di controllo il **Server Office Web Apps viene distribuito in una rete esterna** se questo server è distribuito nella rete perimetrale o in un'altra area di rete esterna al firewall interno che separa la rete perimetrale, le reti meno attendibili e Internet dalla distribuzione interna.</span><span class="sxs-lookup"><span data-stu-id="e5dc1-109">Select the check box **Office Web Apps Server is deployed in an external network** if this server is deployed in your perimeter network or other network zone that is outside of your internal firewall separating the perimeter network, less trusted networks, and the Internet from your internal deployment.</span></span>
