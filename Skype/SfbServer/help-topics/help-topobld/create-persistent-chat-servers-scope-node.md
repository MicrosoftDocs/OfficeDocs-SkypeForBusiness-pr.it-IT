# <a name="create-persistent-chat-servers-scope-node"></a>Nodo dell'ambito di creazione di server Persistent Chat
 
Fare clic con il pulsante destro del mouse sul **pool di Chat** persistente in Generatore di topologie per eseguire le operazioni seguenti:
  
- **Nuovo pool di Chat persistente:** consente di avviare la procedura guidata Definisci nuovo **pool di Chat persistente** utilizzata per definire un nuovo server o pool di Chat persistente.
    
Oltre all'opzione Nuovo pool di Chat persistente, è possibile eseguire le attività generali seguenti:
  
- **Topologia**. È possibile utilizzare questa opzione per definire una nuova topologia, aprire una topologia esistente, scaricare la topologia corrente, salvare una copia della topologia, pubblicare la topologia, installare un database, unire componenti di versioni precedenti di Lync Server o rimuovere la distribuzione.
    
- **Guida**. È possibile utilizzare questa opzione per accedere alla Guida per il nodo attivo.
