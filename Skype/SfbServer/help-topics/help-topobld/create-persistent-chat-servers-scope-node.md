# <a name="create-persistent-chat-servers-scope-node"></a>Nodo dell'ambito di creazione di server Persistent Chat
 
Fare clic con il pulsante destro del mouse su **Pool Chat persistente** in Generatore di topologie per eseguire le operazioni seguenti:
  
- **Nuovo pool di Chat persistente**: avvia la procedura guidata Definisci nuovo **pool di Chat persistente** utilizzata per definire un nuovo server o un nuovo pool di Chat persistente.
    
Oltre all'opzione Nuovo pool di Persistent Chat, è possibile eseguire le attività generali seguenti:
  
- **Topologia**. È possibile utilizzare questa opzione per definire una nuova topologia, aprire una topologia esistente, scaricare la topologia corrente, salvare una copia della topologia, pubblicare la topologia, installare un database, unire componenti da versioni precedenti di Lync Server o rimuovere la distribuzione.
    
- **Guida**. È possibile utilizzare questa opzione per accedere alla Guida per il nodo attivo.
