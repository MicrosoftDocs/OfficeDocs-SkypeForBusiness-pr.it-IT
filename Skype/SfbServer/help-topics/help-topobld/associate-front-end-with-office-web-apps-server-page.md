# <a name="associate-front-end-with-office-web-apps-server-page"></a>Pagina per l'associazione di Front End Server a Office Web Apps Server

Il server Office Web Apps offre servizi simili a quelli di un relatore che carica un documento per l'utilizzo in una riunione, ma abilita funzionalità aggiuntive non supportate nelle versioni precedenti.

È possibile selezionare un server Office Web Apps per la distribuzione se si è scelto Servizio di conferenza come una delle opzioni di funzionalità per la configurazione di Skype for Business Server 2015.

Questa finestra di dialogo contiene le opzioni seguenti:

1. **Associare un pool a un server Office Web Apps:** selezionare questa casella di controllo se si fornirà e si distribuirà un server Office Web Apps per gli utenti delle conferenze.

2. Se si è selezionata la casella di controllo, usare l'elenco per selezionare un server Office Web Apps già definito.

3. Fare clic su **Nuovo** per definire un nuovo server Office Web Apps per la distribuzione.
