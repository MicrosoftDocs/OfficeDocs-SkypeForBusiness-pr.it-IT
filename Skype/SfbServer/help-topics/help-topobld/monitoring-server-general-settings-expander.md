# <a name="monitoring-server-general-settings-expander"></a>Espansione delle impostazioni generali di Monitoring Server
 
In Generatore di topologie è possibile modificare le proprietà di un singolo Monitoring Server facendo  clic con il pulsante destro del mouse sul Monitoring Server nell'albero della console e quindi scegliendo Azione sulla barra degli strumenti oppure facendo clic su un'attività nel riquadro Azioni e quindi scegliendo **Modifica** proprietà e modificando una delle opzioni seguenti:
  
- **FQDN**, per modificare il nome di dominio completo del server che si vuole distribuire come Monitoring Server.
    
- **Archivio SQL**, per cambiare l'istanza di SQL Server da usare per il monitoraggio del database di SQL Server. Se si cambia il database di SQL Server di un server Monitoring Server, è necessario riavviare il server Monitoring Server perché la modifica abbia effetto.
