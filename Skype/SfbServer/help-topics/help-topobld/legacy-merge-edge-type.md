# <a name="legacy-merge-edge-type"></a>Tipo di server perimetrale per unione legacy
 
Per un **singolo** Edge Server o un **Edge Server consolidato con bilanciamento del carico** , fare clic su **Avanti** per specificare le informazioni di configurazione. Per un **bordo espanso**, la configurazione di ogni server deve essere cambiata in una configurazione di Edge consolidata prima che possa essere unita a Lync Server 2013.
