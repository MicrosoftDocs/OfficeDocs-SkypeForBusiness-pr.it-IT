# <a name="legacy-merge-edge-type"></a>Unione legacy - Tipo di server perimetrale
 
Per un **Server perimetrale singolo** o un **Server perimetrale consolidato con carico bilanciato**, fare clic su **Avanti** per specificare le informazioni di configurazione. Per un **server perimetrale espanso,** la configurazione di ogni server deve essere modificata in una configurazione perimetrale consolidata prima di poter essere unita a Lync Server 2013.
