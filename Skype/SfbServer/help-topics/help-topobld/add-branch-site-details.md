# <a name="add-branch-site-details"></a>Aggiungere i dettagli del sito di succursale

Per specificare i dettagli del sito per un sito, eseguire le operazioni seguenti:

Specificare il nome di una città per il nuovo sito di succursale. Il valore relativo alla città è facoltativo.

Specificare il nome di una provincia per il nuovo sito di succursale. Il valore relativo alla provincia è facoltativo.

Specificare il nome di un codice paese per il nuovo sito di succursale. Il valore relativo al codice paese è facoltativo.

Per ulteriori informazioni, vedere [Sites](https://technet.microsoft.com/library/022cb6dd-37e2-4882-a53e-5ddfdbc6f53a.aspx).
