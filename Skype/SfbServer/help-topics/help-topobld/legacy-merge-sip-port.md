# <a name="legacy-merge-sip-port"></a>Porta SIP per l'unione legacy
 
La porta SIP predefinita del protocollo TCP (Transmission Control Protocol) per la comunicazione con client, pool Front End legacy e server è la 5061. Se non è stato utilizzato il valore predefinito, selezionare **Porta SIP per tutti i pool e i server:** e immettere il nuovo valore.
  
Se la distribuzione legacy è configurata per l'accesso automatico dei client, selezionare **Individuazione automatica della configurazione della porta SIP per ciascun pool** e fare clic su **Avanti** per continuare.
