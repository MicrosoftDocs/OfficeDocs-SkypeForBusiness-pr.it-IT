# <a name="add-archiving-server"></a>Aggiungere il server di archiviazione
 
Per distribuire un server di archiviazione nella progettazione della topologia, è necessario specificare il nome di dominio completo (FQDN) del server in cui si desidera distribuire l'archiviazione. Prima di pubblicare una topologia che include l'archiviazione e l'installazione di Skype for Business Server, è necessario che siano stati completati tutti i prerequisiti per la distribuzione dell'archiviazione.
