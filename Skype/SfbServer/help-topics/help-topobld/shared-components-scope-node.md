# <a name="shared-components-scope-node"></a>Nodo dell'ambito dei componenti condivisi
 
Fare clic con il pulsante destro del mouse sui **componenti condivisi** nell'albero della gerarchia del generatore di topologia per creare nuovi componenti condivisi o accedere alle opzioni del livello di topologia.
  
![Nodo dell'ambito dei componenti condivisi](../../media/Shared_Components_Scope_Node.jpg)
  
Selezionare **nuovo** per creare un nuovo FileStore, un nuovo gateway IP/PSTN, un nuovo server di Office Web Apps, un nuovo archivio di SQL Server o un nuovo trunk.
  
> [!NOTE]
> Vedere gli argomenti della Guida per ognuna di queste opzioni. 
  
![Nodo dell'ambito dei componenti condivisi](../../media/Shared_Components_Scope_NodeB.jpg)
  
Si seleziona **topologia** per fare clic su opzioni per aprire un nuovo documento di topologia. Aprire un documento esistente della topologia, scaricare la topologia corrente dall'Central Management store, salvare una copia del documento della topologia corrente, pubblicare il documento della topologia corrente, installare il database per creare le configurazioni di database attualmente configurate. Unire Office Communications Server 2007 R2 alla distribuzione corrente o rimuovere la distribuzione.
