# <a name="associate-branch-office-appliance-with-front-end"></a>Associare Branch Office Appliance a Front End Server
 
Per associare il Survivable Branch Appliance o il Survivable Branch Server a un pool Front End o a un server edizione Standard, nella pagina Selezionare il **pool Front End** selezionare il pool Front End o un server edizione Standard dall'elenco a discesa Pool Front **End.** Fare clic su **Avanti** per continuare.
