# <a name="associate-branch-office-appliance-with-front-end"></a>Appliance filiale associata con front-end
 
Per associare il Survivable Branch Appliance o il Survivable Branch Server con un pool Front end o un server Standard Edition, nella pagina **selezionare il pool di front-end** selezionare il pool Front end o un server Standard Edition dal **pool Front-End** elenco a discesa. Fare clic su **Avanti** per continuare.
