# <a name="legacy-merge-results"></a>Risultati dell'unione legacy

Per iniziare a unire una distribuzione legacy con Skype for Business Server, fare clic su **Avanti**. Per informazioni dettagliate sull'esecuzione dell'unione guidata di Generatore di topologie, vedere [Merge Using Topology Builder Merge Wizard](https://technet.microsoft.com/library/c3f3c425-dab6-4dcd-bf0e-d7fde05f2ebf.aspx) nella documentazione sulla migrazione.
