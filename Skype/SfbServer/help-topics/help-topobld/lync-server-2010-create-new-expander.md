# <a name="lync-server-2010-create-new-expander"></a>Espansione dell'impostazione di creazione di nuovi elementi di Lync Server 2010
 
È possibile creare nuovi ruoli del server nel nodo Lync Server 2010 di Generatore di topologie facendo clic con il pulsante destro del mouse su **Lync Server 2010,** scegliere Nuovo **e** quindi selezionare il ruolo del server da creare.
