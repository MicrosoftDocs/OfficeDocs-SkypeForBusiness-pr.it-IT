# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Aggiungere la pagina front end SQL Store per Lync Server 2010
 
Si definiscono le proprietà degli archivi di SQL Server per il nuovo pool Front-end che si sta configurando.
  
> [!IMPORTANT]
> Se si sta configurando un server Standard Edition, questa finestra di dialogo verrà visualizzata ma non sarà possibile configurare le impostazioni. Fare clic su **Avanti** per continuare.
  
Impostazioni che è possibile configurare:
  
- **Usare un archivio di SQL Server definito in precedenza**: selezionare un archivio di SQL Server esistente nell'elenco per lo Store del pool Front-end.
    
- Selezionare **Definisci un nuovo archivio di SQL Server** per creare un nuovo archivio di SQL Server.
    
    Si digita il nome di dominio completo di SQL Server in **FQDN di SQL Server**
    
    L' **istanza predefinita** è selezionata inizialmente. Se si preferisce definire un'istanza, selezionare **istanza denominata** e digitare il nome dell'istanza che è necessario creare.
    
  **Indietro**: Torna alla finestra di dialogo precedente
  
  **Next**: esegue il commit delle modifiche e passa alla finestra di dialogo passaggio successivo
  
  **Annulla**: Elimina le modifiche e Annulla la configurazione
  
  **Guida**: Visualizza le informazioni della guida, ad esempio questo argomento
