# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Sviluppare il dashboard della qualità delle chiamate per Skype for Business Server
 
**Riepilogo:** Informazioni su come sviluppare soluzioni per il dashboard qualità chiamata. Call Quality dashboard è uno strumento per Skype for Business Server.
  
Questo argomento fornisce la documentazione per lo sviluppo di Call Quality Dashboard (Call Quality Dashboard) per Skype for Business Server.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Sviluppare il dashboard della qualità delle chiamate per Skype for Business Server

Call Quality dashboard può essere esteso e integrato nell'ambiente di creazione di report usando lo sviluppo personalizzato. 
  
Sono disponibili due set di API Web, uno per l'accesso ai dati nel cubo (tramite API dati) e uno per l'accesso ai metadati del report (tramite l'API del repository). 
  
- [API dati per Call Quality Dashboard (Call Quality Dashboard) in Skype for Business Server](data-api.md)
    
- [API del repository per Call Quality Dashboard (Call Quality Dashboard) in Skype for Business Server](repository-api.md)
