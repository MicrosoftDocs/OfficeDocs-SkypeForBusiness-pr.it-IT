# <a name="monitoring-planning-tool"></a>Strumento di pianificazione del monitoraggio

In Skype for Business Server 2015, il monitoraggio viene utilizzato per raccogliere informazioni sull'utilizzo e dati QoE (Quality of Experience) sulle sessioni di comunicazione in cui sono coinvolti gli utenti.

Per ulteriori informazioni sulla pianificazione di Monitoring Server, vedere [Planning for Monitoring.](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-monitoring)
