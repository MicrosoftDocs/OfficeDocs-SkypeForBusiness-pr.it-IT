# <a name="federation-planning-tool"></a>Strumento di pianificazione della federazione
 
I server perimetrali possono essere configurati per consentire agli utenti interni ed esterni di accedere ai contatti nelle organizzazioni o nei servizi partner.
  
 La federazione funziona come per Microsoft Lync 2013. Per ulteriori informazioni sulla preparazione dell'ambiente per la federazione, vedere i seguenti argomenti:
  
- [Pianificazione della federazione di Lync Server 2013 e Office Communications Server](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-lync-server-and-office-communications-server-federation)
    
- [Pianificazione della connettività di messaggistica istantanea pubblica in Lync Server 2013](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-public-instant-messaging-connectivity)
    
- [Pianificazione della federazione XMPP (Extensible Messaging and Presence Protocol) in Lync Server 2013](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-extensible-messaging-and-presence-protocol-xmpp-federation)
