# <a name="mobility-planning-tool"></a>Mobilità (strumento di pianificazione)
 
La funzionalità mobilità in Skype for Business Server 2015 supporta le funzionalità di Skype for business nei dispositivi mobili.
  
La funzionalità di mobilità è la stessa di Lync Server 2013. Per altre informazioni su come preparare l'ambiente per la mobilità, vedere [pianificare la mobilità per Skype for Business Server](../../plan-your-deployment/mobility.md).
