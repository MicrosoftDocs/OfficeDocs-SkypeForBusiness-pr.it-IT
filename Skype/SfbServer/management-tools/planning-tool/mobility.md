# <a name="mobility-planning-tool"></a>Strumento di pianificazione della mobilità
 
La funzionalità per dispositivi mobili di Skype for Business Server 2015 supporta Skype for Business nei dispositivi mobili.
  
La funzionalità per dispositivi mobili è la stessa di Lync Server 2013. Per ulteriori informazioni sulla preparazione dell'ambiente per dispositivi mobili, vedere [Plan for Mobility for Skype for Business Server](../../plan-your-deployment/mobility.md).
