# <a name="mobility-planning-tool"></a>Mobilità (strumento di pianificazione)
 
La funzionalità per dispositivi mobili di Skype for Business Server 2015 supporta le funzionalità di Skype for business su un dispositivo mobile.
  
La funzionalità di mobilità è identica a quella di Lync Server 2013. Per ulteriori informazioni sulla preparazione dell'ambiente per la mobilità, vedere [Plan for Mobility for Skype for Business Server](../../plan-your-deployment/mobility.md).
