# <a name="skype-for-business-server-2015-management-tools"></a>Strumenti di gestione di Skype for Business Server 2015
 
**Riepilogo:** Informazioni sugli strumenti di gestione dei servizi in Skype for Business Server 2015.
  
Il software di comunicazione Skype for Business Server 2015 (in precedenza Lync Server), offre soluzioni di messaggistica istantanea, presenza, conferenze e telefonia che supportano i requisiti di collaborazione a livello aziendale. Gli strumenti per gestire questi servizi sono flessibili e potenti. 
  
## <a name="skype-for-business-server-2015-tools"></a>Strumenti di Skype for Business Server 2015

||[Microsoft Call Quality Methodology Scorecard, v1.5](https://go.microsoft.com/fwlink/p/?LinkId=615208) (download con estensione zip) <br/> [CQM Poster for Skype for Business](https://go.microsoft.com/fwlink/p/?LinkID=617898) <br/> [Poster CQM per Lync 2013](https://go.microsoft.com/fwlink/p/?LinkId=391841) <br/> |Versione aggiornata della scorecard CQM (Microsoft Call Quality Methodology) per Lync Server e Skype for Business Server 2015. CQM divide un'implementazione lync/Skype for Business in dieci aree discrete che influiscono sulla qualità, definendo gli obiettivi e un piano di correzione per ognuna. CQM è un framework per affrontare i problemi di qualità delle chiamate: è possibile modificarlo o estenderlo per risolvere le condizioni specifiche della rete  <br/> Il poster CQM consente di conoscere CQM, la metodologia di qualità delle chiamate per Lync e Skype for Business che consente di individuare ed eliminare i problemi che influiscono sulla qualità delle chiamate e sull'esperienza utente per le implementazioni lync/Skype for Business che includono funzionalità di voIP aziendale.  <br/>**Nota:** Questi strumenti non verranno aggiornati per Skype for Business Server 2019. |
