# <a name="deploy-edge-server-in-skype-for-business-server"></a>Distribuire il server perimetrale in Skype for Business Server
 
**Riepilogo:** Informazioni su come distribuire un server perimetrale o un pool di server perimetrali nell'ambiente Skype for Business Server.
  
Perché distribuire un server perimetrale o un pool di server perimetrali nell'ambiente Skype for Business Server? È necessario che gli utenti esterni non connessi alla rete interna dell'organizzazione siano in grado di interagire con gli utenti interni. Questi utenti esterni potrebbero essere utenti remoti autenticati e anonimi, partner federati o altri client mobili.
