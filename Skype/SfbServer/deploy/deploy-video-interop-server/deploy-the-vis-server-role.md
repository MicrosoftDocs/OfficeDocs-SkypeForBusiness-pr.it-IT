# <a name="deploy-the-vis-server-role-in-skype-for-business-server"></a>Distribuire il ruolo del server VIS in Skype for Business Server
 
**Riepilogo:** Distribuire il ruolo Video Interop Server (VIS) in Skype for Business Server.
  
Per configurare il servizio VIS nel server appena creato in Generatore di topologie, avviare la distribuzione guidata di Skype for Business Server, premere Installa o Aggiorna il sistema **Skype for Business Server** e seguire questi passaggi della procedura guidata:
  
1.  Selezionare **Installa archivio di configurazione locale.**
    
2. Selezionare **Installazione o Rimuovi componenti di Skype for Business Server.**
    
3. Selezionare **Richiedi, Installa o Assegna certificati.**
    
4. Selezionare **Avvia servizi.**
    
Il software per questo servizio è ora installato e in esecuzione. You may open the Services mmc tool to see if the **Skype for Business Server Video Interop Server** service is running along with other Skype for Business Server services. Successivamente, è necessario configurare il server o il pool VIS.
