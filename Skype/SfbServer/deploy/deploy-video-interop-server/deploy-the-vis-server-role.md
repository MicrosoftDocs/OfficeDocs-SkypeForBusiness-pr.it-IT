# <a name="deploy-the-vis-server-role-in-skype-for-business-server"></a>Distribuire il ruolo del server VIS in Skype for Business Server
 
**Riepilogo:** Distribuire il ruolo di video Interop Server (VIS) in Skype for Business Server.
  
Per configurare il servizio VIS nel server appena creato in Generatore di topologia, avviare la distribuzione guidata di Skype for Business Server, premere **Install o Update Skype for Business Server System** e seguire questa procedura nella procedura guidata:
  
1.  Selezionare **Installa archivio configurazione locale**.
    
2. Selezionare **Imposta o Rimuovi componenti di Skype for Business Server**.
    
3. Selezionare **Richiedi, installa o assegna certificati**.
    
4. Selezionare **Avvia servizi**.
    
Il software per questo servizio è ora installato ed eseguito. È possibile aprire lo strumento MMC Servizi per verificare se il servizio **Skype for Business Server video Interop server** è in uso insieme ad altri servizi Skype for Business Server. Devi quindi configurare il server o il pool VIS.
