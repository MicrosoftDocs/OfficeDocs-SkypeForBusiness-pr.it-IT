# <a name="requirements-for-your-skype-for-business-2015-environment"></a>Requisiti per l'ambiente Skype for Business 2015
 
**Riepilogo:** Struttura degli argomenti relativi ai requisiti di sistema e ai requisiti di rete o infrastruttura per Skype for Business Server 2015. Se è necessario installare o configurare qualsiasi elemento come prerequisito per l'installazione e la distribuzione di Skype for Business Server 2015, sarà qui.
  
Questa è la pagina di destinazione per gli articoli sui requisiti relativi a una distribuzione di Skype for Business Server 2015. Sono inclusi sia i requisiti del server che gli aspetti più generali per la preparazione, tra cui la struttura del dominio e della rete e i certificati.
  
## <a name="whats-in-this-topic"></a>Informazioni in questo argomento

I due articoli nella sezione requisiti per l'ambiente illustrano cosa è necessario avere e/o configurare prima di poter procedere e distribuire correttamente o eseguire l'aggiornamento a Skype for Business Server 2015 nell'ambiente server dell'organizzazione.
  
### <a name="server-requirements"></a>Requisiti per i server

I requisiti del server per [Skype for Business Server 2015](server-requirements.md) sono gli elementi necessari per prepararsi nei singoli server che si intende utilizzare nella distribuzione.
  
### <a name="environmental-requirements"></a>Requisiti di ambiente

[I requisiti ambientali per Skype for Business Server 2015](environmental-requirements.md) sono una gamma più ampia di elementi che devono essere pronti nell'infrastruttura prima di distribuire Skype for Business Server 2015.
