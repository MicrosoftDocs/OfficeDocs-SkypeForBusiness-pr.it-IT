# <a name="requirements-for-your-skype-for-business-2015-environment"></a>Requisiti per l'Skype for Business 2015
 
**Riepilogo:** Descrizione degli argomenti relativi ai requisiti di sistema e ai requisiti di rete o di infrastruttura per Skype for Business Server 2015. Se è necessario installare o configurare qualsiasi elemento come prerequisito per l'installazione e la distribuzione Skype for Business Server 2015, sarà qui.
  
Questa è la pagina di destinazione per gli articoli sui requisiti relativi a una Skype for Business Server 2015. Sono inclusi sia i requisiti del server che gli aspetti più generali da preparare, inclusa la struttura di dominio e di rete, e i certificati.
  
## <a name="whats-in-this-topic"></a>Informazioni in questo argomento

I due articoli nella sezione requisiti per l'ambiente illustrano cosa è necessario avere e/o configurare prima di poter procedere con la distribuzione o l'aggiornamento a Skype for Business Server 2015 nell'ambiente server dell'organizzazione.
  
### <a name="server-requirements"></a>Requisiti per i server

[I requisiti del server Skype for Business Server 2015](server-requirements.md) sono gli elementi che è necessario prepararsi nei singoli server che si intende utilizzare nella distribuzione.
  
### <a name="environmental-requirements"></a>Requisiti di ambiente

[I requisiti ambientali Skype for Business Server 2015](environmental-requirements.md) sono una gamma più ampia di elementi che devono essere pronti nell'infrastruttura prima di Skype for Business Server 2015.
