# <a name="get-ccappliancestatus"></a>Get-CcApplianceStatus
 
Il cmdlet Get-CcApplianceStatus fornisce informazioni di diagnostica per l Skype for Business Cloud Connector Edition appliance.
  
Questo cmdlet si applica Skype for Business Cloud Connector Edition 2.1.
  
```
    Get-CcApplianceStatus 
```

## <a name="detailed-description"></a>Descrizione dettagliata

Il cmdlet Get-CcApplianceStatus, se eseguito nel computer host Cloud Connector, fornisce informazioni di diagnostica per l'Skype for Business Cloud Connector.
  
## <a name="parameters"></a>Parametri

Nessuno
  
## <a name="input-types"></a>Tipi di input

Nessuna. Il cmdlet Get-CcApplianceStatus non accetta input da pipeline.
  
## <a name="return-types"></a>Tipi restituiti

Il comando restituisce lo stato dell'appliance.
