# <a name="region-table"></a>Tabella Region
 
La tabella Region è una tabella di supporto. Ogni record rappresenta un paese definito nell'impostazione di configurazione di rete.
  
|**Colonna**|**Tipo di dati**|**Chiave/indice**|**Dettagli**|
|:-----|:-----|:-----|:-----|
|**RegionKey** <br/> |int  <br/> |Principale  <br/> |Numero univoco che identifica il paese/area geografica.  <br/> |
|**RegionName** <br/> |nvarchar(128)  <br/> |Univoco  <br/> |Nome del paese.  <br/> |
