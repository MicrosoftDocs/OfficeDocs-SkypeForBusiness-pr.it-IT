La tabella MacAddress è una tabella di supporto. Ogni record rappresenta una sola origine.
  
|**Colonna**|**Tipo di dati**|**Chiave/indice**|**Dettagli**|
|:-----|:-----|:-----|:-----|
|**MacAddressKey** <br/> |int  <br/> |Principale  <br/> |Numero univoco che identifica l'indirizzo Mac.  <br/> |
|**MacAddress** <br/> |varchar (256)  <br/> |Univoci  <br/> |Stringa dell'indirizzo Mac.  <br/> |
