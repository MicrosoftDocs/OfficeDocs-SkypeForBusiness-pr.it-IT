|:-----|:-----|:-----|:-----|
|**SubnetIP** <br/> |int  <br/> |Primaria, esterna  <br/> |Rappresentazione integer per la subnet.  <br/> |
|**NextUpdateTS** <br/> |datetime  <br/> ||Solo per uso interno.  <br/> |
