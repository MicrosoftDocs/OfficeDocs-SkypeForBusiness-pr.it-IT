# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision contiene il numero di revisione utilizzato con la tabella tblAdminLock per ottenere la coerenza tra più client amministratore.
  
**Colonne**

|**Colonna**|**Tipo**|**Descrizione**|
|:-----|:-----|:-----|
|sysRevision  <br/> |bigint, non null  <br/> |Numero revisione.  <br/> |
