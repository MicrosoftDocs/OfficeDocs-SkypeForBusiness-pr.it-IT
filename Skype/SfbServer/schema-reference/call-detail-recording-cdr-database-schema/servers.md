# <a name="servers-table"></a>Tabella Servers
 
La tabella Servers è una tabella di supporto in cui vengono archiviate informazioni sui vari server. Ogni record nella tabella rappresenta un server.
  
|**Colonna**|**Tipo di dati**|**Chiave/indice**|**Dettagli**|
|:-----|:-----|:-----|:-----|
|**ServerId** <br/> |int  <br/> |Principale  <br/> |Numero univoco che identifica il server.  <br/> |
|**ServerFQDN** <br/> |nvarchar (256)  <br/> | <br/> |Nome di dominio completo del server.  <br/> |
