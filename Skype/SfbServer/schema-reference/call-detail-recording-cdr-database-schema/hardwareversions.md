# <a name="hardwareversions-table-in-skype-for-business-server-2015"></a>Tabella HardwareVersions in Skype for Business Server 2015
 
La tabella HardwareVersions è una tabella di supporto. In ogni record sono archiviate informazioni su una versione hardware del dispositivo.
  
|**Colonna**|**Tipo di dati**|**Chiave/indice**|**Dettagli**|
|:-----|:-----|:-----|:-----|
|**VersionId** <br/> |int  <br/> |Principale  <br/> |Numero univoco che identifica questa versione hardware.  <br/> |
|**Versione** <br/> |nvarchar(256)  <br/> | <br/> |Versione hardware.  <br/> |
