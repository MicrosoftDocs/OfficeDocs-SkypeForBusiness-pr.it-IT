# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>Tabella Manufacturers Skype for Business Server 2015
 
La tabella Manufacturers è una tabella di supporto. In ogni record sono archiviate informazioni su un produttore di un dispositivo (telefono da tavolo).
  
|**Colonna**|**Tipo di dati**|**Chiave/indice**|**Dettagli**|
|:-----|:-----|:-----|:-----|
|**ManufacturerId** <br/> |int  <br/> |Principale  <br/> |Numero univoco che identifica il produttore.  <br/> |
|**Produttore** <br/> |nvarchar(256)  <br/> | <br/> |Nome produttore.  <br/> |
