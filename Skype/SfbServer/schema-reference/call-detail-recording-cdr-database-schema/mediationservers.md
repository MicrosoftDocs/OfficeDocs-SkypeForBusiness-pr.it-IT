|**Colonna**|**Tipo di dati**|**Chiave/indice**|**Dettagli**|
|:-----|:-----|:-----|:-----|
|**MediationServerId** <br/> |int  <br/> |Principale  <br/> |Numero univoco che identifica il Mediation Server.  <br/> |
|**MediationServer** <br/> |nvarchar(256)  <br/> | <br/> |Nome mediation server.  <br/> |
