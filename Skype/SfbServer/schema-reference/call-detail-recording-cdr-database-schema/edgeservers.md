# <a name="edgeservers-table-in-skype-for-business-server-2015"></a>Tabella EdgeServers in Skype for Business Server 2015
 
La tabella EdgeServers è una tabella di supporto. Ogni record archivia le informazioni su un server perimetrale coinvolto in chiamate che includono record nel database.
  
|**Colonna**|**Tipo di dati**|**Chiave/indice**|**Dettagli**|
|:-----|:-----|:-----|:-----|
|**EdgeServerId** <br/> |int  <br/> |Principale  <br/> |Numero univoco che identifica questo Edge Server.  <br/> |
|**EdgeServer** <br/> |nvarchar (256)  <br/> | <br/> |Nome del server perimetrale.  <br/> |
