# <a name="pools-table"></a>Tabella Pools
 
La tabella Pools è una tabella di supporto in cui sono archiviate le informazioni sui vari pool. Ogni record della tabella rappresenta un pool.
  
|**Colonna**|**Tipo di dati**|**Chiave/indice**|**Dettagli**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |int  <br/> |Principale  <br/> |Numero univoco che identifica il pool.  <br/> |
|**PoolFQDN** <br/> |nvarchar(256)  <br/> | <br/> |Nome di dominio completo del pool.  <br/> |
