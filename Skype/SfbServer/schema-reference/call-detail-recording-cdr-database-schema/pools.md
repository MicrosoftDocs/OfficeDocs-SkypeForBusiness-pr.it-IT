# <a name="pools-table"></a>Tabella Pools
 
La tabella Pools è una tabella di supporto in cui vengono archiviate informazioni sul pool diverso. Ogni record nella tabella rappresenta un pool.
  
|**Colonna**|**Tipo di dati**|**Chiave/indice**|**Dettagli**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |int  <br/> |Principale  <br/> |Numero univoco che identifica questo pool.  <br/> |
|**PoolFQDN** <br/> |nvarchar (256)  <br/> | <br/> |FQDN del pool.  <br/> |
